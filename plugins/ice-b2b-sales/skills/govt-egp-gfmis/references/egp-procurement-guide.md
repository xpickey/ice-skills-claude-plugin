# คู่มือการจัดซื้อจัดจ้างผ่านระบบ e-GP

อ้างอิง: พรบ.การจัดซื้อจัดจ้างและการบริหารพัสดุภาครัฐ พ.ศ. 2560 + ระเบียบกระทรวงการคลังว่าด้วยการจัดซื้อจัดจ้างฯ พ.ศ. 2560

---

## 1. ภาพรวมระบบ e-GP (Electronic Government Procurement)

### 1.1 ช่องทางการใช้งาน
- **เว็บไซต์หลัก**: www.gprocurement.go.th
- **หน่วยงาน**: ประกาศแผน/ผล/สัญญาในระบบ + เว็บไซต์ของตนเอง
- **หนังสือเวียน ว.322**: วงเงิน ≥ 5,000 บาท ต้องบันทึกในระบบ e-GP
  - ยกเว้น: น้ำมันที่ซื้อ < 10,000 ลิตร ไม่ต้องทำในระบบ e-GP

### 1.2 สิ่งที่ต้องทำในระบบ e-GP
| ขั้นตอน | กำหนด |
|--------|-------|
| ประกาศแผนการจัดซื้อจัดจ้างประจำปี | ก่อนดำเนินการจัดซื้อจัดจ้าง |
| ประกาศ TOR / เอกสารเชิญชวน | ตามระยะเวลาที่กำหนดในระเบียบ |
| ประกาศราคากลาง | ก่อนวันยื่นข้อเสนอ |
| ประกาศผลผู้ชนะ | ทันทีหลังมีคำสั่งอนุมัติ |
| เผยแพร่สาระสำคัญสัญญา | ภายใน 30 วันหลังลงนาม |

## 2. กระบวนการจัดซื้อจัดจ้างแบบละเอียด

### 2.1 ขั้นที่ 1: แผนการจัดซื้อจัดจ้างประจำปี (ระเบียบฯ ข้อ 11)

**เมื่อไร**: หลังได้รับความเห็นชอบวงเงินงบประมาณจากหน่วยงานที่เกี่ยวข้อง

**องค์ประกอบแผน**:
1. ชื่อโครงการที่จะจัดซื้อจัดจ้าง
2. วงเงินที่จะจัดซื้อจัดจ้างโดยประมาณ
3. ระยะเวลาที่คาดว่าจะจัดซื้อจัดจ้าง
4. รายการอื่นตามที่กรมบัญชีกลางกำหนด

**ขั้นตอน**:
```
เจ้าหน้าที่จัดทำแผน → เสนอหัวหน้าหน่วยงานเห็นชอบ
    → หัวหน้าเจ้าหน้าที่ประกาศเผยแพร่
    → ระบบ e-GP + เว็บไซต์ + ปิดประกาศสถานที่
```

**กรณียกเว้น** ไม่ต้องจัดทำแผน (7 กรณี):
- ม.56(1)(ค) – วิธีคัดเลือก กรณีจำเป็นเร่งด่วน
- ม.56(1)(ฉ) – วิธีคัดเลือก พัสดุใช้ในราชการลับ
- ม.56(2)(ข) – วิธีเฉพาะเจาะจง วงเงิน ≤ 500,000 บาท
- ม.56(2)(ง) – วิธีเฉพาะเจาะจง ฉุกเฉิน
- ม.56(2)(ฉ) – วิธีเฉพาะเจาะจง ขายทอดตลาด
- ม.70(3)(ข) – จ้างที่ปรึกษาเฉพาะเจาะจง วงเงิน ≤ 500,000 บาท
- ม.70(3)(ฉ)/ม.82(3) – จำเป็นเร่งด่วน/ความมั่นคง

### 2.2 ขั้นที่ 2: TOR และราคากลาง (ระเบียบฯ ข้อ 21)

**ผู้รับผิดชอบ**: หัวหน้าหน่วยงานแต่งตั้งคณะกรรมการ หรือมอบหมายเจ้าหน้าที่/บุคคลใดบุคคลหนึ่ง

**11 หัวข้อ TOR บังคับ**:
1. ที่มาหรือความเป็นมา
2. วัตถุประสงค์ของโครงการ
3. คุณสมบัติของผู้เสนอราคา
4. คุณลักษณะเฉพาะ/ขอบเขตงาน/แบบรูปรายการก่อสร้าง
5. เกณฑ์ในการพิจารณาคัดเลือกข้อเสนอ
6. ระยะเวลาการดำเนินการ
7. ระยะเวลาในการส่งมอบ
8. การกำหนดระยะเวลารับประกันความชำรุดบกพร่อง + อัตราค่าปรับ
9. การจ่ายเงินล่วงหน้า
10. วงเงินในการจัดหา
11. หน่วยงานผู้รับผิดชอบ

**ลำดับการใช้ราคากลาง (มาตรา 4)**:
```
ลำดับ 1: ราคาจากหลักเกณฑ์คณะกรรมการราคากลาง        ← ใช้ก่อน
ลำดับ 2: ราคาจากฐานข้อมูลอ้างอิง กรมบัญชีกลาง
ลำดับ 3: ราคามาตรฐาน สำนักงบประมาณ/หน่วยงานกลาง
ลำดับ 4: ราคาสืบจากท้องตลาด
ลำดับ 5: ราคาที่เคยซื้อ/จ้างครั้งล่าสุดภายใน 2 ปีงบประมาณ
ลำดับ 6: ราคาอื่นตามแนวทางปฏิบัติหน่วยงาน           ← ใช้หลังสุด
```

> 💡 **IT Note**: วิธีเฉพาะเจาะจง — ไม่มีระเบียบกำหนดให้ต้องสืบราคาครบ 3 ราย สืบ 1 รายก็เพียงพอ

**การเผยแพร่ราคากลาง**: ต้องประกาศในระบบ e-GP ตาม มาตรา 63 ก่อนวันยื่นข้อเสนอ

### 2.3 ขั้นที่ 3: รายงานขอซื้อ/ขอจ้าง (ระเบียบฯ ข้อ 22)

**8 รายการในรายงาน**:
1. เหตุผลและความจำเป็นที่ต้องซื้อหรือจ้าง
2. ขอบเขตงาน/คุณลักษณะเฉพาะ/แบบรูปรายการ
3. ราคากลาง
4. วงเงิน (งบประมาณ หรือประมาณการ)
5. กำหนดเวลาที่ต้องการ
6. วิธีที่จะซื้อ/จ้าง + เหตุผล
7. หลักเกณฑ์การพิจารณาข้อเสนอ
8. ข้อเสนออื่น (แต่งตั้งคณะกรรมการ, ออกประกาศ ฯลฯ)

**ยกเว้น** (3 กรณี):
- เร่งด่วนอันเนื่องจากเหตุไม่คาดหมาย [ม.56(1)(ค)]
- ฉุกเฉิน [ม.56(2)(ง)]
- วงเงิน ≤ 100,000 บาท → ทำเฉพาะรายการที่เห็นว่าจำเป็น

**คณะกรรมการ** (ประธาน 1 + กรรมการ ≥ 2 คน):

| คณะกรรมการ | หน้าที่ |
|-----------|--------|
| พิจารณาผล e-bidding | ตรวจสอบเอกสาร/คุณสมบัติ/ราคา |
| พิจารณาผลสอบราคา | เหมือน e-bidding |
| ซื้อ/จ้างวิธีคัดเลือก | กำหนดผู้รับเชิญ/พิจารณาผล |
| ซื้อ/จ้างวิธีเฉพาะเจาะจง | ต่อรองราคา/พิจารณาผล |
| **ตรวจรับพัสดุ** | **ตรวจรับงาน → มติเอกฉันท์เท่านั้น** |

> ⚠️ ประชุมคณะกรรมการ: องค์ประชุม ≥ กึ่งหนึ่ง; ประธานต้องอยู่ทุกครั้ง; มติเสียงข้างมาก (ยกเว้น คณะกรรมการตรวจรับ → เอกฉันท์)

### 2.4 ขั้นที่ 4: ดำเนินการจัดซื้อจัดจ้าง

#### วิธีที่ 1: วิธีประกาศเชิญชวนทั่วไป / e-bidding

**ขั้นตอน e-bidding**:
```
1. จัดทำประกาศ + เอกสารประกวดราคา (IFB/RFP)
2. ประกาศในระบบ e-GP ≥ 5 วันทำการ
3. รับซื้อ/แจกเอกสาร
4. รับฟังความคิดเห็น TOR (กรณีประกาศเชิญชวน) [มาตรา 59]
5. ผู้ยื่นข้อเสนอส่งข้อเสนอทางอิเล็กทรอนิกส์
6. คณะกรรมการพิจารณาผลตรวจสอบเอกสาร + คุณสมบัติ
7. เรียงลำดับคะแนน → เลือกผู้ที่มีคะแนนสูงสุด [ม.65]
8. รายงานผลต่อหัวหน้าหน่วยงาน
```

**คุณสมบัติผู้ยื่นข้อเสนอ (มาตรา 64)**:
- มีความสามารถตามกฎหมาย
- ไม่เป็นบุคคลล้มละลาย
- ไม่อยู่ระหว่างเลิกกิจการ
- ไม่ถูกระงับการยื่นข้อเสนอ [ม.106 วรรค 3]
- ไม่ถูกแจ้งเวียนเป็นผู้ทิ้งงาน [ม.109]
- คุณสมบัติอื่นตามที่คณะกรรมการนโยบายประกาศกำหนด

#### วิธีที่ 2: วิธีคัดเลือก

**กรณีที่ใช้วิธีคัดเลือก [ม.56(1)]**:
```
(ก) ประกาศแล้วไม่มีผู้ยื่น / ไม่ผ่านคัดเลือก
(ข) พัสดุคุณลักษณะเฉพาะพิเศษ / ซับซ้อน / ผู้ประกอบการจำกัด
(ค) จำเป็นเร่งด่วน อันเนื่องจากเหตุที่ไม่อาจคาดหมายได้  ← IT Emergency
(ง) ต้องระบุยี่ห้อโดยลักษณะการใช้งาน / ข้อจำกัดเทคนิค  ← Specific Brand
(จ) จำเป็นต้องซื้อจากต่างประเทศ / ผ่านองค์การระหว่างประเทศ
(ฉ) พัสดุลับ / งานปกปิดลับ / ความมั่นคงชาติ
(ช) งานจ้างซ่อมที่ต้องถอดตรวจก่อนประมาณค่าซ่อม
(ซ) กรณีอื่นตามกฎกระทรวง
```

**ขั้นตอน**: จัดทำหนังสือเชิญชวน (ไม่ประกาศสาธารณะ) → เชิญ ≥ 3 ราย → รับข้อเสนอ → พิจารณาผล

#### วิธีที่ 3: วิธีเฉพาะเจาะจง

**กรณีที่ใช้วิธีเฉพาะเจาะจง [ม.56(2)]**:
```
(ก) ใช้ e-bidding + คัดเลือกแล้ว แต่ไม่มีผู้ยื่น/ไม่ผ่าน
(ข) วงเงิน ≤ 500,000 บาท                           ← ใช้บ่อยที่สุด
(ค) คุณสมบัติโดยตรงรายเดียว / ตัวแทนจำหน่ายรายเดียวในไทย
(ง) ฉุกเฉิน อุบัติภัย ภัยธรรมชาติ โรคติดต่ออันตราย  ← COVID/ภัยพิบัติ
(จ) พัสดุเกี่ยวพันกับที่จัดซื้อแล้ว + ซื้อเพิ่มเพื่อสมบูรณ์/ต่อเนื่อง
(ฉ) พัสดุที่จะขายทอดตลาดโดยหน่วยงานรัฐ
(ช) ที่ดิน/สิ่งปลูกสร้างเฉพาะแห่ง
(ซ) กรณีอื่นตามกฎกระทรวง
```

**ขั้นตอน**: เจรจาต่อรองราคาโดยตรงกับผู้ขาย/ผู้รับจ้าง 1 ราย → บันทึกผลการต่อรอง

### 2.5 ขั้นที่ 5: อนุมัติสั่งซื้อ/สั่งจ้าง

| วิธี | ผู้มีอำนาจ | วงเงิน |
|-----|-----------|-------|
| ประกาศเชิญชวน (ข้อ 84) | หัวหน้าหน่วยงาน | ≤ 200,000,000 บาท |
| | ผู้มีอำนาจเหนือหนึ่งชั้น | > 200,000,000 บาท |
| วิธีคัดเลือก (ข้อ 85) | หัวหน้าหน่วยงาน | ≤ 100,000,000 บาท |
| | ผู้มีอำนาจเหนือหนึ่งชั้น | > 100,000,000 บาท |
| วิธีเฉพาะเจาะจง (ข้อ 86) | หัวหน้าหน่วยงาน | ≤ 50,000,000 บาท |
| | ผู้มีอำนาจเหนือหนึ่งชั้น | > 50,000,000 บาท |
| หัวหน้าเจ้าหน้าที่พัสดุ (ข้อ 79) | หัวหน้าเจ้าหน้าที่พัสดุ | ≤ 500,000 บาท |

**ผู้มีอำนาจเหนือหนึ่งชั้นแยกตามประเภทราชการ**:
| ประเภท | หัวหน้าหน่วยงาน | ผู้มีอำนาจเหนือ |
|--------|----------------|----------------|
| ส่วนกลาง | อธิบดี / หัวหน้าส่วนราชการ (นิติบุคคล) | ปลัดกระทรวง/ปลัดทบวง |
| ส่วนภูมิภาค | ผู้ว่าราชการจังหวัด | ปลัดกระทรวงต้นสังกัด |
| ส่วนท้องถิ่น | นายก อบจ./เทศมนตรี/อบต. | ผู้ว่าราชการจังหวัด |

### 2.6 ขั้นที่ 6: ประกาศผลผู้ชนะ (มาตรา 66)

**ช่องทางประกาศ**:
1. ระบบ e-GP: www.gprocurement.go.th
2. เว็บไซต์หน่วยงาน
3. ปิดประกาศ ณ สถานที่ปิดประกาศของหน่วยงาน
4. แจ้งอีเมลผู้ยื่นข้อเสนอทุกราย (ตามแบบกรมบัญชีกลางกำหนด)

**กรณีที่ไม่ต้องบันทึกในระบบ e-GP** (หนังสือเวียน ว.322):
- ราชการส่วนกลาง/ภูมิภาค/ท้องถิ่น: วงเงิน < 5,000 บาท
- รัฐวิสาหกิจ/หน่วยงานรัฐอื่น: วงเงิน ≤ 100,000 บาท
- กรณีฉุกเฉินเร่งด่วนที่ดำเนินการตามระเบียบปกติไม่ทัน (ข้อ 79 วรรค 2)

**ลงนามสัญญาได้เมื่อ**:
- พ้น 7 วันทำการ (ระยะอุทธรณ์) + ไม่มีผู้อุทธรณ์
- หรือกรณีมีอุทธรณ์: คณะกรรมการพิจารณาอุทธรณ์แจ้งให้ดำเนินการต่อได้

**ยกเว้น** (ลงนามได้เลย): จัดซื้อเร่งด่วน [ม.56(1)(ค)] / วิธีเฉพาะเจาะจง / วงเงินน้อยตามกฎกระทรวง

### 2.7 ขั้นที่ 7: การอุทธรณ์ (มาตรา 114–119)

**ผู้มีสิทธิ์อุทธรณ์**: ผู้ที่ยื่นข้อเสนอ ซึ่งเห็นว่าหน่วยงานไม่ปฏิบัติตามหลักเกณฑ์จนเป็นเหตุให้ตนไม่ชนะ

**สิ่งที่อุทธรณ์ไม่ได้ (มาตรา 115)**:
- การเลือกวิธีจัดซื้อจัดจ้าง
- การยกเลิกการจัดซื้อจัดจ้างตาม ม.67
- การละเว้นการอ้างถึงกฎหมายในประกาศ
- กรณีอื่นตามกฎกระทรวง

**วิธีการอุทธรณ์**:
- ทำเป็นหนังสือ ลงลายมือชื่อ ใช้ถ้อยคำสุภาพ
- ระบุข้อเท็จจริง + เหตุผลให้ชัดเจน
- แนบเอกสารหลักฐานที่เกี่ยวข้อง
- ใช้แบบ **อธ.2** (แบบที่คณะกรรมการพิจารณาอุทธรณ์กำหนด)

**Timeline อุทธรณ์**:
```
วันประกาศผล
    │ ≤ 7 วันทำการ
    ▼
ยื่นอุทธรณ์ต่อหน่วยงาน (แบบ อธ.2)
    │ หน่วยงานพิจารณา ≤ 7 วันทำการ
    ├── เห็นด้วย → ดำเนินการตามความเห็น
    └── ไม่เห็นด้วย (ทั้งหมด/บางส่วน)
            │ ≤ 3 วันทำการ หลังครบ 7 วัน
            ▼
        ส่งคณะกรรมการพิจารณาอุทธรณ์
            │ ≤ 30 วัน (ขยายได้ 2 ครั้ง ครั้งละ ≤ 15 วัน)
            ├── อุทธรณ์ฟังขึ้น + มีผลนัยสำคัญ
            │   → สั่งจัดซื้อใหม่ / เริ่มจากขั้นตอนที่สมควร
            └── อุทธรณ์ฟังไม่ขึ้น / ไม่มีผลนัยสำคัญ
                → แจ้งหน่วยงานดำเนินการต่อ
                [คำวินิจฉัยเป็นที่สุด]
                ↓
            ฟ้องศาลได้เฉพาะเรื่องค่าเสียหาย
            (ไม่กระทบสัญญาที่ลงนามแล้ว)
```

### 2.8 ขั้นที่ 8: การทำสัญญา (มาตรา 93–99)

**หลักการ**: ต้องทำตามแบบที่คณะกรรมการนโยบายกำหนด (โดยความเห็นชอบของสำนักงานอัยการสูงสุด)

**แบบสัญญา 15 ประเภท** (ดูรายละเอียดใน SKILL.md หมวด 7)

**กรณีพิเศษ — ทำข้อตกลงแทนสัญญาได้ (มาตรา 96)**:
1. วิธีคัดเลือก [ม.56(1)(ค)] หรือวิธีเฉพาะเจาะจง [ม.56(2)(ข)(ง)(ฉ)]
2. จัดซื้อจัดจ้างจากหน่วยงานของรัฐด้วยกัน
3. ส่งมอบครบภายใน 5 วันทำการ
4. การเช่าที่ไม่มีค่าใช้จ่ายอื่น

**การจ้างช่วง (มาตรา 95)**:
- ห้ามจ้างช่วงทั้งหมดหรือบางส่วน เว้นแต่ได้รับอนุญาต
- ค่าปรับหากฝ่าฝืน: **≥ 10%** ของวงเงินงานที่จ้างช่วง

**หลักประกันสัญญา**:
- ประเภท: เงินสด / เช็ค/ดราฟท์ธนาคาร / หนังสือค้ำประกันธนาคาร / หนังสือค้ำประกันบริษัทเงินทุน / พันธบัตรรัฐบาล
- อัตรา: **5%** ของวงเงิน (ปกติ) / > 5% แต่ ≤ 10% (กรณีสำคัญพิเศษ)
- ยกเว้น: คู่สัญญาเป็นหน่วยงานรัฐ
- การคืน: ≤ **15 วัน** หลังพ้นข้อผูกพัน

**เผยแพร่สาระสำคัญสัญญา (มาตรา 98)**:
- ต้องประกาศใน e-GP + เว็บไซต์หน่วยงาน (รวมถึงการแก้ไขสัญญา)

### 2.9 ขั้นที่ 9: การบริหารสัญญาและตรวจรับพัสดุ (มาตรา 100–105)

**คณะกรรมการตรวจรับพัสดุ** (มาตรา 100):
- รับผิดชอบทั้งการบริหารสัญญา + ตรวจรับพัสดุ
- วงเงินน้อย → แต่งตั้งบุคคลเดียวได้

**หน้าที่ (ระเบียบฯ ข้อ 175)**:
1. ตรวจรับพัสดุ ณ สถานที่ที่กำหนดในสัญญา
2. ตรวจสอบให้ครบถ้วนตามหลักฐานที่ตกลงกัน (อาจเชิญผู้เชี่ยวชาญ/ส่งทดสอบได้)
3. ตรวจรับในวันที่นำส่ง และดำเนินการให้เสร็จโดยเร็ว
4. ตรวจครบ → รับและทำใบตรวจรับ ≥ 2 ฉบับ (ให้ผู้รับจ้าง 1 ฉบับ)
5. ตรวจพบไม่ครบ/ไม่ถูกต้อง → รายงาน + แจ้งผู้รับจ้างภายใน **3 วันทำการ**
6. พัสดุเป็นชุด ขาดส่วนใด = ถือว่ายังไม่ส่งมอบ
7. กรรมการบางคนไม่ยอมรับ → เสนอหัวหน้าหน่วยงานสั่งการ

**การพิจารณาแก้ไขสัญญา (มาตรา 97)**:
- แก้ไขได้หาก: ไม่ทำให้หน่วยงานเสียประโยชน์ / เพื่อประโยชน์หน่วยงาน/สาธารณะ
- กรณีเสียเปรียบ → ส่งร่างสัญญาให้อัยการสูงสุดพิจารณา

**ค่าปรับ (ระเบียบฯ ข้อ 162)**:

| ประเภทงาน | อัตราค่าปรับ |
|-----------|------------|
| สัญญาทั่วไป | 0.01%–0.20%/วัน ของพัสดุที่ยังไม่รับมอบ |
| งานจ้างผลสำเร็จพร้อมกัน | 0.01%–0.10%/วัน ของราคาจ้าง (≥ 100 บาท/วัน) |
| งานก่อสร้างกระทบจราจร | 0.25%/วัน ของราคาจ้าง |
| งานจ้างที่ปรึกษา | 0.01%–0.10%/วัน ของราคาจ้าง |

> ⚠️ พัสดุเป็นชุด: ขาดส่วนประกอบแม้ชิ้นเดียว = ส่งมอบไม่ได้เลย → ปรับเต็มราคาทั้งชุด

**การงด/ลดค่าปรับ หรือขยายเวลา (มาตรา 102)**:
- เหตุจากความผิด/บกพร่องหน่วยงาน (เช่น ส่งมอบพื้นที่ล่าช้า)
- เหตุสุดวิสัย
- พฤติการณ์ที่คู่สัญญาไม่ต้องรับผิดตามกฎหมาย
- เหตุอื่นตามกฎกระทรวง

## 3. Quick Reference: กรณีจัดซื้อจัดจ้าง IT ที่พบบ่อย

### Case 1: ซื้อ Software License ≤ 500,000 บาท
```
วิธี: เฉพาะเจาะจง [ม.56(2)(ข)]
แบบสัญญา: #4 (สัญญาซื้อขายและอนุญาตให้ใช้สิทธิโปรแกรมคอมพิวเตอร์)
ราคากลาง: สืบ 1 ราย (ไม่ต้องสืบ 3 ราย)
หลักประกัน: 5% (หรือยกเว้นถ้าส่งมอบใน 5 วันทำการ)
```

### Case 2: จ้างพัฒนาระบบ > 500,000 บาท
```
วิธี: e-bidding (ประกาศเชิญชวนทั่วไป)
แบบสัญญา: #15 (สัญญาจ้างทำของ)
TOR: ต้องครบ 11 หัวข้อ
ราคากลาง: สืบหลายราย/ฐานข้อมูล กรมบัญชีกลาง
ทรัพย์สินทางปัญญา: ต้องระบุ Source Code เป็นของรัฐ
```

### Case 3: จ้าง IT Consulting (ที่ปรึกษา)
```
วิธี: จ้างที่ปรึกษา (หมวด 7) — ไม่ใช่หมวด 6
แบบสัญญา: #14 (สัญญาจ้างบริษัทที่ปรึกษา)
ที่ปรึกษา: ต้องขึ้นทะเบียนศูนย์ข้อมูลที่ปรึกษา กค.
```

### Case 4: ระบบล่มฉุกเฉิน ต้องซื้อทันที
```
วิธี: คัดเลือก [ม.56(1)(ค)] — จำเป็นเร่งด่วนอันเนื่องจากเหตุไม่คาดหมาย
เงื่อนไข: ต้องบันทึกเหตุผลครบถ้วน + ระบุว่าหากใช้ e-bidding จะไม่ทันการ
ไม่ต้องจัดทำแผน + อาจไม่ต้องทำรายงานขอซื้อตามปกติ
```

### Case 5: ต่ออายุ Cloud Service หรือ Maintenance
```
วิธี: เฉพาะเจาะจง [ม.56(2)(จ)] — พัสดุเกี่ยวพันกับที่จัดซื้อไว้แล้ว
หรือ e-bidding หากวงเงินเกิน 500,000 บาท
สัญญา: #5 (เช่า Cloud) หรือ #6 (Maintenance)
⚠️ ห้ามต่อสัญญาอัตโนมัติ ต้องผ่านกระบวนการจัดซื้อใหม่ทุกครั้ง
```

## 4. คำถามที่พบบ่อย (FAQ)

**Q: บอกเลิกสัญญาแล้ว หลักประกันนำไปใช้อย่างอื่นได้ไหม?**
A: ตามระเบียบฯ เมื่อบอกเลิกสัญญา ให้ **ยึดหลักประกัน** ไม่สามารถนำไปใช้วัตถุประสงค์อื่น ให้หารือกรมบัญชีกลาง

**Q: การสืบราคากลาง จำเป็นต้องครบ 3 รายไหม?**
A: วิธีเฉพาะเจาะจง — ระเบียบ**ไม่ได้กำหนด** สืบ 1 รายหรือ 3 รายก็ได้

**Q: เครื่องถ่ายเอกสาร ซื้อหรือเช่าได้ไหม?**
A: ได้ทั้งคู่ ตาม ระเบียบฯ ข้อ 92 การเช่าสังหาริมทรัพย์ให้นำข้อกำหนดเกี่ยวกับการซื้อมาใช้โดยอนุโลม

**Q: การติดอากรแสตมป์สัญญาจ้างทำของ?**
A: ผู้รับจ้างต้องติดอากรแสตมป์ **1 บาทต่อ 1,000 บาท** (หรือเศษ 1,000 บาท); หากเกิน 200,000 บาท ต้องตีตราสารที่กรมสรรพากร

**Q: ลงนามสัญญาแล้วจะแก้ไขได้ไหม?**
A: แก้ไขได้เฉพาะกรณีที่กำหนดใน ม.97 (ไม่เสียประโยชน์หน่วยงาน / เพื่อประโยชน์หน่วยงาน/สาธารณะ) โดยต้องขออนุมัติผู้มีอำนาจ

**Q: จ้างช่วงงาน IT ให้ Subcontractor ได้ไหม?**
A: ได้เฉพาะบางส่วนที่ **ได้รับอนุญาต** จากหน่วยงาน; ห้ามจ้างช่วงทั้งหมด; ค่าปรับหากฝ่าฝืน ≥ 10% ของวงเงินงานนั้น
