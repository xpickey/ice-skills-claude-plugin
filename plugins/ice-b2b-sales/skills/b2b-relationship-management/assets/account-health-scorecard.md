## State-Triggered Playbook

> Health score is not a report — it is a trigger.

### If 🟢 Green
- [ ] Confirm advocacy posture (reference / case-study / advisory board)
- [ ] Identify expansion white-space (`renewal-expansion-motion.md`)
- [ ] Schedule next QBR / EBR
- [ ] Test the relationship: ask for a small favor (intro, panel speaker, beta participation)

### If 🟡 Yellow
- [ ] KAM diagnosis — which component(s) drove the drop?
- [ ] 30-day action plan with 3–5 named actions, owners, dates
- [ ] Notify Sales Leader (informational)
- [ ] Adjust cadence — extra working-level touches between scheduled QBRs
- [ ] Re-score in 30 days

### If 🟠 Orange
- [ ] Sales-Leader-led 60-day recovery plan
- [ ] Direct outreach: our exec → their exec
- [ ] Documented root cause + recovery actions
- [ ] Formal customer communication
- [ ] Internal monthly review (Delivery + CS + Finance)
- [ ] Re-score weekly

### If 🔴 Red
- [ ] Executive war-room governance (`escalation-recovery.md`)
- [ ] 14-day recovery plan with executive sponsorship on both sides
- [ ] Daily internal standup, first 2 weeks
- [ ] Customer-facing weekly status with named owners
- [ ] Consider: dedicated resources, fee adjustments, executive site visit
- [ ] Day 60 decision: meaningful recovery? If not, plan a graceful exit
