# Arctic Template — Designer Brief (iCE-CI Integrated)

> **Version:** V01R01 | **Date:** 2026-04-28
> **Concept:** "ฟ้าไอซ์บลู สะอาด เย็นสดชื่น" — Modern, Crisp, Transparent

---

## 1. Theme Concept (แนวคิดของ Template)

Arctic ออกแบบมาเพื่อสื่อสาร "ความทันสมัย ความล้ำหน้าทางเทคโนโลยี และความโปร่งใส"
เหมาะอย่างยิ่งสำหรับงาน Tech Solution Pitch, Software Demo, Product Launch, Innovation
Showcase, Cloud/AI/Data Strategy, และงานนำเสนอที่ต้องการให้ผู้ฟังรู้สึกถึงความเฉียบคม
และความใหม่ของเทคโนโลยี โดยใช้พื้นหลัง Alice Blue สว่างสดชื่น ตัดด้วย iCE Main Blue
และ Dark Turquoise เพื่อสร้างมิติของแสงและความเย็น

อารมณ์ของ Template: คมชัด สดชื่น โปร่งใส มีระบบระเบียบ ทันสมัย เป็นมิตรกับ Data

## 6. Image Treatment

- โทนภาพ: เย็น (Cool Tone) อุณหภูมิสี 6500–7500K
- Filter: เพิ่ม Brightness 5–10%, ลด Saturation 5%
- รูปทรง: สี่เหลี่ยมขอบมน 8 px (มุมแหลมกว่า Linen)
- Glow Effect: รูปสำคัญใส่ Outer Glow สี Turquoise Opacity 40%
- Style: ภาพ Tech, Data Visualization, Crystal Texture

---

## 7. Iconography Style

- สไตล์: Outline เส้นบาง 1.5 px ปลายแหลม
- ขนาด: 24 / 32 / 48 px
- สีหลัก: `#1E66A4`
- สีรอง: `#00CED1` (Turquoise)
- Recommend: Icon Set ตระกูล Lucide หรือ Phosphor Icons

---

## 8. Animation & Transition

- Slide Transition: Push From Right 0.3s
- Object Animation: Slide-In From Left 0.25s
- Data Point: Count-Up Animation สำหรับตัวเลข
- หลีกเลี่ยง: Bounce, Spin

## 10. 15 Layout Patterns

### Layout 01 — Title / Cover
- **Header:** ไม่มี (Full Bleed)
- **Image:** Background ภาพคริสตัล/น้ำแข็ง Opacity 25%
- **Text:** Headline ซ้ายบน 64 pt iCE Blue + Subtitle ใต้ headline 20 pt Turquoise
- **Logo:** มุมขวาบน 80 px

### Layout 02 — Section Divider
- **Header:** ไม่มี
- **Background:** Gradient Diagonal Alice Blue → Frost Mist
- **Text:** หมายเลข Section ตัวเลขใหญ่ Outline สี iCE Blue 90 pt + ชื่อ Section H1
- **Accent:** เส้นแนวตั้งสี Turquoise สูง 40%

### Layout 03 — Two-Column Text
- **Header:** Title Bar 72 px H2 ซ้าย
- **Body:** สองคอลัมน์ Body 18 pt
- **Divider:** เส้น Hairline 1 px สี Frost 90 ตรงกลาง
- **Bullet:** สี่เหลี่ยมเล็ก 6 px สี Turquoise

### Layout 04 — Text Left + Image Right
- **Header:** Title Bar 72 px
- **Image:** ขวา 55% ขอบมน 8 px พร้อม Outer Glow Turquoise
- **Text:** ซ้าย 45% H3 + Body
- **Caption:** ใต้ภาพ Steel Gray 14 pt

### Layout 05 — Image Left + Text Right
- (กลับด้านจาก Layout 04)
- **Use Case:** การแสดง Product Screenshot

### Layout 06 — Full-Bleed Image with Tech Overlay
- **Image:** เต็มหน้า + Overlay เส้นกริด iCE Blue Opacity 15%
- **Text:** Headline ซ้ายบน สีขาว Display
- **Logo:** ขวาบน สีขาว
- **Use Case:** Hero Slide สำหรับ Tech Pitch

### Layout 07 — Three-Column Cards
- **Cards:** 3 การ์ดพื้น Frost Mist ขอบมน 8 px Padding 24 px
- **Card Header:** Icon 32 px Turquoise + H3 iCE Blue
- **Hover State (Digital):** Border 2 px iCE Blue

### Layout 08 — Quote / Testimonial
- **Background:** Alice Blue
- **Quote Mark:** ใหญ่ Outline เส้น iCE Blue 200 pt
- **Quote Body:** 32 pt Italic Dark Slate
- **Attribution:** Avatar วงกลม 64 px + ชื่อ + ตำแหน่ง

### Layout 09 — Big Number / Stat Highlight
- **Header:** Title Bar 72 px H2 บอกบริบท
- **Stat:** ตัวเลข 200 pt สี iCE Blue กึ่งกลาง
- **Unit:** "%" หรือ "x" สี Turquoise ติดท้าย
- **Trend Arrow:** ลูกศรเฉียงขึ้นสี Emerald
- **Body:** อธิบายแหล่งที่มา 16 pt

### Layout 10 — Timeline / Roadmap
- **Timeline:** เส้นไล่ Gradient iCE Blue → Turquoise
- **Milestones:** วงกลมเรืองแสง 20 px พร้อม Glow Effect
- **Labels:** H3 + Caption สลับบน-ล่าง

### Layout 11 — Comparison Table
- **Header Row:** สี iCE Blue ตัวอักษรขาว Padding 16 px
- **Stripe:** สลับ Alice Blue / Frost Mist
- **Highlight Column:** Border 2 px Turquoise + Background Turquoise 10
- **Checkmark:** ✓ สี Emerald, ✗ สี Steel Gray

### Layout 12 — Process Flow / Diagram
- **Step Box:** ขอบมน 8 px พื้น Frost Mist
- **Connector:** ลูกศรปลายแหลมสี Turquoise + เส้น Dotted
- **Step Number:** วงกลม 36 px iCE Blue ตัวขาว
- **Use Case:** Workflow, Pipeline, Architecture

### Layout 13 — Bullet List / Key Points
- **Bullet:** ✓ Checkmark สี Turquoise
- **Indent:** 32 px
- **Line Spacing:** 1.6
- **Highlight:** คำสำคัญใต้ขีด Turquoise

### Layout 14 — Q&A / Discussion
- **Background:** Frost Mist
- **Icon:** เครื่องหมายคำถาม "?" Outline 240 pt iCE Blue โปร่งแสง 40%
- **Headline:** "Questions?" Display iCE Blue กึ่งกลาง
- **Subtitle:** "เปิดให้ซักถาม / Open Floor" Steel Gray

### Layout 15 — Closing / Thank You
- **Background:** Gradient Linear Alice Blue → Turquoise
- **Headline:** "Thank You" 80 pt iCE Blue
- **Contact:** Icon อีเมล/โทร/เว็บ พร้อมข้อมูล Caption
- **Logo:** ขวาบน
