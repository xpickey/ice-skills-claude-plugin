## 🔵 McKINSEY DNA (text-structured, hypothesis-driven)

```
LAYOUT PATTERNS:
  • ACTION TITLES — title = ประโยคสมบูรณ์ที่เป็นข้อสรุป ไม่ใช่หัวข้อ
    ✅ "ตลาดเยอรมันเติบโต 3 เท่าของยุโรป"   ❌ "ภาพรวมตลาด"
  • STANDALONE DOCUMENT — แต่ละ slide เป็นเอกสารอิสระ อ่านได้ด้วยตัวเอง + highlight takeaway
  • GHOST DECK / DIVIDER — slide สรุปที่เป็นทั้ง executive overview + outline section ถัดไป

STORYLINE FRAMEWORKS:
  • PYRAMID PRINCIPLE — นำด้วยข้อสรุป → 3-4 key argument → evidence (กลับธรรมชาติ ให้ exec อ่านง่าย)
  • SCQA — Situation → Complication → Question → Answer
  • MECE — Mutually Exclusive, Collectively Exhaustive (ไม่ซ้ำ ไม่ขาด)

CHART TYPES: Waterfall · 2×2 matrix · Driver tree · Value chain · Marimekko · Scenario comparison
สถิติ: ~20 slide types = 80% ของ layout (มาตรฐานชัด)
```

## 🟢 BCG DNA (framework + matrices, action-oriented)

```
SIGNATURE FRAMEWORKS:
  • Growth-Share Matrix (foundational) + 2×2 assessment grids (segment attractiveness, competitive positioning)
  • Waterfall — disaggregate ข้อมูลเป็น drivers (impact ของแต่ละ driver ต่อ total)

STRUCTURE:
  • "Where are we today → Where we want to be → How do we get there"
  • Summary-first: exec summary นำ → detailed appendix (deck สั้น 9-27 หน้า สกัดจาก doc 100+ หน้า)
  • Text density: ใช้ color + bold highlight key message ใน text-heavy slide

CHART TYPES: Benchmarking · Trend (then-now) · Ecosystem diagram (color-coded) · Funnel · Survey/sentiment + quotes

STORYLINE: Problem → Analysis → Recommendations → Implications (burning platform → findings → CTA for CEO)
  เน้นตอบ "So what?" เสมอ
```

## 🔴 BAIN DNA (results / financial-impact)

```
STRUCTURE: purpose/vision → current situation → potential levers for improvement
  modular: interim report = abbreviated version ของ final deliverable

VISUAL: color-coding link initiatives → overall areas (chromatic hierarchy สำหรับ initiative tracking)

RESULTS-ORIENTATION:
  • frame recommendation ด้วย financial impact (estimated cost savings) — ไม่ใช่ conceptual framework
  • quantifiable improvement opportunities เป็นแกน
```

## เปรียบเทียบ 3 DNA (เลือกตาม context)

| Context | DNA | จุดเด่น |
|---|---|---|
| Proposal วิเคราะห์ลึก, exec-facing | McKinsey | Action Title + Pyramid (structured) |
| Strategy, portfolio, positioning | BCG | Matrix + where-to-where + CTA |
| Value case, ROI, cost-savings | Bain | Financial-impact framing |
