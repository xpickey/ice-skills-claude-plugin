## 7. Self-check before delivery

- [ ] Each Why has a confidence level
- [ ] Each Why has at least one proof unit
- [ ] Why-Now has a real compelling event (not invented)
- [ ] Why-Invest is built from customer-specific baselines
- [ ] Why-Us names where we don't fit
- [ ] Why-Stay is part of the original case, not an afterthought
- [ ] Champion-arming version is in the Champion's language
- [ ] Thai/APAC calibration applied (face, bunkhun, hierarchy, cash-monthly view if SME, defensibility if government)
- [ ] Incumbent vendor is acknowledged, not disparaged
