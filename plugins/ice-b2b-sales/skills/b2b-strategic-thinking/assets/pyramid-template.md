# Pyramid Principle Template

Use this scaffold whenever the output is for an executive or decision-maker. Headline first, then 3 reasons (MECE, parallel), then evidence. Length is not the same as rigor.

---

## [Title — single line, anchored in the decision being recommended]

**Audience:** [CFO / CIO / CEO / Steering Committee / Account Sponsor / Board]
**Decision being made:** [one sentence]
**Confidence:** [High / Medium / Low]

---

### Headline (the one sentence the reader must remember)

> [Single sentence stating the recommendation. Falsifiable. Takes a position. If only this sentence survives, the reader still gets the answer.]

---

### Three reasons (MECE; parallel in form)

**1. [Reason headline — same form as #2 and #3]**
- Supporting point with evidence
- Supporting point with evidence
- Source / proof unit

**2. [Reason headline]**
- Supporting point with evidence
- Supporting point with evidence
- Source / proof unit

**3. [Reason headline]**
- Supporting point with evidence
- Supporting point with evidence
- Source / proof unit

---

### What would change this answer

[One short paragraph naming the assumption that, if wrong, flips the recommendation. This is what makes the recommendation executive-grade.]

---

### Suggested next move

[One bounded, time-boxed action. Not a list of options. The single next step that learns the most or commits the least.]

---

### Appendices (optional, only if the audience asks for depth)

- A1 — Detailed assumptions and data sources
- A2 — Counter-arguments considered
- A3 — Alternative options rejected and why
- A4 — Risk register and mitigation
- A5 — Implementation outline

---

## Quality checklist (run before delivering)

- [ ] Headline survives if every supporting bullet is deleted
- [ ] Three reasons are MECE (no overlap, no gaps)
- [ ] Reasons are parallel in form (all noun phrases, or all action statements, etc.)
- [ ] Each reason has a proof unit
- [ ] Confidence is honest, not theatre
- [ ] "What would change this answer" is named
- [ ] Next move is a single bounded action
- [ ] Total length matches audience altitude (CFO: 1 page; board: 2 pages; deep dive: as long as needed)
- [ ] If Thai/APAC context: relationship/face notes added where they shift the decision
