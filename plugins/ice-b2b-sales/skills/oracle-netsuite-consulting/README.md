# Oracle NetSuite Cloud ERP & EPM Consulting Skill

## Overview

This Claude Skill provides expert guidance on Oracle NetSuite Cloud ERP and EPM solutions, with specialized focus on Thailand and APAC localization. It delivers Big Four consulting-level advice on implementation, configuration, and optimization of NetSuite systems.

## What This Skill Covers

### NetSuite ERP Modules
- **Financials**: General Ledger, AP, AR, Fixed Assets, Cash Management, Revenue Recognition
- **Procurement**: Purchase requisitions, SuiteProcurement, vendor management, P2P processes
- **Supply Chain**: Inventory management, order management, demand/supply planning, warehouse operations

### NetSuite EPM/Performance Management
- Planning, budgeting and forecasting
- Financial consolidation and close (FCCS-style)
- Account reconciliation and transaction matching (ARCS-style)
- Profitability and cost management (PCMCS-style)

### Regional Localization
- **Thailand**: VAT, WHT, e-tax invoices, Por Por 30, Por Ngor Dor, BOI reporting, Thai TFRS
- **APAC**: Singapore, Malaysia, Indonesia, Vietnam, Philippines, India, China, Japan, Korea, Australia, New Zealand (GST/VAT, WHT, statutory reports)

### SuiteCloud Platform
- SuiteScript development and customization
- SuiteFlow workflow automation
- SuiteAnalytics reporting and dashboards
- Integration architecture and patterns

### Big Four Methodology
- Discovery and requirements gathering
- Solution and data design
- Build, configure and test
- Training and deployment
- Optimization and continuous improvement

## When to Use This Skill

## Installation

1. Download the `oracle-netsuite-consulting.zip` file
2. Open Claude.ai and navigate to Settings
3. Go to the Skills section
4. Click "Upload Skill"
5. Select the downloaded zip file
6. Enable the skill in your conversations
