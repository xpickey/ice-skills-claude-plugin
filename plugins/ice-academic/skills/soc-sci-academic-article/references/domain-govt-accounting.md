# Domain Knowledge: Government Accounting — บัญชีภาครัฐ

ความรู้พื้นฐานสำหรับการเขียนบทความวิชาการที่เกี่ยวข้องกับระบบบัญชีภาครัฐ IPSAS และ COA

---

## 1. มาตรฐานการบัญชีภาครัฐ (IPSAS)

### ภาพรวม
- IPSAS = International Public Sector Accounting Standards
- ออกโดย IPSASB (International Public Sector Accounting Standards Board)
- ไทยนำ IPSAS มาปรับใช้ตั้งแต่ พ.ศ. 2546 (เกณฑ์คงค้าง)
- กรมบัญชีกลางเป็นผู้กำหนดมาตรฐานบัญชีภาครัฐไทย

### IPSAS ที่สำคัญ

| IPSAS | ชื่อ | สาระสำคัญ |
|---|---|---|
| IPSAS 1 | Presentation of Financial Statements | การนำเสนองบการเงินภาครัฐ |
| IPSAS 5 | Borrowing Costs | ต้นทุนการกู้ยืม |
| IPSAS 9 | Revenue from Exchange Transactions | รายได้จากการแลกเปลี่ยน |
| IPSAS 12 | Inventories | สินค้าคงเหลือ |
| IPSAS 17 | Property, Plant and Equipment | ที่ดิน อาคาร อุปกรณ์ |
| IPSAS 23 | Revenue from Non-Exchange Transactions | รายได้ภาษี/เงินอุดหนุน |
| IPSAS 31 | Intangible Assets | สินทรัพย์ไม่มีตัวตน |
| IPSAS 33 | First-time Adoption | การนำ IPSAS มาใช้ครั้งแรก |
| IPSAS 35 | Consolidated Financial Statements | งบการเงินรวม |

### Accrual Basis vs Cash Basis
- **Cash Basis** — บันทึกเมื่อรับ/จ่ายเงินจริง (ระบบเดิม)
- **Accrual Basis** — บันทึกเมื่อเกิดสิทธิ/ภาระผูกพัน (ระบบปัจจุบัน)
- ไทยเปลี่ยนจาก Cash → Modified Accrual → Full Accrual

## 2. ผังบัญชีภาครัฐ (COA — Chart of Accounts)

### โครงสร้าง COA ภาครัฐไทย
```
Fund Code          → กองทุน/แหล่งเงิน
Department Code    → หน่วยงาน/หน่วยเบิกจ่าย
Program Code       → แผนงาน
Project Code       → โครงการ
Account Code       → บัญชี (สินทรัพย์/หนี้สิน/ทุน/รายได้/ค่าใช้จ่าย)
```

### ประเภทบัญชีหลัก
| หมวด | รหัส | ตัวอย่าง |
|---|---|---|
| สินทรัพย์ | 1XXXX | เงินสด เงินฝากธนาคาร สินทรัพย์ถาวร |
| หนี้สิน | 2XXXX | เจ้าหนี้ เงินรับฝาก หนี้สินระยะยาว |
| ทุน/สินทรัพย์สุทธิ | 3XXXX | ทุนของหน่วยงาน รายได้สูง(ต่ำ)กว่าค่าใช้จ่าย |
| รายได้ | 4XXXX | รายได้จากงบประมาณ รายได้อื่น |
| ค่าใช้จ่าย | 5XXXX | เงินเดือน ค่าจ้าง ค่าวัสดุ ค่าสาธารณูปโภค |

## 3. Cost Accounting ภาครัฐ

### ต้นทุนต่อหน่วยผลผลิต
- หน่วยงานรัฐต้องคำนวณต้นทุนต่อหน่วยผลผลิต/โครงการ
- ตาม มติ ครม. เรื่องการคำนวณต้นทุนผลผลิต
- วิธีการ: Activity-Based Costing (ABC) / Direct + Indirect Cost Allocation

### งบการเงินภาครัฐ
- งบแสดงฐานะการเงิน (Statement of Financial Position)
- งบแสดงผลการดำเนินงานทางการเงิน (Statement of Financial Performance)
- งบแสดงการเปลี่ยนแปลงสินทรัพย์สุทธิ
- งบกระแสเงินสด (Cash Flow Statement)
- หมายเหตุประกอบงบการเงิน

## 4. การตรวจสอบภาครัฐ

### สำนักงานการตรวจเงินแผ่นดิน (สตง.)
- ตรวจสอบการเงินและการบัญชีของทุกหน่วยงานรัฐ
- ตรวจสอบความถูกต้อง / ความมีประสิทธิภาพ / ความคุ้มค่า
- รายงานต่อรัฐสภา

### การตรวจสอบภายใน
- ม.79 พรบ.วินัยการเงินการคลัง 2561 — บังคับทุกหน่วยงาน
- Internal Audit + Internal Control + Risk Management
- มาตรฐาน: COSO Internal Control Framework

## 5. ทฤษฎีที่เกี่ยวข้องสำหรับบทความวิชาการ

- **Institutional Theory** — วิเคราะห์การรับเอา IPSAS/Accrual Basis
- **NPM** — วิเคราะห์การปฏิรูปบัญชีจาก Cash → Accrual
- **Agency Theory** — วิเคราะห์ปัญหาความโปร่งใสในรายงานการเงินภาครัฐ
- **Accountability Theory** — วิเคราะห์ความรับผิดชอบทางการเงิน
- **Legitimacy Theory** — วิเคราะห์การรายงานเพื่อความชอบธรรม
