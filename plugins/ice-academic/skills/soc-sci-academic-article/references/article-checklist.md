# Pre-Submission Checklist — บทความวิชาการ

ใช้ checklist นี้ตรวจทุกครั้งก่อนส่งบทความ ทุกข้อต้องผ่านทั้งหมด

---

## PART 1: คุณภาพวิชาการ (Academic Quality)

- [ ] เป็น Analysis / Criticism / Synthesis จริง (ไม่ใช่ Research Report)
- [ ] ไม่มี Methodology / Population / Sample / Data Collection / Statistical Analysis
- [ ] นำเสนอ/อ้างอิงทฤษฎีหลักอย่างน้อย 2–3 ทฤษฎี
- [ ] อ้างอิง Scholarly Sources ≥ 15 แหล่ง
- [ ] นำเสนออย่างเป็นกลาง: มีทั้งด้านสนับสนุน + ข้อจำกัด/ข้อวิจารณ์
- [ ] มีคุณูปการชัดเจนต่อองค์ความรู้ใน 5 สาขาของวารสาร
- [ ] Word Count ≤ 8,000 คำ (ไม่รวม Abstract + บรรณานุกรม)
- [ ] แหล่งอ้างอิงไม่ hallucinate — ทุกรายการตรวจสอบได้

## PART 2: โครงสร้างและความสมบูรณ์ (Structure & Completeness)

- [ ] **ชื่อเรื่อง**: มีทั้งไทย + อังกฤษ บ่งบอกประเด็นวิเคราะห์ชัดเจน
- [ ] **บทคัดย่อ**: ครบ 4 องค์ประกอบ (ภูมิหลัง + วัตถุประสงค์ + สาระวิเคราะห์ + สรุป)
- [ ] **บทคัดย่อ**: ≤350 คำ (รวมทั้งไทยและอังกฤษ)
- [ ] **Keywords**: 3–5 คำ ทั้ง 2 ภาษา
- [ ] **บทนำ**: ครบ 4 องค์ประกอบ (ภูมิหลัง / วัตถุประสงค์ / ขอบเขต / ประโยชน์)
- [ ] **แนวคิดและทฤษฎี**: มี Theoretical Framework + นิยามศัพท์ชัดเจน
- [ ] **เนื้อหาวิเคราะห์**: วิเคราะห์ผ่านทฤษฎีได้จริง (ไม่ใช่แค่อธิบาย)
- [ ] **เนื้อหาวิเคราะห์**: แบ่ง 3–4 ประเด็น แต่ละประเด็นใช้ CLAIM-THEORY-EVIDENCE-SYNTHESIS
- [ ] **บทสรุป**: มีทั้ง สรุปการวิเคราะห์ + คุณูปการ + ข้อเสนอแนะ
- [ ] **บรรณานุกรม**: APA 7th Edition ถูกต้องทุกรายการ

## PART 3: รูปแบบ (Format)

- [ ] Font: TH SarabunPSK ทั้งเอกสาร
- [ ] ชื่อเรื่องไทย: 18pt Bold จัดกลาง
- [ ] ชื่อเรื่องอังกฤษ: 16pt Bold จัดกลาง
- [ ] ชื่อผู้เขียน: 14pt จัดกลาง (มีเชิงอรรถระบุสังกัด+Email)
- [ ] หัวข้อ: 16pt Bold
- [ ] เนื้อหา: 16pt Regular
- [ ] Margin: บน 3.0 / ล่าง 2.5 / ซ้าย 3.5 / ขวา 2.5 cm
- [ ] Line spacing: 1.0 (single)
- [ ] Indent ย่อหน้าแรก: 1.25 cm
- [ ] เลขหน้า: กลางล่าง
- [ ] ไม่มีข้อมูลระบุตัวตนผู้แต่งใน manuscript (blind review)
- [ ] Properties ของไฟล์ Word ไม่มีชื่อผู้เขียน

## PART 4: ภาษาและการเขียน

- [ ] ภาษาวิชาการระดับทางการ ไม่ใช้ภาษาพูด
- [ ] ศัพท์เทคนิคภาษาอังกฤษใส่วงเล็บครั้งแรก
- [ ] ใช้ศัพท์สม่ำเสมอทั้งบทความ (consistency)
- [ ] ไม่มีประโยคยาวเกินไป (≤3 บรรทัด/ประโยค)
- [ ] มีคำเชื่อมระหว่างย่อหน้าและระหว่างประเด็น

## PART 5: ThaiJO System

- [ ] Account ThaiJO พร้อม
- [ ] ไฟล์เป็น .docx (ไม่ใช่ PDF)
- [ ] เลือกประเภท "บทความวิชาการ" ในระบบ
- [ ] Metadata ครบ: Title / Abstract / Keywords (ไทย+อังกฤษ)
- [ ] Author information กรอกในระบบ (ไม่ใส่ใน manuscript)
