---

## วิธีใช้ Matrix นี้

1. ระบุ **โดเมนเนื้อหา** ของบทความ (คอลัมน์)
2. ดูทฤษฎีที่ใช้ได้ (✓ = เหมาะมาก, ○ = ใช้ได้, — = ไม่เกี่ยว)
3. เลือก ≥2 ทฤษฎี เพื่อมุมมองที่รอบด้าน

## Recommended Combinations (ตัวอย่างการจับคู่ที่ได้ผลดี)

### สำหรับหัวข้อ GFMIS / e-GP
- **Primary**: NPM + Institutional Theory
- **Buddhist blend**: NPM + ทศพิธราชธรรม + Good Governance
- **Tech-focused**: IS Success Model + TAM + NPM

### สำหรับหัวข้อ Digital Government
- **Primary**: Digital Gov Framework + Diffusion of Innovation
- **Governance**: NPG + Collaborative Governance
- **Buddhist blend**: Digital Gov + อปริหานิยธรรม + Good Governance

### สำหรับหัวข้อ FinTech / IFRS9
- **Primary**: Financial Innovation + Agency Theory
- **Risk-focused**: IFRS9 ECL + Basel III/IV
- **Tech**: TAM + Diffusion of Innovation + CMM

### สำหรับหัวข้อบัญชีภาครัฐ
- **Primary**: Institutional Theory + NPM
- **Accountability**: Agency Theory + Good Governance
- **Buddhist blend**: Accountability + ทศพิธราชธรรม

### สำหรับหัวข้อพุทธบูรณาการ
- **Governance**: ทศพิธราชธรรม + Good Governance + Comparative
- **Leadership**: พรหมวิหาร 4 + Transformational Leadership
- **Development**: ไตรสิกขา + HRD + Competency Model
- **Participation**: อปริหานิยธรรม + สังคหวัตถุ + Collaborative Governance

### สำหรับหัวข้อรัฐศาสตร์
- **Primary**: Democracy Theory + Political Culture
- **Decentralization**: Fiscal Federalism + Decentralization + Comparative
- **Buddhist blend**: Political Ethics + ธรรมเป็นโลกบาล + สุจริต 3
