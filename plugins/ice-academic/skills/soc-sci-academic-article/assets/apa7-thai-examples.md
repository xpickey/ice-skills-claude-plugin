## ประเภทที่ 1: หนังสือภาษาไทย

### รูปแบบ
```
ผู้แต่ง. (ปีพิมพ์). ชื่อหนังสือ (ครั้งที่พิมพ์). สำนักพิมพ์.
```

### ตัวอย่าง
```
สุรพล สุยะพรหม และคณะ. (2567). แนวคิด ทฤษฎีและหลักพุทธธรรมทางการจัดการเชิงพุทธ
    รัฐศาสตร์ และรัฐประศาสนศาสตร์. โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย.

พระพรหมคุณาภรณ์ (ป.อ. ปยุตฺโต). (2559). พุทธธรรม ฉบับปรับขยาย (พิมพ์ครั้งที่ 55).
    ผลิธัมม์.

สมยศ นาวีการ. (2563). การบริหารรัฐกิจ (พิมพ์ครั้งที่ 5). สำนักพิมพ์จุฬาลงกรณ์
    มหาวิทยาลัย.
```

## ประเภทที่ 6: พระไตรปิฎก + คัมภีร์บาลี

### รูปแบบบรรณานุกรม
```
มหาจุฬาลงกรณราชวิทยาลัย. (2539). พระไตรปิฎกภาษาไทย ฉบับมหาจุฬาลงกรณราชวิทยาลัย.
    กรุงเทพฯ: มหาจุฬาลงกรณราชวิทยาลัย.
```

### รูปแบบ In-text
- **(ที.ปา. 11/140/167)** — ทีฆนิกาย ปาฏิกวรรค เล่ม 11 ข้อ 140 หน้า 167

## หลักการทั่วไป APA 7th

1. **Hanging indent**: บรรทัดที่ 2 เป็นต้นไป indent 1.27 cm (0.5 inch)
2. **ชื่อวารสาร**: ตัวเอียง (Italic)
3. **ชื่อหนังสือ**: ตัวเอียง (Italic)
4. **DOI**: ถ้ามี ต้องใส่เสมอ (format: https://doi.org/xxxxx)
5. **URL**: ใส่เมื่อเข้าถึงออนไลน์ (ไม่ต้องใส่วันที่เข้าถึง ยกเว้นเนื้อหาเปลี่ยนบ่อย)
6. **เรียงลำดับ**: ไทยก่อน อังกฤษหลัง เรียงตามตัวอักษร
7. **ภาษาไทย**: ใช้ชื่อ-นามสกุลเต็ม (ไม่ย่อ)
8. **ภาษาอังกฤษ**: นามสกุล, อักษรย่อชื่อ. (Last, F. M.)
