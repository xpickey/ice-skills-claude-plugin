# 00 — แผนที่สมรรถนะนักวิจัย วช. (ELO × KSA × สามเสา × สองชั้น)

ไฟล์นี้คือผลการสกัด Key Requirement, Key Functional & Responsibility และ Key Skill จากหลักสูตร "นักวิจัยรุ่นใหม่ วช."
(ปรับปรุง พ.ศ. ๒๕๖๗) ใช้เป็นไม้บรรทัดตรวจว่างานได้มาตรฐานนักวิจัย วช. แล้วหรือยัง และจะ "ยกระดับขั้นสูง" ต่อได้อย่างไร

หลักสูตรวางผลลัพธ์การเรียนรู้ (ELO) ๔ ข้อ กระจายลงสามด้าน **KSA** — Knowledge (ความรู้), Skill (ทักษะ),
Attitude (คุณลักษณะ/ทัศนคติ)

---

## 1. ELO ๔ ข้อ และรหัส KSA รายด้าน (ตามคู่มือหลักสูตร ตารางที่ ๑.๑)

**ELO 1 — พัฒนาโจทย์การวิจัยได้ตามยุทธศาสตร์ชาติและบริบทของกลุ่มเป้าหมาย**
- K1-1 ประยุกต์ใช้หลักจริยธรรมและมาตรฐานการวิจัยเข้ากับการวิจัย
- K1-2 พัฒนาหัวข้อการวิจัยให้สอดคล้องกับแผน ววน. และความต้องการของกลุ่มเป้าหมาย
- S1-1 ใช้เทคโนโลยีดิจิทัล/ปัญญาประดิษฐ์สืบค้นนวัตกรรม บทความวิจัย และทรัพย์สินทางปัญญา
- S1-2 หาช่องว่างของความรู้และพัฒนาโจทย์การวิจัย
- A1-1 จัดสรรบทบาทของคณะผู้วิจัยอย่างมีจริยธรรม / A1-2 มีความคิดริเริ่มสร้างสรรค์และมองเห็นประเด็นการวิจัย

**ELO 2 — ออกแบบการวิจัยได้สอดคล้องกับคำถาม วัตถุประสงค์ และผลผลิต**
- K2-1 ออกแบบการวิจัยให้สอดคล้องกับโจทย์ คำถาม วัตถุประสงค์ และผลผลิต
- K2-2 คัดเลือกเครื่องมือและเลือกใช้สถิติให้สอดคล้องกับรูปแบบการวิจัย
- S2-1 ใช้ซอฟต์แวร์ที่เกี่ยวข้องกับการออกแบบการวิจัยและสถิติ
- A2-1 ทำงานเป็นทีมและผสานแนวคิดเพื่อพัฒนาการออกแบบการวิจัย

**ELO 3 — พัฒนาข้อเสนอการวิจัยเชิงบูรณาการได้สอดคล้องกับยุทธศาสตร์ชาติและบริบท**
- K3-1 ประยุกต์ใช้หลักการและเขียนข้อเสนอการวิจัย
- S3-1 นำเสนอแนวคิดและแลกเปลี่ยนองค์ความรู้กับสมาชิกในกลุ่ม นอกกลุ่ม และผู้ทรงคุณวุฒิ
- A3-1 ใช้จริยธรรม/มาตรฐานเฉพาะด้านในการเขียนข้อเสนอ / A3-2 ไม่คัดลอกและตรวจการลอกเลียนได้ /
  A3-3 รับฟังความเห็นผู้อื่นและไม่ย่อท้อต่อการปรับปรุงแก้ไข

**ELO 4 — วางแนวทางการใช้ประโยชน์และเผยแพร่ผลการวิจัยสู่กลุ่มเป้าหมาย**
- K4-1 อธิบายหลักการและรูปแบบการใช้ประโยชน์จากงานวิจัย
- S4-1 วางแนวทางการใช้ประโยชน์และเส้นทางสู่ผลกระทบตามบริบทของกลุ่มเป้าหมาย / S4-2 ใช้ระบบ NRIIS จัดการ
  โครงการและรายงานการใช้ประโยชน์
- A4-1 เผยแพร่และอ้างอิงผลงานโดยคำนึงถึงหลักจริยธรรมและจรรยาบรรณนักวิจัย

## 2. สามเสาสมรรถนะ (สกัดจาก Course Outline + ELO)

**เสาที่ ๑ — Requirement (ต้องมี): ** บูรณภาพและความซื่อสัตย์ทางวิชาการ, เข้าใจกระบวนทัศน์/ปรัชญาการวิจัย,
รู้ยุทธศาสตร์ชาติ/แผน ววน., มีพื้นฐานในสาขาของตน (OECD field), ยึดจรรยาบรรณนักวิจัย วช. ๙ ข้อ และมาตรฐานการวิจัย,
เคารพสิทธิผู้เข้าร่วมและรับผิดชอบต่อสังคม

**เสาที่ ๒ — Functional & Responsibility (ทำได้): ** กำหนดปัญหา/คำถาม/วัตถุประสงค์/สมมติฐาน, ทบทวน-วิเคราะห์-
สังเคราะห์วรรณกรรมพร้อมอ้างอิง, ออกแบบการวิจัยและเลือกตัวแปร, จัดการ-วิเคราะห์ข้อมูล+สถิติ, เขียนข้อเสนอ+impact
pathway+TRL/SRL, บริหารโครงการและเครือข่าย, เผยแพร่อย่างมีจริยธรรม และทำหน้าที่พี่เลี้ยง/ผู้ประเมิน

**เสาที่ ๓ — Skill (ใช้จริง): ** ตั้งคำถาม/คิดเชิงวิพากษ์, สืบค้น+สังเคราะห์วรรณกรรม+ใช้ AI อย่างรับผิดชอบ,
เลือก/ออกแบบระเบียบวิธี, ออกแบบเครื่องมือเก็บข้อมูล, วิเคราะห์ข้อมูล+เลือกสถิติ+ใช้ซอฟต์แวร์, เขียนเชิงวิชาการ
ด้วยน้ำเสียงตน+ตรวจการลอกเลียน, สื่อสารผลกระทบ+ทำงานเครือข่าย

## 3. การจับคู่กับ ๖ หมวด

```
หมวด ๑ จริยธรรม/มาตรฐาน  → K1-1 A1-1 A3-1 A3-2 A4-1   (ฐานที่รองทุกหมวด)
หมวด ๒ ปรัชญา+โจทย์วิจัย   → ELO1: K1-2 S1-1 S1-2 A1-2
หมวด ๓ ทบทวนวรรณกรรม     → S1-1 (สืบค้น) + สังเคราะห์ + A3-2
หมวด ๔ ออกแบบการวิจัย     → ELO2: K2-1 K2-2 S2-1 A2-1
หมวด ๕ ข้อเสนอ+ผลกระทบ   → ELO3+ELO4: K3-1 K4-1 S3-1 S4-1 A3-3
หมวด ๖ เครือข่ายบูรณาการ   → S4-2 (NRIIS) + A2-1 + การใช้ประโยชน์ร่วม
```

## 4. สองชั้นของสมรรถนะ (Baseline → Advanced)

skill นี้สร้างนักวิจัยที่ "ทำได้ตามหลักสูตร" แล้วต่อยอดสู่ "งานวิจัยขั้นสูง" ตารางเทียบสองชั้น

| ด้าน | ชั้นมาตรฐาน (ตามหลักสูตร) | ชั้นขั้นสูง (เหนือหลักสูตร) |
|------|---------------------------|------------------------------|
| โจทย์ | ตั้งปัญหา/คำถาม/วัตถุประสงค์ที่ตรงกัน | วางบนทฤษฎี ใช้ bibliometrics/foresight หาแนวโน้มและ gap |
| วรรณกรรม | สังเคราะห์เชิงประเด็น + อ้างอิงถูก | systematic review/PRISMA, meta-analysis, สืบค้นสิทธิบัตร |
| ออกแบบ | เลือกวิธี/ตัวอย่าง/สถิติให้ตรงโจทย์ | SEM/PLS-SEM, ออกแบบทดลอง/กึ่งทดลอง, mixed-methods design, power analysis |
| ข้อเสนอ | เขียนได้ครบ + impact pathway + TRL/SRL | theory of change, logic model, pre-registration, แผนงบ/ความเสี่ยงเชิงลึก |
| เผยแพร่ | อ้างอิงมีจริยธรรม ไม่ลอกเลียน | ตีพิมพ์ Scopus/WoS, open access, reproducible/open-science workflow |
| เครือข่าย | ทำงานกลุ่ม วางกิจกรรมต่อเนื่อง | transdisciplinary consortium, ข้อตกลง IP/data, ทุนนานาชาติ |

เทคนิคชั้นขั้นสูงรวมอยู่ใน `11-advanced-toolkit.md`

## 5. เกณฑ์ตรวจคุณภาพแบบ KSA (ใช้กับ deliverable ทุกชิ้น)

- **Knowledge** — ข้อสรุปอิงทฤษฎี/วรรณกรรมจริง ระบุกระบวนทัศน์ และเชื่อมยุทธศาสตร์ชาติ/ความต้องการกลุ่มเป้าหมายได้
- **Skill** — วิธีที่เลือกสอดคล้องคำถาม ทำซ้ำได้ (reproducible) มีร่องรอยการตัดสินใจ ใช้เครื่องมือ/ซอฟต์แวร์เหมาะสม
- **Attitude** — เคารพสิทธิผู้เข้าร่วม เปิดเผยผลประโยชน์ทับซ้อน ให้เครดิตผู้อื่น รับฟังและปรับตาม feedback

เกณฑ์คะแนนอ้างอิง: ผ่านดีมาก ≥ ร้อยละ ๘๐ / ผ่านดี ร้อยละ ๖๐-๘๐ / ไม่ผ่าน < ร้อยละ ๖๐
