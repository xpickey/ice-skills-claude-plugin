# 05 — การเขียนข้อเสนอการวิจัยและเส้นทางสู่ผลกระทบ (หมวด ๕)

ELO ที่เกี่ยวข้อง: ELO 3 (เขียนข้อเสนอเชิงบูรณาการ) ผ่าน K3-1, S3-1, A3-1/A3-3 และ ELO 4 (วางแนวทางใช้ประโยชน์/เผยแพร่)
ผ่าน K4-1, S4-1, S4-2, A4-1 ผลงานหลักของหลักสูตรคือ **บทสรุปข้อเสนอโครงการ ไม่เกิน ๓,๐๐๐ คำ**

---

## 1. ความรู้: โครงสร้างข้อเสนอการวิจัย

ทุกส่วนต้องสอดคล้องเป็นเส้นเดียวกับโจทย์ (ดู `02-research-problem.md`)

1. ชื่อเรื่องและบทคัดย่อ/บทสรุปผู้บริหาร
2. ความเป็นมาและความสำคัญ (rationale & background) — ปิดท้ายด้วยช่องว่างที่จะเติม + เชื่อมยุทธศาสตร์ชาติ
3. คำถาม วัตถุประสงค์ และสมมติฐาน/ข้อสันนิษฐาน
4. การทบทวนวรรณกรรมและกรอบแนวคิด (พร้อมตัวแปร)
5. ระเบียบวิธีวิจัย — แบบแผน, ประชากร/ตัวอย่าง, เครื่องมือ, การเก็บ-วิเคราะห์ข้อมูล, สถิติ
6. จริยธรรมและมาตรฐานการวิจัยที่เกี่ยวข้อง (มนุษย์/สัตว์/ชีวภาพ/PDPA)
7. แผนงาน ระยะเวลา (Gantt) งบประมาณ และผลผลิต/ผลลัพธ์
8. เส้นทางสู่ผลกระทบและ TRL/SRL (ดูข้อ ๒-๓)
9. เอกสารอ้างอิงและภาคผนวก (เครื่องมือ, แบบยินยอม)

## 2. ความรู้: แผนที่สู่ผลกระทบ (Impact Pathway)

หัวใจที่หลักสูตรเน้นเป็นพิเศษ คือ ลากเส้นจากกิจกรรมวิจัยถึงการเปลี่ยนแปลงจริง

```
Input  →  Activity  →  Output  →  Outcome  →  Impact
ทุน/คน    ทำอะไร      ผลผลิต      การเปลี่ยน      ผลกระทบระยะยาว
เครื่องมือ  (วิจัย)     ที่จับต้องได้  พฤติกรรม/      ต่อเศรษฐกิจ/สังคม/
                                  การใช้ประโยชน์  สิ่งแวดล้อม
```

ระบุชัดว่า **ใครคือผู้ใช้ประโยชน์ (beneficiaries/users)** — ภาคอุตสาหกรรม ชุมชน นโยบาย หรือวิชาการ —
และผลกระทบจะวัดด้วยตัวชี้วัดใด เชื่อมกับยุทธศาสตร์ชาติและความต้องการของพื้นที่/ประเทศ

## 3. ความรู้: TRL และ SRL

- **TRL (Technology Readiness Level)** ๑-๙ จากหลักการพื้นฐาน (๑) ถึงใช้งานจริงเต็มรูปแบบ (๙) ระบุว่างานเริ่ม
  TRL ใดและจะดันถึง TRL ใดเมื่อจบโครงการ
- **SRL (Societal Readiness Level)** ๑-๙ สำหรับงานสังคม/มนุษยศาสตร์ บอกความพร้อมของแนวทางสู่การใช้ประโยชน์ทางสังคม
- หลักสูตรให้เลือกใช้ TRL หรือ SRL ตามบริบทของกลุ่มเป้าหมาย (วัตถุประสงค์ข้อ ๒.๓.๓)

## 4. ทักษะ: ใช้ระบบ NRIIS (S4-2)

ผู้เข้าอบรมต้องมีข้อมูลในระบบ NRIIS (`https://nriis.go.th`) ฝึกใช้ NRIIS ในการจัดการโครงการและรายงานการใช้ประโยชน์
กลุ่มวิจัยที่มีโจทย์พร้อมมักยื่นขอทุนผ่าน NRIIS ทันทีหลังจบอบรม

## 5. ทักษะ: เขียนบทสรุปข้อเสนอ ≤ ๓,๐๐๐ คำ (ผลงานหลัก)

กระชับแต่ครบ ลากเส้นชัดจาก ปัญหา → คำถาม/วัตถุประสงค์ → วิธี → ผลผลิต → เส้นทางสู่ผลกระทบ
เกณฑ์ที่กรรมการมองหา: ความใหม่, ความเป็นไปได้, ความสอดคล้องภายใน, จริยธรรม, และความชัดของผลกระทบต่อผู้ใช้

## 6. ทักษะ: เขียนให้น่าเชื่อถือและมีจุดยืน

- เปิดแต่ละส่วนด้วยประเด็นหลัก แล้วขยาย ไม่ใช่เล่าไปเรื่อย
- มีจุดยืนเชิงวิชาการ ("ผู้วิจัยเห็นว่า...") โดยอิงหลักฐาน
- ยอมรับข้อจำกัดอย่างตรงไปตรงมา ความซื่อตรงเพิ่มความน่าเชื่อถือ
- เขียนด้วยน้ำเสียงของตนเองและตรวจการลอกเลียนก่อนส่ง (ดู `08-writing-integrity.md`)

## 7. เจตคติ: นำเสนอและรับฟัง (S3-1, A3-3)

หลักสูตรให้ "นำเสนอข้อเสนอฉบับย่อ" และแลกเปลี่ยนในกลุ่ม/กับผู้ทรงคุณวุฒิ ฝึกสรุปสามนาที (จุดยืน-วิธี-ผลกระทบ)
และปรับข้อเสนอตาม feedback อย่างไม่ย่อท้อ — เป็นส่วนหนึ่งของคะแนน Attitude

## 8. ชั้นขั้นสูง (Beyond)

- **Theory of Change** — เขียนสมมติฐานเชิงเหตุผล (if-then) จาก activity ถึง impact พร้อมเงื่อนไขจำเป็น
- **Logic model** ที่ละเอียดและ **KPI/ตัวชี้วัดผลกระทบ** ที่วัดได้จริง
- **Pre-registration / registered report** เพื่อความโปร่งใส
- **Risk register และแผนบริหารงบประมาณเชิงลึก** สำหรับข้อเสนอทุนใหญ่/นานาชาติ
- จัดทำ **data management plan (DMP)** และแผน open access ในข้อเสนอ
