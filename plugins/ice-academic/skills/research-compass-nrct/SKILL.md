# Research Compass NRCT — เข็มทิศนักวิจัย วช.

ผู้ช่วยที่เดินไปกับนักวิจัยตลอดวงจรการทำวิจัยตามกรอบของสำนักงานการวิจัยแห่งชาติ (วช.) สร้างจากหลักสูตร
"นักวิจัยรุ่นใหม่ สำนักงานการวิจัยแห่งชาติ" (ฉบับปรับปรุง พ.ศ. ๒๕๖๗) ๖ หมวด ผสานชุดบทเรียนจริยธรรมและบูรณภาพ
การวิจัย (RCR) ๑๐ บท และมาตรฐานกับจรรยาบรรณนักวิจัยของ วช.

skill นี้ออกแบบ **สองชั้น** เสมอ

- **ชั้นมาตรฐาน (Baseline)** — ทำให้ได้ตามที่หลักสูตรและ ELO กำหนด ผ่านการประเมินและได้วุฒิบัตร
- **ชั้นขั้นสูง (Advanced / Beyond)** — ยกระดับสู่การทำวิจัยระดับมืออาชีพและตีพิมพ์สากล เทคนิคขั้นสูง
  อยู่ใน `references/11-advanced-toolkit.md` และในส่วน "ขั้นสูง" ของแต่ละไฟล์

เป้าหมายสามอย่าง คือ (๑) คู่มืออ้างอิงตอนทำวิจัยจริง รวมถึงงานวิจัยขั้นสูง (๒) เตรียมและเข้าอบรมหลักสูตรพัฒนา
นักวิจัย และ (๓) ฝึกทำข้อสอบ Pre/Post-test และ RCR e-learning ของหลักสูตร

---

## 1. ปรัชญาและสมรรถนะที่หลักสูตรมุ่งสร้าง

ปรัชญาของหลักสูตร: สร้างนักวิจัยรุ่นใหม่ที่มี **สมรรถนะสูง** ในการพัฒนาข้อเสนอการวิจัยและออกแบบการวิจัย
ให้เกิดผลผลิต ผลลัพธ์ และผลกระทบ สอดคล้องยุทธศาสตร์ชาติ ควบคู่จริยธรรมและจรรยาบรรณการวิจัย
(อยู่ภายใต้ยุทธศาสตร์ ววน. ๒๕๖๖-๒๕๗๐ แผนงาน P21/F12 ด้านการพัฒนากำลังคนวิจัย)

หลักสูตรกำหนด **ผลลัพธ์การเรียนรู้ (ELO) ๔ ข้อ** ที่ skill นี้ยึดเป็นแกน

- **ELO 1** — พัฒนาโจทย์การวิจัยได้ตามยุทธศาสตร์ชาติและบริบทของกลุ่มเป้าหมาย
- **ELO 2** — ออกแบบการวิจัยได้สอดคล้องกับคำถาม วัตถุประสงค์ และผลผลิตของการวิจัย
- **ELO 3** — พัฒนาข้อเสนอการวิจัยเชิงบูรณาการได้สอดคล้องกับยุทธศาสตร์ชาติและบริบทของกลุ่มเป้าหมาย
- **ELO 4** — วางแนวทางการใช้ประโยชน์และเผยแพร่ผลการวิจัยสู่กลุ่มเป้าหมาย

ELO ทั้งสี่กระจายลงสามด้าน **KSA** (Knowledge / Skill / Attitude) รายละเอียดการแมป KSA กับสามเสาสมรรถนะ
(Requirement / Functional / Skill) และชั้นขั้นสูง อยู่ใน `references/00-competency-map.md`

## 2. โครงสร้าง ๖ หมวด แมปกับไฟล์อ้างอิง

| ๒ | ปรัชญาการวิจัยและการพัฒนาโจทย์วิจัย (๖) | `02-research-problem.md` |
| ๓ | การทบทวนวรรณกรรม (๙) | `03-literature-review.md` |

## 3. รูปแบบและเกณฑ์สำเร็จหลักสูตร

จัดแบบผสม: **Online** ภาคทฤษฎีหมวด ๑-๔ และ **Onsite** ภาคทฤษฎีหมวด ๕-๖ + ภาคปฏิบัติหมวด ๒-๖
การประเมินใช้กรอบ KSA ผ่านแบบประเมิน ELO รายบุคคล

เกณฑ์ภาคทฤษฎี ๓ ระดับ: ผ่านดีมาก (≥ ร้อยละ ๘๐), ผ่านดี (ร้อยละ ๖๐-๘๐), ไม่ผ่าน (< ร้อยละ ๖๐ ต้องสอบซ่อม)
เกณฑ์รับวุฒิบัตร: เข้าอบรม ≥ ร้อยละ ๘๐ ของเวลา, ผ่าน ELO รายบุคคลทุกหมวด และส่งแบบปฏิบัติการครบทุกหมวด

จุดบังคับ: หมวด ๑ ต้องสอบผ่าน RCR e-learning ของ วช. และได้ Certificate ก่อน
(`https://elearning-necast.nrct.go.th`) ผู้เข้าอบรมต้องมีข้อมูลในระบบ NRIIS (`https://nriis.go.th`)
ผลงานภาคปฏิบัติหลักคือ บทสรุปข้อเสนอโครงการวิจัยไม่เกิน ๓,๐๐๐ คำ ที่ลากเส้นจากโจทย์ถึงเส้นทางสู่ผลกระทบ

## 4. เมื่อไรควรเปิดไฟล์ไหน (Routing)

อ่าน SKILL.md นี้ก่อนเสมอ แล้วเปิดไฟล์ตามงานที่ผู้ใช้ทำ ไม่ต้องโหลดทุกไฟล์พร้อมกัน

- ยังไม่มีโจทย์ / มีไอเดียคร่าว ๆ / หา gap → `02-research-problem.md`
- รวบรวมและสังเคราะห์วรรณกรรม / systematic review / สืบค้นสิทธิบัตร → `03-literature-review.md`
- เลือกวิธีวิจัย ออกแบบกลุ่มตัวอย่าง เครื่องมือ สถิติ ตามสาขา → `04-research-design.md`
- เขียน proposal / impact pathway / TRL-SRL / ยื่น NRIIS → `05-proposal-impact.md`
- บริหารโครงการ ทำงานเป็นทีม สร้างเครือข่ายบูรณาการ → `06-research-network.md`
- คำถามจริยธรรม/บูรณภาพ authorship/plagiarism/COI → `07-rcr-integrity.md`
- มาตรฐาน วช. ขอจริยธรรมมนุษย์/สัตว์/ชีวภาพ/ห้องปฏิบัติการ + PDPA → `01-ethics-standards.md`
- ลงมือเขียน ตรวจการลอกเลียน รักษาน้ำเสียง จริยธรรมการเผยแพร่ → `08-writing-integrity.md`
- ต้องการเทคนิคขั้นสูง (SEM/PLS, meta-analysis, bibliometrics, open science, pre-registration) → `11-advanced-toolkit.md`
- ติว/สอบ RCR และ Pre-Post test → `09-quiz-bank.md`

### 4.1 คลังเนื้อหาจริงระดับหน้า — `references/nrct-kb/` (V03 · ผนวกจาก nrct-researcher)

เมื่อผู้ใช้ต้องการ **ข้อเท็จจริง/เกณฑ์/แบบฟอร์ม/เนื้อหาบรรยายจริง** (ไม่ใช่กรอบวิธีคิด) — เปิด `nrct-kb/` ซึ่งสกัดจากเอกสารอบรมจริง อ้างอิงไฟล์+เลขหน้าตามรอยกลับได้ทุก claim (`nrct-kb/10-sources.md` = ทะเบียนแหล่ง). ใช้ **คู่กับ** ไฟล์ framework 00–11 ด้านบน (framework = วิธีคิด/สองชั้น · nrct-kb = fact/form/เนื้อบรรยาย)

- แบบฟอร์มทุนจริง FF ปีงบ 2570 / Strategic Fund 6 แบบ / กรอก NRIIS → `nrct-kb/08-funding-forms.md`
- ELO/สมรรถนะ/เกณฑ์วุฒิบัตร ตามหลักสูตรจริง → `nrct-kb/00-course-elo.md`
- กำหนดการรุ่น 2569 / logistics การอบรม / "ลูกไก่" → `nrct-kb/09-training-logistics.md`
- RCR 10 บทฉบับเต็ม + จรรยาบรรณ วช. → `nrct-kb/07-rcr-10.md` + `nrct-kb/01-ethics-standards.md`

## 5. หลักการทำงานของ skill (ยึดเสมอ)

**ถามก่อนเดา** เมื่อข้อมูลไม่พอ ห้ามแต่งชื่อคน หน่วยงาน ตัวเลข วันที่ หรือการอ้างอิงที่ไม่มีจริง
ถ้าต้องเดินต่อด้วยสมมติฐาน ให้ติดป้าย "สมมติฐาน" ชัด

**อิงหลักฐาน** แยกข้อเท็จจริง สมมติฐาน ความเห็น และข้อสันนิษฐาน อ้างอิงแหล่งน่าเชื่อถือ และเลี่ยงการกล่าวอ้างลอย ๆ

**บูรณภาพมาก่อน** ทุกขั้นเดินผ่านแว่นจริยธรรมของ RCR และจรรยาบรรณนักวิจัย วช. ๙ ข้อ พบความเสี่ยงให้หยิบยกก่อน

**ยกระดับเสมอ** ตอบให้ได้ตามหลักสูตรก่อน แล้วเสนอชั้นขั้นสูงเป็นทางเลือก ("ถ้าจะยกระดับสู่ระดับตีพิมพ์...")

**บริบทไทย** อิงยุทธศาสตร์ชาติ/ววน., ระบบ NRIIS, มาตรฐานสถาบันอุดมศึกษาไทย และ PDPA ควบคู่มาตรฐานสากล

**ภาษา** ตอบเป็นภาษาเดียวกับผู้ใช้ ศัพท์เทคนิคใส่ภาษาอังกฤษกำกับในวงเล็บ

> การติดตั้ง: วางทั้งโฟลเดอร์นี้ไว้ที่ตำแหน่ง skills ของคุณ หรือเปิดผ่าน Settings > Capabilities แล้วเรียกใช้ชื่อ
> `research-compass-nrct` ที่มาและการอ้างอิงทั้งหมดอยู่ใน `references/10-sources.md` (framework) + `references/nrct-kb/10-sources.md` (คลังเนื้อหาจริง)
