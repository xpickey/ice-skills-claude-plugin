## 2. Chapter 2 Outline (เลือกตามชื่อหัวข้อ)

### 2.1 หัวข้อมี "บูรณาการ/ประยุกต์" → กรอบ = 2.7

```
๒.๑ หลักพุทธธรรม (X2) — Pure ห้ามผสม
๒.๒ แนวคิดเกี่ยวกับการบูรณาการ / ประยุกต์
๒.๓ แนวคิดและทฤษฎี IV1 (≥ 1 หน้า + ≥ 5 อ้างอิง)
๒.๔ แนวคิดและทฤษฎี IV2 (ถ้ามี)
๒.๕ แนวคิดและทฤษฎี DV
๒.๖ งานวิจัยที่เกี่ยวข้อง
   ๒.๖.๑ งานวิจัยเกี่ยวกับหลักธรรม (≥ 5 เรื่อง — ป.เอก เท่านั้น)
   ๒.๖.๒ งานวิจัยเกี่ยวกับ IV
   ๒.๖.๓ งานวิจัยเกี่ยวกับ DV
๒.๗ กรอบแนวคิด ⭐
   - Part 1: บทเกริ่น (Why-this-Theory)
   - Part 2: แผนภาพ + ลูกศร IV → DV
```

### 2.2 หัวข้อไม่มี "บูรณาการ/ประยุกต์" → กรอบ = 2.6

```
๒.๑ หลักพุทธธรรม (X2)
๒.๒ แนวคิดและทฤษฎี IV1
๒.๓ แนวคิดและทฤษฎี IV2 (ถ้ามี)
๒.๔ แนวคิดและทฤษฎี DV
๒.๕ งานวิจัยที่เกี่ยวข้อง
๒.๖ กรอบแนวคิด ⭐
```

> **หมายเหตุงานท่าน:** "การพัฒนาสมรรถนะเชิงพุทธ..." = ไม่มี "บูรณาการ/ประยุกต์" → ใช้ Outline 2.2 → กรอบ = 2.6

## 3. Common Mistakes เฉพาะบทที่ 2

โหลดจาก `03-literature-review.md` §8 — 23 CPs

### Top Critical CPs

- **CP-2:** มจร อ้างอิง < 60%
- **CP-21:** ไม่มีทบทวนทฤษฎี IV1
- **CP-25:** Theory-Variable Mismatch
- **CP-30:** < 5 แหล่งต่อตัวแปร
- **CP-31:** แก้ชื่อทฤษฎีจากต้นทาง
- **CP-33:** งานวิจัยพุทธธรรมไม่ตรงประเด็น
- **CP-35:** กรอบไม่นิ่ง
- **CP-42:** ขาด Why-this-Theory
- **CP-46:** อ้างพระไตรปิฎกผิด
