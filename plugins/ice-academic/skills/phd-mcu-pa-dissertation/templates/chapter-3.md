## 2. Chapter 3 Outline (Mixed Methods Default)

```
บทที่ ๓ วิธีดำเนินการวิจัย

๓.๑ รูปแบบการวิจัย (Mixed Methods Sequential Explanatory)
   - Quant เป็นตัวนำ / Qual เป็นตัวสนับสนุน
   - การเก็บข้อมูล Quant ก่อน → Qual ตามมา

๓.๒ การวิจัยเชิงปริมาณ
   ๓.๒.๑ ประชากรและกลุ่มตัวอย่าง
      - ประชากร: ระบุ N + อ้างอิง
      - กลุ่มตัวอย่าง: Yamane (N>10K) / Krejcie & Morgan (N<10K)
      - วิธีสุ่ม: Stratified / Cluster / Purposive
   ๓.๒.๒ เครื่องมือที่ใช้ในการวิจัย
      - แบบสอบถาม 4 ตอน
      - 5-Point Likert Scale (1.00-5.00)
      - IOC ≥ 0.50 (ผู้เชี่ยวชาญ 5 + ภายนอก ≥ 2)
      - Try out 30 ชุด → Cronbach's α ≥ 0.70
   ๓.๒.๓ การเก็บรวบรวมข้อมูล
      - วิธีเก็บ + ระยะเวลา + Response Rate ≥ 70%
   ๓.๒.๔ การวิเคราะห์ข้อมูล
      - Descriptive: Frequency, %, Mean, SD
      - Inferential: t-test, ANOVA, Regression, Path Analysis
      - SEM (Optional, Sample ≥ 200)

๓.๓ การวิจัยเชิงคุณภาพ
   ๓.๓.๑ ผู้ให้ข้อมูลสำคัญ (Mixed: ≥ 17 / Qual: ≥ 25)
      - 5 กลุ่ม: นโยบาย / ปฏิบัติงาน / รับบริการ / นักวิชาการ / พระพุทธศาสนา
      - 3-4 คน/กลุ่ม สมดุล
   ๓.๓.๒ เครื่องมือ
      - แบบสัมภาษณ์เชิงลึก (Semi-structured)
      - แบบบันทึก FGD (8-12 + อาจารย์ ≥ 1)
      - IOC ≥ 0.50
   ๓.๓.๓ การเก็บรวบรวมข้อมูล
   ๓.๓.๔ การวิเคราะห์ข้อมูล
      - Content Analysis / Thematic Analysis
      - Coding + Triangulation

๓.๔ จริยธรรมการวิจัย (Optional)
```

## 4. Quick Audit

✅ Mixed Methods Sequential Explanatory ระบุชัด
✅ ลำดับสารบัญ ↔ การเก็บข้อมูล ตรงกัน
✅ ตัด "เชิงปริมาณ/คุณภาพ" ใน sub-section
✅ ประชากรชัด + N + อ้างอิง
✅ สูตรกลุ่มตัวอย่างเหมาะกับ N
✅ ผู้ให้ข้อมูลสำคัญ ≥ 17 (Mixed)
✅ ผู้ทรงคุณวุฒิ FGD 8-12 + ไม่ซ้ำ + อาจารย์ ≥ 1
✅ ผู้เชี่ยวชาญ 5 + ภายนอก ≥ 2 + 3 ด้าน
✅ มาตรวัด 5 ระดับครบ
✅ Try out 30 ชุด
✅ Cronbach α ≥ 0.70
✅ สถิติระบุชัด (Descriptive + Inferential)
