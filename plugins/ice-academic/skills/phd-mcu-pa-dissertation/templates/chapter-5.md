## 2. Chapter 5 Outline

```
บทที่ ๕ สรุป อภิปรายผลและข้อเสนอแนะ

๕.๑ สรุปผลการวิจัย
   - สรุปตามวัตถุประสงค์แต่ละข้อ (1-3)
   - มิให้แยกสรุป Quant/Qual — รวมตามวัตถุประสงค์

๕.๒ อภิปรายผลการวิจัย
   - อภิปรายเป็นข้อ ๆ ตามผลตามวัตถุประสงค์
   - หรือ ตามตัวแปรที่ศึกษา / องค์ความรู้ใหม่
   - เชื่อมโยงกับงานวิจัยที่อ้างไว้ในบทที่ ๒
     (สอดคล้อง/ไม่สอดคล้องกับใคร อย่างไร)
   - ระบุว่า "ค้นพบอะไร" + "เหตุใดจึงเป็นเช่นนั้น"
   - อ้างเฉพาะ ป.เอก หรือบทความวิจัย (ห้าม ป.โท)

๕.๓ ข้อเสนอแนะ
   ๕.๓.๑ ข้อเสนอแนะเชิงนโยบาย
      - มาจาก Quant ค่าเฉลี่ยต่ำ + Qual ที่ผู้ให้สัมภาษณ์เสนอ
      - เรื่องใหญ่ — สำหรับหน่วยงาน/บุคคลกำหนดนโยบาย
      - จำนวนข้อไม่มาก แต่ครอบคลุม

   ๕.๓.๒ ข้อเสนอแนะเชิงปฏิบัติการ
      - ละเอียดกว่า ๕.๓.๑
      - จำนวนข้อมากกว่า ๕.๓.๑
      - นำไปสู่การปฏิบัติได้จริง

   ๕.๓.๓ ข้อเสนอแนะเพื่อการวิจัยครั้งต่อไป
      - เนื้อหาใกล้เคียงที่ยังไม่ครอบคลุม
      - ระเบียบวิธีวิจัยที่ควรใช้
      - สั้น กะทัดรัด
```

## 3. กฎเหล็ก สำหรับบทที่ 5 (CRITICAL)

### 3.1 อ้าง ป.เอก เท่านั้นใน Discussion
- ✗ **ห้าม** อ้างวิทยานิพนธ์ ป.โท
- ✓ ดุษฎีนิพนธ์ ป.เอก / บทความวิจัย peer-reviewed

### 3.2 อภิปรายเชื่อมบทที่ 2
- ทุกประเด็น → เชื่อมงานวิจัยใน 2.6
- ระบุ "สอดคล้อง/ไม่สอดคล้องกับใคร อย่างไร"

### 3.3 ข้อเสนอแนะมาจากผลจริง
- ✗ **ห้าม** เสนอแนะลอย ๆ
- ✓ ๕.๓.๑ จาก Quant ต่ำ + Qual ผู้ให้สัมภาษณ์เสนอ

### 3.4 ๕.๓.๒ ละเอียดกว่า ๕.๓.๑
- ๕.๓.๑ Strategic level (เรื่องใหญ่)
- ๕.๓.๒ Operational level (รายละเอียด — มากกว่า)
