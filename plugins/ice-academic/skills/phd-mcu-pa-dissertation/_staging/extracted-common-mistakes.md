## Pattern ที่ปรากฏซ้ำหลายครั้ง (Critical Themes)

1. **Variable Consistency Chain** — ปรากฏ 6+ ครั้ง — ตัวแปรต้องตรงกันจาก ขอบเขต → สมมติฐาน → นิยาม → กรอบ → เครื่องมือ → ผล
2. **5 แหล่งต่อตัวแปร** — ปรากฏ 3+ ครั้ง — citation per variable
3. **Theory Source Attribution** — ปรากฏ 4+ ครั้ง — ทุกตัวแปรต้องมีที่มาจริง
4. **Buddhist Integration ห้ามเป็นเปลือก** — ปรากฏ 4+ ครั้ง — ต้องเชื่อมโยงและให้เหตุผล "Why this Dhamma"
5. **MCU Manual Compliance** — ปรากฏ 5+ ครั้ง — คู่มือเป็น Single Source of Truth
6. **Editorial Polish** — ปรากฏ 5+ ครั้ง — "แสดง", "เชิงปริมาณ" ใน sub, "หมายถึง", "และทฤษฎี"
