## 3. Section A — Typography & Page Setup

### 3.1 Font (ตามคู่มือ มจร หน้า 123)

**บังคับ:** TH SarabunPSK เพียงแบบเดียวเท่านั้น (ทั้งภาษาไทยและภาษาอังกฤษ)

**ขนาดและรูปแบบตามแต่ละส่วน:**

| ส่วน | ขนาด | รูปแบบ | ตำแหน่ง |
|-----|------|--------|---------|
| ปก | ตามรูปแบบปก | — | — |
| บทที่ + ชื่อบท | 20 pt | ตัวหนา | กลางหน้า |
| หัวข้อสำคัญ (1.1) | 18 pt | ตัวหนา | ชิดซ้าย |
| หัวข้อย่อย (1.1.1) | 16 pt | ตัวหนา | เยื้องซ้าย 0.7" |
| เนื้อหา + เลขหน้า | 16 pt | ตัวธรรมดา | — |
| เชิงอรรถ | 14 pt | ตัวธรรมดา | indent 0.7" |
| ตาราง/แผนภาพ | อาจเล็กลง | ปกติ | ในกรอบหน้า |

### 3.2 Page Setup (คู่มือ มจร หน้า 123-124)

**กระดาษ:**
- ขนาด A4 (210 × 297 mm)
- เนื้อกระดาษ: ปอนด์ ไม่มีบรรทัด
- น้ำหนัก: 80 แกรม
- พิมพ์เพียงหน้าเดียว

**Margins (ระยะขอบ):**
- ด้านบน: **1.5 นิ้ว (3.81 cm)**
- ด้านซ้าย: **1.5 นิ้ว (3.81 cm)**
- ด้านล่าง: **1 นิ้ว (2.54 cm)**
- ด้านขวา: **1 นิ้ว (2.54 cm)**
- หัวกระดาษ: ห่างจากขอบบน **1 นิ้ว (2.54 cm)**

**Top of Chapter (เริ่มบทใหม่):**
- ห่างจากขอบบน **2 นิ้ว (5.08 cm)**
- พิมพ์ "บทที่..." ตรงกลางหน้า
- พิมพ์ชื่อบทบรรทัดต่อมา ขนาด 20 pt

### 3.3 Paragraph Spacing (คู่มือ มจร หน้า 125-126)

**First Line Indent:**
- บรรทัดแรกทุกย่อหน้า: **0.7 นิ้ว (1.75 cm)**
- เริ่มที่ตัวอักษรที่ 8

**Paragraph Spacing:**
- Before: **6 pt** (ทุกย่อหน้า)
- After: 0 pt
- Line Spacing: **Single (1.0)** ภายในย่อหน้าเดียวกัน

**Justify:**
- ทุกย่อหน้า: Justify both sides (ชิดสองข้าง)
- ห้าม จัดกลาง / ชิดซ้ายอย่างเดียว ในเนื้อหา

### 3.4 Title Wrapping Rules

- ชื่อบทยาว > 48 ตัวอักษร หรือ > 1 บรรทัด → แบ่งเป็น 2-3 บรรทัดแบบสามเหลี่ยมหัวกลับ (พีระมิดคว่ำ)
- **ห้ามตัดคำ** เช่น "มหา-จุฬาลงกรณ์ราชวิทยาลัย" ห้ามแบ่ง — ต้องไปบรรทัดใหม่ทั้งคำ
- ห้ามขีดเส้นใต้ชื่อบท

## 5. Section C — Table of Contents (สารบัญ)

### 5.1 องค์ประกอบที่ต้องมี

1. **สารบัญ (เนื้อหา)** — บทที่ 1-5 + ภาคผนวก
2. **สารบัญตาราง** — ทุกตารางในเล่ม
3. **สารบัญแผนภาพ** — ทุกแผนภาพในเล่ม

### 5.2 รูปแบบ (คู่มือ มจร หน้า 5-7)

**สารบัญ:**
- ขึ้นหน้าใหม่แยกต่างหากจากเนื้อหา
- ครอบ 5 บท + บรรณานุกรม + ภาคผนวก + ประวัติผู้วิจัย
- หัวข้อย่อยตามรูปแบบมาตรฐาน
- หัวข้อในสารบัญ ↔ หัวข้อในเนื้อหา **ตรงกัน 100%**
- เลขหน้าในสารบัญ ↔ เลขหน้าจริง **ตรงกัน 100%**

**สารบัญตาราง:**
- เลขลำดับตาราง = เลขบท + ลำดับ (เช่น ตารางที่ 2.1, 2.2)
- ชื่อตารางยาว ≥ 2 แถว → เลขลำดับอยู่แถวแรก / หมายเลขหน้าอยู่แถวสุดท้าย

**สารบัญแผนภาพ:**
- เลขลำดับแผนภาพ = เลขบท + ลำดับ
- รูปแบบเดียวกับสารบัญตาราง

### 5.3 TOC Audit

✅ สารบัญขึ้นหน้าใหม่
✅ ครอบ 5 บท + ส่วนแนบครบ
✅ หัวข้อตรงกับเนื้อหา 100%
✅ เลขหน้าตรงกับเนื้อหา 100%
✅ มีสารบัญตาราง (ถ้ามีตาราง)
✅ มีสารบัญแผนภาพ (ถ้ามีแผนภาพ)

## 6. Section D — Symbols / Abbreviations (คำอธิบายสัญลักษณ์และคำย่อ)

### 6.1 รูปแบบ (คู่มือ มจร หน้า 8)

- **เฉพาะคำย่อพระไตรปิฎก** ที่ปรากฏในเชิงอรรถเท่านั้น
- ครอบทุกแห่งที่อ้าง
- เฉพาะที่อ้างจริงในเล่ม
- **แยกเป็น 2 ฉบับ:** ภาษาบาลี + ภาษาไทย
- คำอธิบายฉบับบาลี → พิมพ์แบบภาษาบาลี
- คำอธิบายฉบับไทย → พิมพ์แบบภาษาไทย

### 6.2 ตัวอย่างคำย่อพระไตรปิฎก

```
ที.ม. (บาลี)  =  สุตฺตนฺตปิฎก ทีฆนิกาย มหาวคฺค
ที.ม. (ไทย)   =  พระสุตตันตปิฎก ทีฆนิกาย มหาวรรค

ขุ.ธ. (บาลี)  =  สุตฺตนฺตปิฎก ขุทฺทกนิกาย ธมฺมปทคาถา
ขุ.ธ. (ไทย)   =  พระสุตตันตปิฎก ขุททกนิกาย ธรรมบท

องฺ.จตุกฺก. (บาลี) = สุตฺตนฺตปิฎก องฺคุตฺตรนิกาย จตุกฺกนิปาต
องฺ.จตุกฺก. (ไทย)  = พระสุตตันตปิฎก อังคุตตรนิกาย จตุกกนิบาต
```

### 6.3 Symbols Audit

✅ ครอบทุกคำย่อพระไตรปิฎกที่ใช้ในเล่ม
✅ แยกบาลี vs ไทย ชัดเจน
✅ ลำดับเรียงตามอักษร

## 7. Section E — Footnote Format (เชิงอรรถ)

### 7.1 รูปแบบทั่วไป (คู่มือ มจร หน้า 28-29)

**ตำแหน่ง:** ส่วนล่างของหน้าที่จะอ้างอิง — แยกจากเนื้อเรื่องด้วยเส้นขีดคั่น
- เส้นคั่น: ขวางจากขอบซ้าย ยาวประมาณ 20 ตัวพิมพ์
- ห่างจากบรรทัดสุดท้ายของเนื้อเรื่อง: Before 12 pt
- ห่างจากบรรทัดแรกของเชิงอรรถ: After 6 pt

**Indent:**
- บรรทัดแรกย่อหน้าเข้ามาจากขอบซ้าย: **0.7 นิ้ว (1.75 cm)** — เท่ากับการย่อหน้า
- บรรทัดต่อมา: ชิดขอบซ้ายมือทุกบรรทัด
- บรรทัดสุดท้าย: ห่างจากขอบกระดาษด้านล่าง 1 นิ้ว

**ขนาด:** 14 พ้อยท์

### 7.2 การเรียงลำดับเลข

- เริ่มที่ ๑ ในแต่ละบท นับต่อเนื่องจนจบบทนั้น
- ขึ้นบทใหม่ → เริ่ม ๑ ใหม่
- ไม่แยกเชิงอรรถภาษาไทย vs อังกฤษ

### 7.3 รูปแบบเชิงอรรถแต่ละประเภท (รายละเอียดใน 11-citation-footnote.md)

**คัมภีร์พระไตรปิฎก:**
```
ขุ.ธ. (บาลี) ๒๕/๑๒/๙๒, ขุ.ธ. (ไทย) ๒๕/๑๒/๙๒.
```

**หนังสือ (ครั้งแรกในบท):**
```
ผู้แต่ง, ชื่อเรื่อง, (สถานที่: สำนักพิมพ์, ปี), หน้า.
```

**บทความวารสาร:**
```
ผู้แต่ง, "ชื่อบทความ", ชื่อวารสาร, ปีที่/ฉบับที่ (เดือน-ปี): หน้า.
```

**วิทยานิพนธ์:**
```
ผู้แต่ง, "ชื่อเรื่อง", (ดุษฎีนิพนธ์, สาขา, มหาวิทยาลัย, ปี), หน้า.
```

**สัมภาษณ์:**
```
สัมภาษณ์ ผู้ให้สัมภาษณ์, ตำแหน่ง, วันที่.
```

**การอ้างซ้ำ (Ibid.):**
- ไม่มีเอกสารคั่น: "เรื่องเดียวกัน, หน้า ..."
- มีเอกสารคั่น: "ผู้แต่ง, ชื่อเรื่อง, หน้า ..."

### 7.4 Footnote Audit

✅ เริ่ม ๑ ใหม่ทุกบท
✅ ขนาด 14 pt
✅ Indent 0.7 นิ้ว
✅ Before: 12 pt / After: 6 pt
✅ เส้นคั่น 20 ตัวพิมพ์
✅ รูปแบบครบ (ผู้แต่ง, ชื่อเรื่อง, สถานที่: สำนักพิมพ์, ปี, หน้า.)
✅ พระไตรปิฎกระบุ บาลี/ไทย + เล่ม/ข้อ/หน้า
✅ การอ้างซ้ำใช้ "เรื่องเดียวกัน" / ระบุชื่อ + หน้า ตามบริบท

## 8. Section F — Bibliography (บรรณานุกรม)

> รายละเอียดเชิงลึก → `11-citation-footnote.md`

### 8.1 โครงสร้าง (คู่มือ มจร หน้า 27 + 120)

```
บรรณานุกรม
1. ภาษาไทย
   ก. ข้อมูลปฐมภูมิ (Primary Sources)
      - พระไตรปิฎก
   ข. ข้อมูลทุติยภูมิ (Secondary Sources)
      (1) หนังสือ
      (2) บทความ
      (3) วิทยานิพนธ์/ดุษฎีนิพนธ์
      (4) สัมภาษณ์
      (5) เว็บไซต์
2. ภาษาอังกฤษ
   ก. Primary Sources
   ข. Secondary Sources
      (1) Books
      (2) Articles
      (3) Theses/Dissertations
      (4) Websites
```

### 8.2 รูปแบบการเรียง

**ภาษาไทย:**
- เรียงตามลำดับชื่อผู้แต่ง ก-ฮ โดยเคร่งครัด
- คำนำหน้าชื่อ → ไว้หลัง/ต่อจากนามสกุล

**ภาษาอังกฤษ:**
- นามสกุลก่อนชื่อเสมอ
- เรียงตามลำดับนามสกุล A-Z

✅ มีหมวด ก. ปฐมภูมิ + ข. ทุติยภูมิ

## 9. Section G — Appendix (ภาคผนวก)

### 9.1 หน้าบอกตอนภาคผนวก (คู่มือ มจร หน้า 121)

```
ภาคผนวก
(20 พ้อยท์, ตัวหนา, จัดกลางหน้ากระดาษ)
```

### 9.2 ลำดับเอกสารใน Appendix (แนะนำ)

```
ภาคผนวก ก แบบสอบถาม / แบบสัมภาษณ์
ภาคผนวก ข ตาราง IOC + รายชื่อผู้เชี่ยวชาญ 5 คน
ภาคผนวก ค Cronbach's Alpha + Try out 30 ชุด
ภาคผนวก ง หนังสือเชิญผู้เชี่ยวชาญ + ผลตอบรับ
ภาคผนวก จ รายชื่อผู้ให้ข้อมูลสำคัญ + Focus Group
ภาคผนวก ฉ ข้อมูลประชากร + กลุ่มตัวอย่าง
ภาคผนวก ช เอกสารยินยอม (Informed Consent)
ภาคผนวก ซ ข้อมูลสนับสนุนอื่น ๆ
```

### 9.3 Appendix Audit

## 12. Common Mistakes Library (8 Checkpoints)

**[CP-86] Margin ไม่ตรงคู่มือ มจร** [CRITICAL]
- ✗ บน+ซ้าย ≠ 1.5" / ล่าง+ขวา ≠ 1"
- ✓ บน+ซ้าย = 3.81 cm / ล่าง+ขวา = 2.54 cm
- *Source: Fmt-F5 + คู่มือ มจร P124*

**[CP-87] First Line Indent ผิด** [High]
- ✗ ไม่ Indent / Indent < 0.7"
- ✓ Indent 0.7" (1.75 cm) ทุกย่อหน้า
- *Source: Fmt-F4 + คู่มือ มจร P125*

**[CP-88] Paragraph Spacing ขาด** [High]
- ✗ Before 0 / Single line ทั้งย่อหน้า
- ✓ Before 6 pt + Single ภายในย่อหน้า
- *Source: คู่มือ มจร P125*

**[CP-89] เลขหน้าเริ่มใหม่ทุกบท** [High]
- ✗ บทที่ 2 เริ่มหน้า 1 ใหม่
- ✓ ต่อเนื่องจากบทที่ 1
- *Source: Fmt-F3 + คู่มือ มจร P126*

**[CP-90] หน้าแรกบทใส่เลขหน้า** [High]
- ✗ บทที่ 1 หน้า 1 มีเลข "1"
- ✓ หน้าแรกของบทไม่ใส่เลข แต่นับรวม
- *Source: คู่มือ มจร P127*

**[CP-91] เลขหน้าวางผิดตำแหน่ง** [Medium]
- ✗ กลางหน้า / ล่างขวา
- ✓ ห่างขอบบน 1" + แนวเดียวกับขอบขวา
- *Source: คู่มือ มจร P127*

**[CP-92] ตัดคำข้ามบรรทัด** [Medium]
- ✗ "มหา-จุฬาลงกรณ์ราชวิทยาลัย" ตัดที่ "มหา-"
- ✓ ยกคำทั้งคำไปบรรทัดใหม่
- *Source: คู่มือ มจร P124*

**[CP-93] หน้าบอกตอนภาคผนวกใส่เลขหน้า** [Medium]
- ✗ "ภาคผนวก" หน้านี้มีเลข
- ✓ ไม่ใส่เลขแต่นับรวม
- *Source: คู่มือ มจร P121*
