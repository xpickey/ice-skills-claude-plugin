## 2. Footnote Format Specification

### 2.1 Footnote Layout (มาตรฐาน มจร)

```
┌─────────────────────────────────┐
│ [เนื้อหาบทที่กำลังเขียน...]      │
│ ตามที่ Spencer ระบุไว้ ¹           │
│                                  │
│ [Before: 12pt before footer]    │
│ ─────────────────────            │  ← เส้นคั่น 20 ตัวพิมพ์
│ [After: 6pt after line]         │
│       ¹ ผู้แต่ง, ชื่อเรื่อง, ...   │  ← Indent 0.7"
│ [continued line ชิดซ้าย]         │
│ [last line ห่างขอบล่าง 1"]      │
└─────────────────────────────────┘
```

**Specification:**
- ขนาด: 14 พ้อยท์
- Indent: 0.7 นิ้ว (1.75 cm) บรรทัดแรก
- บรรทัดต่อมา: ชิดขอบซ้าย
- Spacing: Before 12 pt (จาก content) / After 6 pt (จาก เส้นคั่น)
- เส้นคั่น: 20 ตัวพิมพ์
- เริ่ม ๑ ใหม่ทุกบท
- ไม่แยกภาษาไทย/อังกฤษ — เรียงตามลำดับที่ปรากฏ

## 3. Footnote Format ทุกประเภทแหล่ง

### 3.1 พระไตรปิฎก (Tipitaka)

**Pattern:**
```
ชื่อย่อคัมภีร์ (บาลี|ไทย) เล่ม/ข้อ/หน้า.
```

**Examples:**
```
✓ ขุ.ธ. (บาลี) ๒๕/๑๒/๙๒, ขุ.ธ. (ไทย) ๒๕/๑๒/๙๒.

✓ ที.ม. (ไทย) ๑๑/๓๐๕/๒๖๐.

✓ องฺ.จตุกฺก. (ไทย) ๒๑/๖๙/๙๒.
```

**ผิด:**
```
✗ ขุ.ธ. ๒๕/๑๒/๙๒ (ไม่ระบุ บาลี/ไทย)

✗ Khuddaka Nikāya, Dhammapada, 25/12/92 (ไม่ใช้รูปแบบ มจร)
```

### 3.2 หนังสือ (Book) — ครั้งแรกในบท

**Pattern:**
```
¹ ผู้แต่ง, ชื่อเรื่อง, (สถานที่: สำนักพิมพ์, ปี), หน้า.
```

**Examples:**
```
✓ ¹ พระพรหมคุณาภรณ์ (ป.อ. ปยุตฺโต), พุทธธรรม ฉบับปรับขยาย, พิมพ์ครั้งที่ ๓๔,
   (กรุงเทพมหานคร: บริษัท สหธรรมิก จำกัด, ๒๕๕๖), หน้า ๗๓.

✓ ² Spencer, L.M. and Spencer, S.G., Competence at Work: Models for Superior
   Performance, (New York: Wiley, 1993), p. 21.

✓ ³ รศ.ดร.สุรพล สุยะพรหม และคณะ, แนวคิด ทฤษฎีและหลักพุทธธรรม, (พระนครศรีอยุธยา:
   โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย, ๒๕๖๗), หน้า ๑๔๔.
```

**ผิด:**
```
✗ ¹ Spencer (1993). Competence at Work. Wiley.
   (ใช้ APA ไม่ใช่ มจร footnote format)

✗ ² พระพรหมคุณาภรณ์, พุทธธรรม, สหธรรมิก.
   (ขาดเลขครั้ง + ปี + หน้า)
```

### 3.3 บทความวารสาร (Journal Article)

**Pattern:**
```
ผู้แต่ง, "ชื่อบทความ", ชื่อวารสาร, ปีที่/ฉบับที่ (เดือน-ปี): หน้า.
```

**Examples:**
```
✓ ¹ สุรพล สุยะพรหม, "การประยุกต์หลักพุทธธรรมในการบริหารจัดการเชิงพุทธ",
   วารสาร มจร สังคมศาสตร์ปริทรรศน์, ปีที่ ๘ ฉบับที่ ๔ (ตุลาคม-ธันวาคม ๒๕๖๒): ๑-๑๑.

✓ ² McClelland, D.C., "Testing for Competence Rather Than for 'Intelligence'",
   American Psychologist, 28(1) (1973): 1-14.
```

**ผิด:**
```
✗ ¹ สุรพล สุยะพรหม (2562). "การประยุกต์...". มจร.
   (ขาดวารสาร + ปีที่ + เลขหน้า)
```

### 3.4 ดุษฎีนิพนธ์ / วิทยานิพนธ์ (Dissertation/Thesis)

**Pattern:**
```
ผู้แต่ง, "ชื่อเรื่อง", (ระดับ, สาขา, มหาวิทยาลัย, ปี), หน้า.
```

**Examples:**
```
✓ ¹ พระมหากฤษฎา กิตฺติโสภโณ, "การพัฒนารูปแบบการบริหารจัดการเชิงพุทธของวัด
   ในประเทศไทย", (ดุษฎีนิพนธ์, สาขาวิชารัฐประศาสนศาสตร์, มหาวิทยาลัย
   มหาจุฬาลงกรณราชวิทยาลัย, ๒๕๖๑), หน้า ๑๕๐.

✓ ² John A. Smith, "Buddhist Leadership in Modern Organizations", (Doctoral
   dissertation, Public Administration, MCU, 2020), p. 200.
```

**กฎเหล็ก สำหรับ ป.เอก:** อ้างเฉพาะดุษฎีนิพนธ์ระดับ **ปริญญาเอก** ในบทที่ 2 + 5 — ห้าม ป.โท (วิทยานิพนธ์)

### 3.5 บทความใน Conference Proceedings

**Pattern:**
```
ผู้แต่ง, "ชื่อบทความ", ใน ชื่อ Proceedings, (สถานที่: ผู้จัด, ปี), หน้า.
```

**Example:**
```
✓ ¹ นาย ก แสนดี, "การบริหารจัดการเชิงพุทธในยุคดิจิทัล", ใน รายงานการประชุม
   วิชาการระดับชาติ ครั้งที่ ๕, (พระนครศรีอยุธยา: มจร, ๒๕๖๖), หน้า ๔๕-๕๖.
```

### 3.6 รายงานวิจัย / รายงานสำนัก (Report)

**Pattern:**
```
ผู้แต่ง/หน่วยงาน, ชื่อรายงาน, (สถานที่: หน่วยงาน, ปี), หน้า.
```

**Example:**
```
✓ ¹ สำนักงาน ก.พ., คู่มือการกำหนดสมรรถนะในราชการพลเรือน: คู่มือสมรรถนะหลัก,
   (กรุงเทพมหานคร: บริษัท ประชุมช่าง จำกัด, ๒๕๕๓), หน้า ๒.
```

### 3.7 เอกสารกฎหมาย / ราชกิจจานุเบกษา (Legal Document)

**Pattern:**
```
ชื่อ พรบ./ระเบียบ, ราชกิจจานุเบกษา ฉบับ เล่ม/ตอน/หน้า (วันเดือนปี).
```

**Example:**
```
✓ ¹ พระราชบัญญัติการจัดซื้อจัดจ้างและการบริหารพัสดุภาครัฐ พ.ศ. ๒๕๖๐,
   ราชกิจจานุเบกษา เล่ม ๑๓๔ ตอนที่ ๒๔ ก หน้า ๑๓ (๒๔ กุมภาพันธ์ ๒๕๖๐).
```

### 3.8 สัมภาษณ์ (Interview)

**Pattern:**
```
สัมภาษณ์ ชื่อผู้ให้สัมภาษณ์, ตำแหน่ง, วันที่.
```

**Examples:**
```
✓ ¹ สัมภาษณ์ นาย ก แสนดี, ผู้อำนวยการสำนักนโยบายฯ, ๑๕ มีนาคม ๒๕๖๖.

✓ ² สัมภาษณ์ผู้ให้ข้อมูลสำคัญ คนที่ ๑, ตำแหน่งผู้บริหาร, ๑๐ พฤษภาคม ๒๕๖๖.
   (กรณี Anonymize)
```

### 3.9 สนทนากลุ่มเฉพาะ (Focus Group)

**Pattern:**
```
สนทนากลุ่มเฉพาะ ผู้ทรงคุณวุฒิกลุ่มที่ N, วันที่.
```

**Example:**
```
✓ ¹ สนทนากลุ่มเฉพาะ ผู้ทรงคุณวุฒิ ๙ คน, ๑๒ มิถุนายน ๒๕๖๖.
```

### 3.10 เว็บไซต์ (Website)

**Pattern:**
```
ผู้แต่ง, "ชื่อหัวข้อ", ชื่อเว็บไซต์, [Online], แหล่งที่มา: URL [วันที่เข้าถึง].
```

**Example:**
```
✓ ¹ สำนักงานคณะกรรมการการอุดมศึกษา, "มาตรฐานการอุดมศึกษา ๒๕๖๑",
   [Online], แหล่งที่มา: https://www.mua.go.th/standard-2561.pdf
   [๑๐ มกราคม ๒๕๖๖].
```

### 3.11 สื่อโสตทัศน์ (Audio-Visual)

**Pattern:**
```
ผู้กำกับ/ผลิต, ชื่อเรื่อง, [ประเภทสื่อ], (สถานที่: ผู้ผลิต, ปี).
```

**Example:**
```
✓ ¹ มจร, การสัมมนาดุษฎีนิพนธ์ ปร.ด. รปศ., [Video], (พระนครศรีอยุธยา: มจร, ๒๕๖๖).
```

### 3.12 Government Document (เอกสารราชการ)

**Pattern:**
```
หน่วยงาน, ชื่อเอกสาร, เลขที่เอกสาร, (สถานที่: หน่วยงาน, ปี).
```

**Example:**
```
✓ ¹ กระทรวงมหาดไทย, ระเบียบกระทรวงมหาดไทยว่าด้วยการรับเงิน, ระเบียบ
   ที่ ๑๒/๒๕๖๖, (กรุงเทพมหานคร: กระทรวงมหาดไทย, ๒๕๖๖).
```

## 4. การอ้างซ้ำ (Repeated Citation)

### 4.1 อ้างซ้ำต่อเนื่อง (ไม่มีเอกสารคั่น)

ใช้ **"เรื่องเดียวกัน, หน้า [ใหม่]."**

**Example:**
```
¹ ผู้แต่ง, ชื่อเรื่อง, (สถานที่: สำนักพิมพ์, ปี), หน้า ๑๒.
² เรื่องเดียวกัน, หน้า ๑๕.
³ เรื่องเดียวกัน, หน้า ๒๐.
```

**ภาษาอังกฤษ:**
```
¹ Spencer, ..., p. 12.
² Ibid., p. 15.
³ Ibid., p. 20.
```

### 4.2 อ้างซ้ำมีเอกสารคั่น

ใช้ **"ผู้แต่ง, ชื่อเรื่อง (ย่อ), หน้า."**

**Example:**
```
¹ พระพรหมคุณาภรณ์, พุทธธรรม, ๒๕๕๖, หน้า ๗๓.
² สุรพล สุยะพรหม, แนวคิด ทฤษฎีฯ, ๒๕๖๗, หน้า ๑๔๔.
³ พระพรหมคุณาภรณ์, พุทธธรรม, หน้า ๗๕.
```

### 4.3 Op. cit. (op. cit. / op.cit.)

**กฎ มจร:** ไม่นิยมใช้ Op. cit. ใช้ "ผู้แต่ง, ชื่อเรื่อง, หน้า" แทน

## 5. Bibliography Format

### 5.1 โครงสร้างทั้งหมด

```
บรรณานุกรม

๑. ภาษาไทย

   ก. ข้อมูลปฐมภูมิ

      มหาจุฬาลงกรณราชวิทยาลัย. พระไตรปิฎกภาษาบาลี ฉบับมหาจุฬาเตปิฏกํ ๒๕๐๐.
         กรุงเทพมหานคร: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย, ๒๕๓๕.

      มหาจุฬาลงกรณราชวิทยาลัย. พระไตรปิฎกภาษาไทย ฉบับมหาจุฬาลงกรณราช
         วิทยาลัย. กรุงเทพมหานคร: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย, ๒๕๓๙.

   ข. ข้อมูลทุติยภูมิ

      (๑) หนังสือ:
         จำนงค์ ทองประเสริฐ. ประวัติพระพุทธศาสนาในเอเชียอาคเณย์. พระนคร:
            มูลนิธิอภิธรรมมหาธาตุวิทยาลัย, ๒๕๑๔.

         พระพรหมคุณาภรณ์ (ป.อ. ปยุตฺโต). พุทธธรรม ฉบับปรับขยาย. พิมพ์ครั้งที่ ๓๔.
            กรุงเทพมหานคร: บริษัท สหธรรมิก จำกัด, ๒๕๕๖.

      (๒) บทความ:
         สุรพล สุยะพรหม. "การประยุกต์หลักพุทธธรรม". วารสาร มจร สังคมศาสตร์
            ปริทรรศน์. ปีที่ ๘ ฉบับที่ ๔ (ตุลาคม-ธันวาคม ๒๕๖๒): ๑-๑๑.

      (๓) ดุษฎีนิพนธ์/วิทยานิพนธ์:
         พระมหากฤษฎา กิตฺติโสภโณ. "การพัฒนารูปแบบการบริหารจัดการเชิงพุทธ
            ของวัดในประเทศไทย". ดุษฎีนิพนธ์ปรัชญาดุษฎีบัณฑิต สาขาวิชา
            รัฐประศาสนศาสตร์. มหาวิทยาลัยมหาจุฬาลงกรณราชวิทยาลัย, ๒๕๖๑.

      (๔) สัมภาษณ์:
         สัมภาษณ์ นาย ก แสนดี. ผู้อำนวยการสำนักนโยบายฯ. ๑๕ มีนาคม ๒๕๖๖.

      (๕) เว็บไซต์:
         สำนักงานคณะกรรมการการอุดมศึกษา. "มาตรฐานการอุดมศึกษา ๒๕๖๑". [Online].
            แหล่งที่มา: https://www.mua.go.th [๑๐ มกราคม ๒๕๖๖].

๒. ภาษาอังกฤษ

   I. Primary Sources

      Mahachulalongkornrajavidyalaya University. Pāli Tipiṭaka, MCU Edition.
         Bangkok: MCU Press, 1992.

   II. Secondary Sources

      (1) Books:
         McClelland, D.C. Testing for Competence Rather Than for 'Intelligence'.
            Cambridge: Harvard University Press, 1973.

         Spencer, L.M. and Spencer, S.G. Competence at Work: Models for Superior
            Performance. New York: Wiley, 1993.

      (2) Articles:
         McClelland, D.C. "Testing for Competence Rather Than for 'Intelligence'".
            American Psychologist. 28, 1 (1973): 1-14.

      (3) Theses/Dissertations:
         Smith, John A. "Buddhist Leadership in Modern Organizations". Doctoral
            Dissertation in Public Administration. Mahachulalongkornrajavidyalaya
            University, 2020.
```

### 5.2 Bibliography Pattern (Bibliography vs Footnote)

**ความต่างจาก Footnote:**

| Element | Footnote | Bibliography |
|---------|----------|--------------|
| ลำดับ | เลขเชิงอรรถ | เรียงอักษร |
| Indent | บรรทัดแรก 0.7" | Hanging indent |
| ชื่อผู้แต่ง | ตามชื่อจริง | นามสกุลก่อน (Eng) / ชื่อก่อน (ไทย) |
| สถานที่: สำนักพิมพ์ | (สถานที่: สำนักพิมพ์, ปี) | สถานที่: สำนักพิมพ์, ปี. |
| หน้า | หน้า X. | (ไม่มีหน้า สำหรับหนังสือ) |

### 5.3 Hanging Indent ใน Bibliography

```
จำนงค์ ทองประเสริฐ. ประวัติพระพุทธศาสนาในเอเชียอาคเณย์. พระนคร:
   มูลนิธิอภิธรรมมหาธาตุวิทยาลัย, ๒๕๑๔.   ← บรรทัดต่อ Indent
```

**Specification:**
- บรรทัดแรก: ชิดขอบซ้าย
- บรรทัดต่อมา: Indent (Hanging Indent ~0.5")
- Spacing: Before 6 pt

## 6. Edge Cases & Special Cases

### 6.1 ผู้แต่งหลายคน (Multiple Authors)

**2-3 คน:**
```
✓ Spencer, L.M. and Spencer, S.G.
✓ สุรพล สุยะพรหม และ พระมหากฤษฎา กิตฺติโสภโณ
```

**4+ คน:**
```
✓ Spencer, L.M., et al.
✓ สุรพล สุยะพรหม และคณะ
```

### 6.2 ไม่มีผู้แต่ง (No Author)

```
✓ "ชื่อเรื่อง". ใน ชื่อแหล่ง.
✓ Anonymous. ชื่อเรื่อง.
```

### 6.3 ไม่มีปี (No Date)

```
✓ ผู้แต่ง. ชื่อเรื่อง. สถานที่: สำนักพิมพ์, ม.ป.ป.   (ไม่ปรากฏปีที่พิมพ์)
✓ Author. Title. Place: Publisher, n.d.   (no date)
```

### 6.4 ไม่มีสถานที่พิมพ์ (No Place)

```
✓ ผู้แต่ง. ชื่อเรื่อง. ม.ป.ท.: สำนักพิมพ์, ปี.   (ไม่ปรากฏสถานที่พิมพ์)
✓ Author. Title. n.p.: Publisher, year.
```

### 6.5 พิมพ์ครั้งที่ (Edition)

```
✓ พระพรหมคุณาภรณ์. พุทธธรรม. พิมพ์ครั้งที่ ๓๔. กรุงเทพมหานคร: สหธรรมิก, ๒๕๕๖.
✓ Smith, J. Title. 3rd ed. New York: Publisher, 2020.
```

### 6.6 ชื่อบรรณาธิการ (Editor)

```
✓ พระมหา ก. (บรรณาธิการ). ชื่อเรื่อง. สถานที่: สำนักพิมพ์, ปี.
✓ Smith, J. (ed.). Title. Place: Publisher, year.
```

### 6.7 หนังสือแปล (Translation)

```
✓ Smith, J. ชื่อเรื่อง. แปลโดย ผู้แปล. สถานที่: สำนักพิมพ์, ปี.
```

## 7. Citation Quality Audit

### 7.1 Pre-Submission Checklist

✅ ทุก citation ในเชิงอรรถ → มีในบรรณานุกรม
✅ ทุก citation ในบรรณานุกรม → ใช้ใน text
✅ Footnote เริ่ม ๑ ใหม่ทุกบท
✅ Footnote 14 pt + Indent 0.7"
✅ การอ้างซ้ำใช้ "เรื่องเดียวกัน" / "ชื่อ-เรื่อง-หน้า" ตามบริบท
✅ พระไตรปิฎก ระบุ บาลี/ไทย + เล่ม/ข้อ/หน้า
✅ ระดับ ป.เอก เท่านั้นในบทที่ 2 + 5 (ห้าม ป.โท)
✅ บรรณานุกรม จัดหมวด ก. ปฐมภูมิ + ข. ทุติยภูมิ
✅ เรียง ก-ฮ (ไทย) / A-Z นามสกุลก่อน (อังกฤษ)
✅ Hanging indent ใน Bibliography

### 7.2 Bibliography vs In-text — Cross-validate

```
1. List ทุก citation ในเชิงอรรถ (ทุกบท)
2. List ทุก entry ในบรรณานุกรม
3. Cross-check
   - Match: ✓
   - In-text only (missing in Bib): ❌ Add to Bib
   - Bib only (unused in text): ❌ Remove from Bib
```

## 8. Common Mistakes Library (10 Checkpoints)

**[CP-112] รูปแบบเชิงอรรถไม่ครบ** [CRITICAL]
- ✗ ¹ Spencer (1993). Competence at Work. Wiley.
- ✓ ¹ Spencer, L.M. and Spencer, S.G., Competence at Work: Models for Superior Performance, (New York: Wiley, 1993), p. 21.
- *Source: Cit-E1*

**[CP-113] อ้างซ้ำผิดรูปแบบ** [High]
- ✗ ใช้ "Spencer, op.cit., p. 21"
- ✓ ไม่มีคั่น: "เรื่องเดียวกัน, หน้า ๒๑"
- ✓ มีคั่น: "Spencer, Competence at Work, p. 21"
- *Source: Cit-E2*

**[CP-114] พระไตรปิฎกไม่ระบุรูปแบบ มจร** [CRITICAL]
- ✗ "Tipitaka, Khuddaka, 25/12/92"
- ✓ "ขุ.ธ. (ไทย) ๒๕/๑๒/๙๒"
- *Source: Cit-E3 + คู่มือ มจร*

**[CP-115] เชิงอรรถไม่ Indent** [Medium]
- ✗ บรรทัดแรกชิดขอบซ้าย
- ✓ Indent 0.7" (1.75 cm)
- *Source: Cit-E4 + คู่มือ มจร P28*

**[CP-116] บรรณานุกรมหมวดผิด** [High]
- ✗ ไม่แยก ปฐมภูมิ vs ทุติยภูมิ / ผสมไทย-อังกฤษ
- ✓ ก. พระไตรปิฎก / ข. ทุติยภูมิ (หนังสือ-บทความ-ดุษฎี-สัมภาษณ์-เว็บ) / ค. อังกฤษ
- *Source: Cit-E5 + คู่มือ มจร P27*

**[CP-117] เลขย่อพระไตรปิฎกสะกดผิด** [High]
- ✗ "วิรยิพละ" (สะกดผิด)
- ✓ "วิริยพละ"
- *Source: Cit-E7*

**[CP-118] ขาดเลขครั้งที่พิมพ์ (Edition)** [Medium]
- ✗ พุทธธรรม, (กรุงเทพ: สหธรรมิก, ๒๕๕๖)
- ✓ พุทธธรรม, พิมพ์ครั้งที่ ๓๔, (กรุงเทพ: สหธรรมิก, ๒๕๕๖)
- *Source: §6.5*

**[CP-119] อ้าง ป.โท ใน บทที่ 2/5** [CRITICAL]
- ✗ อ้างวิทยานิพนธ์ปริญญาโทใน Discussion
- ✓ เฉพาะดุษฎีนิพนธ์ ป.เอก หรือบทความวิจัย
- *Source: คู่มือ มจร P5 + ระดับ ป.เอก*

**[CP-120] In-text citation ไม่มีในบรรณานุกรม** [CRITICAL]
- ✗ อ้าง "Smith (1993)" แต่ไม่มีในบรรณานุกรม
- ✓ Cross-validate ทุก citation
- *Source: §7.2*

**[CP-121] Hanging Indent ขาดใน Bibliography** [Medium]
- ✗ ทุกบรรทัดชิดขอบซ้าย
- ✓ บรรทัดแรกชิดขอบ / บรรทัดต่อ Indent
- *Source: §5.3*

## 9. Quick Reference Card

### Footnote (เริ่ม ๑ ใหม่ทุกบท)

| Source | Pattern |
|--------|---------|
| Tipitaka | ชื่อย่อ (บาลี\|ไทย) เล่ม/ข้อ/หน้า. |
| Book | ผู้แต่ง, ชื่อเรื่อง, (สถานที่: สำนักพิมพ์, ปี), หน้า. |
| Article | ผู้แต่ง, "ชื่อบทความ", ชื่อวารสาร, ปีที่/ฉบับ (เดือน-ปี): หน้า. |
| Dissertation | ผู้แต่ง, "ชื่อเรื่อง", (ดุษฎีนิพนธ์, สาขา, มหาวิทยาลัย, ปี), หน้า. |
| Interview | สัมภาษณ์ ชื่อ, ตำแหน่ง, วันที่. |
| FGD | สนทนากลุ่มเฉพาะ ผู้ทรงคุณวุฒิ N คน, วันที่. |
| Web | ผู้แต่ง, "หัวข้อ", ชื่อเว็บไซต์, [Online], แหล่งที่มา: URL [วันที่เข้าถึง]. |

### Repeated Citation

| สถานการณ์ | รูปแบบ |
|----------|--------|
| ไม่มีคั่น (continuous) | เรื่องเดียวกัน, หน้า X. |
| ไม่มีคั่น (English) | Ibid., p. X. |
| มีคั่น | ผู้แต่ง, ชื่อเรื่อง (ย่อ), หน้า X. |

### Bibliography Structure

```
1. ภาษาไทย
   ก. ปฐมภูมิ → พระไตรปิฎก
   ข. ทุติยภูมิ → (1) หนังสือ (2) บทความ (3) ดุษฎีนิพนธ์
                  (4) สัมภาษณ์ (5) เว็บไซต์
2. ภาษาอังกฤษ
   I. Primary
   II. Secondary → (1) Books (2) Articles (3) Theses
```
