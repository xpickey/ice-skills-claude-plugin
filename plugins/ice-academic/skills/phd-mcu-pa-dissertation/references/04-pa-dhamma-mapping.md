### 3.1 Theory → Dhamma (Forward Mapping by Group)

### 3.3 Use-case Patterns (Research Pattern Driven)

> ใช้เมื่อมี Topic Pattern ในใจ → หา Theory + Dhamma ที่ใช้คู่กัน

#### Pattern 1: "การพัฒนา X ของบุคลากร"

```
Theory IV: HRD (Nadler) หรือ ไตรสิกขา/ภาวนา ๔
Theory DV: สมรรถนะ (Spencer/McClelland) หรือ คุณภาพชีวิต
Dhamma:   ไตรสิกขา + ภาวนา ๔ + อิทธิบาท ๔
ตัวอย่าง: "การพัฒนาสมรรถนะของบุคลากรเชิงพุทธ"
```

#### Pattern 2: "ภาวะผู้นำกับ Y"

```
Theory IV: Transformational Leadership (Bass)
Theory DV: สมรรถนะ / ความผูกพัน / ประสิทธิผล
Dhamma:   พรหมวิหาร ๔ + สัปปุริสธรรม ๗ + ทศพิธราชธรรม ๑๐
ตัวอย่าง: "ภาวะผู้นำเชิงพุทธกับสมรรถนะองค์กร"
```

#### Pattern 3: "ธรรมาภิบาล อปท."

```
Theory IV: Good Governance 6 (UNDP/ก.พ.)
Theory DV: ประสิทธิผล / ความพึงพอใจประชาชน
Dhamma:   ทศพิธราชธรรม ๑๐ + ราชสังคหวัตถุ ๔
ตัวอย่าง: "ธรรมาภิบาล อปท. เชิงพุทธ"
```

#### Pattern 4: "การมีส่วนร่วมของชุมชน"

```
Theory IV: Cohen & Uphoff 4 ด้าน
Theory DV: คุณภาพชีวิต / ความสำเร็จโครงการ
Dhamma:   อปริหานิยธรรม ๗ + สังคหวัตถุ ๔
ตัวอย่าง: "การมีส่วนร่วมชุมชนเชิงพุทธ"
```

#### Pattern 5: "การปฏิบัติงานด้วยเทคโนโลยีดิจิทัล" ⭐

```
Theory IV: สมรรถนะดิจิทัล (Spencer + Senge Systems Thinking)
Theory DV: ผลการปฏิบัติงาน / ประสิทธิภาพ
Dhamma:   อิทธิบาท ๔ + พละ ๕ (Official + Practical)
ตัวอย่าง: "สมรรถนะดิจิทัลเชิงพุทธของบุคลากรภาครัฐ"
หมายเหตุ: ตรงกับหัวข้อท่าน — ดู §6 Mini Case
```

#### Pattern 6: "การบริหารความเสี่ยง"

```
Theory IV: COSO ERM 5 / Risk Components
Theory DV: ผลกระทบ / ความเสียหาย
Dhamma:   อริยสัจ ๔ + ปธาน ๔
ตัวอย่าง: "การบริหารความเสี่ยงเชิงพุทธ"
```

## 7. Common Mistakes (Mapping-specific)

ตรวจทุกครั้งก่อนสร้างกรอบแนวคิด

**[CP-44] Theory-Dhamma Mismatch (Surface)** [CRITICAL]
- ✗ Map "การเรียนรู้ดิจิทัล" กับ "ทศพิธราชธรรม" (ไม่ตรงประเด็น)
- ✓ Map "การเรียนรู้ดิจิทัล" กับ "อิทธิบาท ๔ + พละ ๕" (มีพื้นฐานจากหนังสือ)
- *Source: ตรงกับ CP-25*

**[CP-45] Many-to-Many Mapping ที่ไม่ Justify** [High]
- ✗ ใส่หลักธรรม 5 หมวดให้ตัวแปรเดียว โดยไม่อธิบาย
- ✓ เลือก 1-2 หมวดหลัก + อธิบาย "ทำไม + ตรงตรงไหน"
- *Source: XL-48 + CP-42*

**[CP-46] อ้างพระไตรปิฎกผิด/ไม่ครบ** [CRITICAL]
- ✗ "อิทธิบาท ๔" โดยไม่อ้างเลขเล่ม-ข้อ-หน้า
- ✓ "อิทธิบาท ๔ ที.ปา. (ไทย) ๑๑/๓๐๕/๒๖๐" — ฉบับ มจร
- *Source: คู่มือ มจร P2.2 + Cit-E3*

**[CP-47] ใช้คำว่า "พุทธบูรณาการ" ลอย** [Medium]
