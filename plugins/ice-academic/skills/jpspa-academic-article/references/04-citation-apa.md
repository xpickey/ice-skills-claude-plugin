## PART A — CITATION FORMAT REFERENCE (Master Template)

วารสารใช้ระบบ **APA นาม-ปี** ทั้งในเนื้อหาและท้ายเล่ม แบ่ง Citation เป็นสองหมวดใหญ่คือการอ้างในเนื้อหา (In-text) และเอกสารอ้างอิงท้ายเล่ม (Reference List)

---

### A.1 การอ้างอิงในเนื้อหา (In-text Citation)

#### A.1.1 ต้นเนื้อความ (Narrative Citation)

วาง **ชื่อ-นามสกุล** แล้วตามด้วย วงเล็บเปิด ปีที่ตีพิมพ์ วงเล็บปิด

| กรณี | รูปแบบ | ตัวอย่าง |
|---|---|---|
| ภาษาไทย — กล่าวว่า | ชื่อ-สกุล (ปี) กล่าวว่า... | กฤษฎา แซ่หลี (2562) กล่าวว่า... |
| ภาษาไทย — สอดคล้อง | ...สอดคล้องกับงานวิจัยของชื่อ-สกุล (ปี) ผลการวิจัยพบว่า... | ...สอดคล้องกับงานวิจัยของกฤษฎา แซ่หลี (2562) ผลการวิจัยพบว่า... |
| ภาษาอังกฤษ — explains | First Last (Year) explains that... | Krisada Saelee (2019) explains that... |
| ภาษาอังกฤษ — consistent | …consistent with the research of First Last (Year), which found that… | …consistent with the research of Krisada Saelee (2019), which found that… |

> **หมายเหตุภาษาไทย:** ใช้ **ชื่อ-นามสกุล เต็ม** ไม่ตัดเหลือเฉพาะนามสกุล — แตกต่างจาก APA ฉบับมาตรฐานสากลที่ใช้นามสกุลอย่างเดียว

#### A.1.2 ท้ายเนื้อความ (Parenthetical Citation)

วงเล็บเปิด ชื่อ-นามสกุล (ไทย) หรือ นามสกุล (อังกฤษ) เครื่องหมายจุลภาค ปีที่พิมพ์ วงเล็บปิด

##### A.1.2.1 พระไตรปิฎกและอรรถกถา

```
(ชื่อคัมภีร์/เล่มที่/ข้อที่/เลขหน้า) (มหาจุฬาลงกรณราชวิทยาลัย, ปีที่พิมพ์)
```

ตัวอย่าง:

> "ดูกรภิกษุทั้งหลาย จักร 4 ประการนี้ เป็นเครื่องเป็นไปแก่มนุษย์และเทวดา..."
> **(องฺ.จตุกก. 21/31/37) (มหาจุฬาลงกรณราชวิทยาลัย, 2539)**

ส่วนประกอบ:
- `องฺ.จตุกก.` — ชื่อคัมภีร์ย่อ (อังคุตตรนิกาย จตุกกนิบาต)
- `21` — เลขเล่ม
- `31` — เลขข้อ
- `37` — เลขหน้า
- `(มหาจุฬาลงกรณราชวิทยาลัย, 2539)` — สำนักพิมพ์ + ปีพิมพ์

##### A.1.2.2 ผู้แต่งภาษาไทย ๑ ราย

```
(ชื่อ-นามสกุล, ปี)
```

ตัวอย่าง: `(เกียรติศักดิ์ สุขเหลือง, 2560)`

##### A.1.2.3 ผู้แต่งภาษาไทย ๒ ราย

ใช้คำว่า **"และ"** เชื่อมผู้เขียนทั้งสอง

```
(ชื่อ-นามสกุลคนที่ 1 และชื่อ-นามสกุลคนที่ 2, ปี)
```

ตัวอย่าง: `(พระมหาสุนันท์ สุนนฺโท และเกียรติศักดิ์ สุขเหลือง, 2560)`

##### A.1.2.4 ผู้แต่งภาษาไทย มากกว่า ๒ ราย

ระบุ **คนแรก** + **"และคณะ"**

```
(ชื่อ-นามสกุลคนแรกและคณะ, ปี)
```

ตัวอย่าง: `(ธิติวุฒิ หมั่นมีและคณะ, 2560)`

##### A.1.2.5 เนื้อความเดียวกันมีผู้อ้างหลายคน

ใช้รายการที่ใกล้ปีปัจจุบันมากที่สุด — ไม่ list ทุกคน

##### A.1.2.6 ผู้แต่งภาษาอังกฤษ ๑ ราย

```
(Lastname, Year)
```

ตัวอย่าง: `(Keown, 2010)`

##### A.1.2.7 ผู้แต่งภาษาอังกฤษ ๒ ราย

ใช้ **`&`** (ไม่ใช่ `and`)

```
(Lastname1 & Lastname2, Year)
```

ตัวอย่าง: `(Hersey & Blanchard, 2010)`

##### A.1.2.8 ผู้แต่งภาษาอังกฤษ มากกว่า ๒ ราย

ใช้ **`et al.`** (มีจุดท้าย `al.`)

```
(Lastname1 et al., Year)
```

ตัวอย่าง: `(Kaiser et al., 2010)`

> **กฎ Reciprocity (สำคัญ):** ทุก Citation ในเนื้อหาต้องปรากฏท้ายเล่ม และทุกรายการท้ายเล่มต้องถูกอ้างในเนื้อหา ไม่มี Ghost Citation

### A.2 เอกสารอ้างอิงท้ายเล่ม — ภาษาไทย

> **Convention การเว้นวรรค:** เครื่องหมาย `/` ในเอกสารต้นฉบับหมายถึง "เว้นวรรค ๑ ตัวอักษร" — ไม่ใช่อักขระจริง

#### A.2.1 พระไตรปิฎก / อรรถกถา

```
ผู้แต่ง./(ปีที่พิมพ์)./ชื่อพระไตรปิฎก อรรถกถา./สถานที่พิมพ์:/สำนักพิมพ์หรือโรงพิมพ์.
```

ตัวอย่าง:

> มหาจุฬาลงกรณราชวิทยาลัย. (2539). พระไตรปิฎกฉบับภาษาไทย ฉบับมหาจุฬาลงกรณราชวิทยาลัย. กรุงเทพฯ: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย.
>
> ________. (2539). พระไตรปิฎกฉบับภาษาบาลี ฉบับมหาจุฬาลงกรณราชวิทยาลัย. กรุงเทพฯ: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย.

> **สังเกต:** เมื่อมีรายการของผู้แต่งคนเดิมต่อกัน ใช้ `________.` (8 underscore) แทนชื่อผู้แต่ง

#### A.2.2 หนังสือ

```
ผู้แต่ง./(ปีที่พิมพ์)./ชื่อหนังสือ/(ครั้งที่พิมพ์)./สถานที่พิมพ์:/สำนักพิมพ์หรือโรงพิมพ์.
```

ตัวอย่าง:

> พระมหาสุทิตย์ อาภากโร. (2548). เครือข่าย: ธรรมชาติ ความรู้ และการจัดการ (พิมพ์ครั้งที่ 2). กรุงเทพฯ: พิสิษฐ์ ไทย ออฟเซต.

> **กฎ:** ครั้งที่พิมพ์ใส่เฉพาะกรณีที่ **ไม่ใช่พิมพ์ครั้งที่ 1** — พิมพ์ครั้งแรก ไม่ต้องระบุ

#### A.2.3 บทความในหนังสือ

```
ผู้แต่ง./(ปีที่พิมพ์)./ชื่อบทความ./ใน ชื่อบรรณาธิการ./ชื่อเรื่อง./(เลขหน้าที่อ้าง)./สถานที่พิมพ์:/สำนักพิมพ์หรือโรงพิมพ์.
```

ตัวอย่าง:

> พระสุกิจจ์ สุจิณโณ. (2559). การสร้างความคิดนามธรรมในวัฒนธรรมไทย. ใน ปวิตร ว่องวีระ, ทฤษฎีและวิธีวิทยาของการวิจัยวัฒนธรรม. (น. 112). กรุงเทพฯ: อัมรินทร์.

#### A.2.4 บทความในวารสาร

```
ผู้แต่ง./(ปีที่พิมพ์)./ชื่อบทความ./ชื่อวารสาร,/ปีที่(ฉบับที่),/เลขหน้าแรก-เลขหน้าสุดท้าย.
```

ตัวอย่าง:

> ธิติวุฒิ หมั่นมี. (2557). การวางแผนและการติดต่อประสานงานเชิงพุทธ. วารสารบัณฑิตศึกษาวิชาการ, 3(1), 25-31.

#### A.2.5 บทความในสารานุกรม

```
ผู้แต่ง./(ปีที่พิมพ์)./ชื่อบทความ./ใน/ชื่อสารานุกรม/(เล่มที่อ้าง, หน้า เลขหน้าที่อ้าง).
```

ตัวอย่าง:

> วุฒิชัย มูลศิลป์. (2549). กฎหมายตราสามดวง. ใน สารานุกรมประวัติศาสตร์ไทย (เล่ม 1, อักษร ก, น. 12-16).

#### A.2.6 หนังสือพิมพ์ (รูปแบบหลัก)

```
ผู้แต่ง./(ปีที่พิมพ์,/วันที่/เดือน)./ชื่อบทความ./ชื่อหนังสือพิมพ์,/เลขหน้า.
```

ตัวอย่าง:

> สุชาติ เผือกสกนธ์. (2549, 9 มิถุนายน). ปรัชญาเศรษฐกิจพอเพียง. ผู้จัดการรายวัน, น. 13.

#### A.2.7 สารนิพนธ์ / วิทยานิพนธ์ / ดุษฎีนิพนธ์

```
ผู้แต่ง./(ปีที่พิมพ์)./ชื่อวิทยานิพนธ์/(ระดับวิทยานิพนธ์ สาขาวิชา)./สถานที่พิมพ์:/สำนักพิมพ์.
```

ตัวอย่าง:

> พระครูพิศิษฏ์ประชานาถ. (2562). การพัฒนาการมีส่วนร่วมของพระสงฆ์ในการพัฒนาชุมชนตามแนววิถีพุทธในจังหวัดสมุทรสงคราม (ดุษฎีนิพนธ์ปริญญาพุทธศาสตรดุษฎีบัณฑิต สาขาวิชาการจัดการเชิงพุทธ). พระนครศรีอยุธยา: มหาวิทยาลัยมหาจุฬาลงกรณราชวิทยาลัย.

#### A.2.8 รายงานการวิจัย

```
ผู้แต่ง./(ปีที่พิมพ์)./ชื่อเรื่อง/(รายงานผลการวิจัย)./สถานที่พิมพ์: สำนักพิมพ์.
```

ตัวอย่าง:

> พระมหากฤษฎา กิตฺติโสภโณ. (2562). รูปแบบการปลูกฝังคุณธรรมและจริยธรรมสำหรับวัยรุ่นไทยยุคใหม่ตามหลักพุทธธรรม ผ่านกลไกการขับเคลื่อนโครงการคลินิกคุณธรรมของพระธรรมวิทยากร (รายงานผลการวิจัย). พระนครศรีอยุธยา: มหาวิทยาลัยมหาจุฬาลงกรณราชวิทยาลัย.

#### A.2.9 สัมภาษณ์

```
ชื่อผู้ที่ได้รับการสัมภาษณ์./(ปีที่สัมภาษณ์, วัน เดือน)./ตำแหน่ง/[บทสัมภาษณ์].
```

ตัวอย่าง:

> พระอุดมสิทธินายก. (2563, 5 กุมภาพันธ์). ผู้อำนวยการหลักสูตรบัณฑิตศึกษา สาขาวิชาการจัดการเชิงพุทธ [บทสัมภาษณ์].

#### A.2.10 สื่ออิเล็กทรอนิกส์

```
ผู้แต่ง./(ปีที่เผยแพร่)./ชื่อเรื่อง./สืบค้นเมื่อ วัน เดือน ปี จาก แหล่งที่มาของข้อมูล.
```

ตัวอย่าง:

> ธรรมะไทย. (2563). พระพุทธศาสนาในประเทศไทย. สืบค้นเมื่อ 20 มีนาคม 2563 จาก http://www.dhammathai.org/thailand/thailand.php.

#### A.2.11 ราชกิจจานุเบกษา

```
ชื่อเรื่อง./(พ.ศ.,วันที่ เดือนที่ประกาศ)./ราชกิจจานุเบกษา./เล่ม ตอน./หน้า.
```

ตัวอย่าง:

> ข้อกำหนดกรมตำรวจ เรื่อง กำหนดชื่อและประเภทของวัตถุที่ออกฤทธิ์ต่อจิตและประสาท... (2537, 13 กรกฎาคม). ราชกิจจานุเบกษา. เล่ม 111 ตอนพิเศษ 30 ง. หน้า 31.

#### A.2.12 หนังสือพิมพ์ (รูปแบบทางเลือก — มีคอลัมน์)

```
ผู้แต่ง./(ปีพิมพ์, วัน เดือน)./ชื่อบทความ./ชื่อหนังสือพิมพ์./น. หรือ p. หรือ pp. เลขหน้า.
```

ตัวอย่าง:

> ไทยรัฐ. (2548, 19 เมษายน). เชียร์พระสอนศีลธรรม. ไทยรัฐ. น. 3.

### A.3 เอกสารอ้างอิงท้ายเล่ม — ภาษาอังกฤษ

#### A.3.1 Tripitaka Commentary

```
Institution Name./(Year)./Title of the Tripitaka./Location:/Publisher.
```

ตัวอย่าง:

> Mahachulalongkornrajavidyalaya University. (1996). Thai Tripitaka. Bangkok: MCU Press.

#### A.3.2 Books

```
Author./(Year)./Title of the book/(Edition ed.)./Location:/Publisher.
```

ตัวอย่าง:

> Hunt, E. K. & Lautzenheiser, M. (2015). History of Economic Thought (3rd ed.). New York: M. E. Sharpe.

#### A.3.3 Book Articles (Chapter)

```
Author./(Year)./Title of the chapter./In Editor/(Eds.),/Title of the book/(xth ed.)./(pp-pp)./Publisher.
```

ตัวอย่าง:

> Sparks, D. L. (2003). Environmental Soil Chemistry: An Overview. In Sparks, D. L. (Eds.), Environmental Soil Chemistry (2nd ed.). (pp. 1-42). Academic Press.

#### A.3.4 Encyclopaedia Articles

```
Author./(Year)./Title of the Article./In Title of Encyclopaedia/(Pages)./Publisher.
```

ตัวอย่าง:

> Wasinsarakorn, W. (2004). Psychology and Developmental Psychology. In Encyclopedia of Education. Collected edition of specific issues, 4th rank (pp. 99-117). Thanat Printing Co., Ltd.

#### A.3.5 Journal Articles

```
Author./(Year)./Title of the Article./Title of Journal,/Volume(Issue), XX-XX.
```

ตัวอย่าง:

> Limprasert, P. (2022). The Research of Punishment Suspension of Buddhist Monk and Secular of The Criminal Justice in Thailand. Journal of Social Science and Buddhistic Anthropology, 7(11), 37-53.

#### A.3.6 Newspaper

```
Author./(Year, Month Day)./Column Name./Title of Newspaper./p. xx.
```

ตัวอย่าง:

> Nanuam, W. (2023, January 4). Plans Proceeding for Joint Petroleum Production with Cambodia. Bangkok Post. p. 22.

#### A.3.7 Thesis and Dissertations

```
Author./(Year)./Title/(Doctoral or Master's Thesis or Dissertation)./Location:/Publisher.
```

ตัวอย่าง:

> Pittayawattanachai, P. (2019). An Analysis of Buddhism Propagation by using Buddhist Arts and Landscape of Cetiyaphum Monastery, Namphong District, Khonkaen (Doctoral dissertation). Ayutthaya: Mahachulalongkornrajavidyalaya University.

#### A.3.8 Interview

```
Key Informant./(Year, Month Day)./Position./(Interview).
```

ตัวอย่าง:

> Anuchid. P. (2022, November 5). Director of the Research Institute. (Interview).

#### A.3.9 Website

```
Author./(Year)./Title./Retrieved Month Day, Year,/from URL.
```

ตัวอย่าง:

> Buddhism Meditation Triratna Find us Community. (2020). Who was the Buddha?. Retrieved March 20, 2020, from https://thebuddhistcentre.com/text/who-was-buddha.

### A.4 Authority Source — พระไตรปิฎก มจร

## PART B — 4-PILLAR VERIFICATION METHOD

วิธีตรวจสอบอ้างอิงของวารสาร JPSPA ใช้ ๔ เสาหลัก แต่ละเสาตรวจคนละมิติ ครอบคลุมทั้ง Format, Content Integrity และ Academic Authority

```
   ┌────────────┐  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐
   │ Pillar 1   │  │ Pillar 2     │  │ Pillar 3     │  │ Pillar 4     │
   │ FORMAT     │  │ RECIPROCITY  │  │ AUTHORITY    │  │ PATTERN      │
   │ ตรงรูปแบบ  │  │ in↔out match │  │ แหล่งเชื่อถือ │  │ จำนวนผู้แต่ง  │
   └────────────┘  └──────────────┘  └──────────────┘  └──────────────┘
```

---

### Pillar 1 — Format Verification (รูปแบบ)

ตรวจว่ารายการแต่ละรายการตรงกับ Master Template ใน PART A

#### Pillar 1 Checklist

ตรวจ ๘ รายการสำหรับทุก Citation

| # | จุดตรวจ | รายละเอียด | ❌ ผิดบ่อย |
|---|---|---|---|
| 1 | ลำดับองค์ประกอบ | ผู้แต่ง → (ปี) → ชื่อ → สถานที่ → สำนักพิมพ์ | สลับลำดับ |
| 2 | เครื่องหมายวรรคตอน | จุด ระหว่างองค์ประกอบ / จุลภาค ระหว่างเล่ม-ฉบับ-หน้า | ใช้ดอก ; แทน . |
| 3 | การเว้นวรรค | เว้น ๑ ตัวอักษรหลัง . และ , | ติดกัน หรือเว้นเกิน |
| 4 | ปีพ.ศ. vs ค.ศ. | ภาษาไทย = พ.ศ. / ภาษาอังกฤษ = ค.ศ. | ปนกัน |
| 5 | สถานที่พิมพ์ | ระบุเมือง — ไม่ใช่ประเทศ — ตามด้วย `:` | ใช้ประเทศ |
| 6 | ครั้งที่พิมพ์ | ใส่เฉพาะ ≥ 2 / Thai = "พิมพ์ครั้งที่ X" / Eng = "Xth ed." | ใส่ "พิมพ์ครั้งที่ 1" |
| 7 | เลขหน้าวารสาร | `pp-pp` ไม่มีคำว่า "หน้า" / "p." | ใส่ "หน้า 25-31" |
| 8 | URL — สื่อ Online | เก็บ URL เต็มไม่ตัด — ระบุวันที่สืบค้น | ใส่แค่ "เว็บไซต์" |

#### Pillar 1 Decision Rule

ถ้ามีจุดผิดแม้ ๑ จุดในรายการใด → **FAIL Pillar 1** สำหรับรายการนั้น

---

### Pillar 2 — Reciprocity Verification (การจับคู่ในเนื้อหา ↔ ท้ายเล่ม)

ตรวจว่า Citation ในเนื้อหาตรงกับรายการท้ายเล่มแบบ One-to-One

#### Pillar 2 Workflow

ขั้นที่หนึ่ง สร้าง **Set A** = Citation ทั้งหมดในเนื้อหา (เฉพาะคีย์ ผู้แต่ง+ปี)
ขั้นที่สอง สร้าง **Set B** = รายการทั้งหมดท้ายเล่ม (เฉพาะคีย์ ผู้แต่ง+ปี)
ขั้นที่สาม ตรวจ ๓ ทาง

```
A ⊆ B  →  ทุก Citation ในเนื้อหา มีรายการท้ายเล่ม
B ⊆ A  →  ทุกรายการท้ายเล่ม ถูกอ้างในเนื้อหา
A = B  →  ผ่าน Reciprocity
```

#### Pillar 2 Failure Modes

**Ghost Citation (A ⊄ B)** — มีในเนื้อหาแต่ไม่มีท้ายเล่ม → ต้องเพิ่มท้ายเล่มหรือลบจากเนื้อหา

**Orphan Reference (B ⊄ A)** — มีท้ายเล่มแต่ไม่มีในเนื้อหา → ต้องอ้างในเนื้อหาหรือลบจากท้ายเล่ม

**Year Mismatch** — เนื้อหาอ้าง 2562 แต่ท้ายเล่มเขียน 2563 → ต้องแก้ให้ตรง

**Author Name Mismatch** — เนื้อหา "พระมหาสุนันท์" ท้ายเล่ม "สุนันท์ พระมหา" → ต้องเรียงรูปแบบเดียวกัน

#### Pillar 2 Decision Rule

ถ้ามี Ghost ≥ 1 หรือ Orphan ≥ 1 หรือ Mismatch ≥ 1 → **FAIL Pillar 2**

---

### Pillar 3 — Authority Verification (ความน่าเชื่อถือของแหล่ง)

ตรวจว่าแหล่งที่อ้างมีอยู่จริง สืบค้นได้ และอยู่ในกรอบที่วารสารยอมรับ

#### Pillar 3 Checklist

| # | จุดตรวจ | เกณฑ์ | สำหรับ |
|---|---|---|---|
| 1 | พระไตรปิฎกฉบับ | ใช้ฉบับ มจร 2539 เท่านั้น (ค.ศ. 1996) | ทุกการอ้างพระไตรปิฎก |
| 2 | เลขย่อคัมภีร์ | ตรงกับตารางใน A.4 หรือพจนานุกรมพุทธศาสตร์ | ทุกการอ้างพระไตรปิฎก |
| 3 | เลขเล่ม-ข้อ-หน้า | สืบค้นได้จริงในเล่มที่ระบุ | ทุกการอ้างพระไตรปิฎก |
| 4 | Citation ๕ ปีล่าสุด | ≥ 50% ของรายการอ้างอิง (TCI Tier 1) | บทความ JPSPA ทุกฉบับ |
| 5 | Citation นานาชาติ | 30-50% ของรายการอ้างอิง (TCI Tier 1) | บทความ JPSPA ทุกฉบับ |
| 6 | แหล่งสืบค้นได้ | หนังสือ/วารสาร/ดุษฎีนิพนธ์ ตีพิมพ์จริง | ทุกรายการ |
| 7 | URL ใช้งานได้ | เว็บไซต์ไม่ Dead Link ณ วันสืบค้น | สื่ออิเล็กทรอนิกส์ |
| 8 | DOI/ISBN ถ้ามี | ระบุเพื่อความแม่นยำ (Optional แต่ Recommended) | บทความนานาชาติ |

#### Pillar 3 Hallucination Guard

ห้าม Fabricate ๖ องค์ประกอบนี้เด็ดขาด

1. ชื่อผู้แต่งที่ไม่มีตัวตน
2. ปีพิมพ์ที่ไม่ตรงกับเล่มจริง
3. เลขเล่ม-ข้อ-หน้าพระไตรปิฎกปลอม
4. ชื่อวารสารปลอม
5. ชื่อสำนักพิมพ์ปลอม
6. URL ที่ไม่เคยมีอยู่

> **กฎเหล็ก:** หากไม่แน่ใจว่าแหล่งอ้างมีอยู่จริง — ตัดออกจากบทความ ดีกว่าใส่ Citation ปลอม

#### Pillar 3 Decision Rule

- ผิดข้อ 1-3 ของพระไตรปิฎก → **FAIL ทันที** (Buddhist Citation Authority)
- 5-Year Recency < 50% → **FAIL** TCI Tier 1
- International Citation < 30% → **FAIL** TCI Tier 1
- พบ Hallucination แม้ ๑ รายการ → **FAIL Critical** (ฝ่าฝืนจริยธรรมร้ายแรง)

---

### Pillar 4 — Pattern Verification (รูปแบบจำนวนผู้แต่ง)

ตรวจว่า Citation จำนวนผู้แต่งใช้รูปแบบถูกต้องตาม A.1.2.2 ถึง A.1.2.8

#### Pillar 4 Decision Tree

```
จำนวนผู้แต่ง?
├── ๑ ราย
│   ├── ภาษาไทย → (ชื่อ-นามสกุล, ปี)
│   └── ภาษาอังกฤษ → (Lastname, Year)
├── ๒ ราย
│   ├── ภาษาไทย → (ชื่อ1 และชื่อ2, ปี)        ← ใช้ "และ"
│   └── ภาษาอังกฤษ → (Last1 & Last2, Year)   ← ใช้ "&"
└── > ๒ ราย
    ├── ภาษาไทย → (ชื่อแรกและคณะ, ปี)
    └── ภาษาอังกฤษ → (Last1 et al., Year)    ← มีจุดท้าย al.
```

#### Pillar 4 Failure Modes

**ผิด ๑** — ใช้ `และ` ในภาษาอังกฤษ → ต้องเป็น `&`
**ผิด ๒** — ใช้ `and` ในภาษาอังกฤษ Parenthetical → ต้องเป็น `&` (ใช้ `and` เฉพาะใน Narrative)
**ผิด ๓** — ลืม `.` หลัง `al` → ต้องเป็น `et al.`
**ผิด ๔** — Italic `et al.` ในเนื้อหา → ไม่ต้อง Italic ตามมาตรฐานวารสาร
**ผิด ๕** — list ผู้แต่งทุกคนเมื่อ > 2 → ใช้คนแรก + "และคณะ" / "et al."
**ผิด ๖** — เนื้อความเดียวกันมีผู้อ้างหลายคน list ทุกคน → เลือกที่ใกล้ปีปัจจุบันที่สุด

#### Pillar 4 Decision Rule

ผิดรูปแบบ ≥ 3 ครั้งในบทความ → **FAIL Pillar 4** (สัญญาณว่าผู้เขียนยังไม่คุ้นเคยกฎ)

### Phase 2 — 4-PILLAR AUDIT

#### Phase 2.1 — Pillar 1 Format Audit

วน Loop ผ่านทุกรายการในตาราง Reference List Inventory เทียบกับ Master Template ที่ตรง (A.2.x หรือ A.3.x)

|---|---|---|---|

| #12 | ใช้ "หน้า 37-53" ในวารสาร | A.3.5 Journal Articles | ตัด "หน้า" เหลือ "37-53" |

| Ghost Citation | สมชาย รักดี 2563 | อ้างในเนื้อหา ไม่มีในท้ายเล่ม | เพิ่มในท้ายเล่ม |
| Orphan Reference | Smith 2018 | มีท้ายเล่ม ไม่อ้างในเนื้อหา | ลบหรืออ้างในเนื้อหา |
| Year Mismatch | พระมหาสุทิตย์ | เนื้อหา 2548 / ท้ายเล่ม 2549 | แก้ให้ตรง |

#### Phase 2.3 — Pillar 3 Authority Audit

ตรวจ ๓ ส่วน

(ก) **Buddhist Authority** — ทุกการอ้างพระไตรปิฎกใช้ฉบับ มจร 2539 และเลขเล่ม-ข้อ-หน้าตรงกับเล่มจริง

(ข) **Recency Ratio** — นับ Citation ทั้งหมด แล้วคำนวณ

```
Recency_5y = (รายการที่ปีพิมพ์ ≥ ปัจจุบัน - 5) / รายการทั้งหมด × 100
International = (รายการภาษาอังกฤษ) / รายการทั้งหมด × 100
```

เกณฑ์ TCI Tier 1: Recency ≥ 50%, International 30-50%

(ค) **Hallucination Probe** — สุ่มเลือก 3-5 รายการที่ "ผู้เขียนไม่แน่ใจว่ามีอยู่จริง" แล้วสืบค้นใน Google Scholar / TCI / WorldCat / Library Catalog

#### Phase 2.4 — Pillar 4 Pattern Audit

วนทุก Citation ในเนื้อหา ตรวจรูปแบบจำนวนผู้แต่งตาม Decision Tree ใน Pillar 4

|---|---|---|
| (Hersey และ Blanchard, 2010) | ใช้ "และ" ในภาษาอังกฤษ | (Hersey & Blanchard, 2010) |
| (Kaiser et al, 2010) | ลืมจุดท้าย al | (Kaiser et al., 2010) |

## PART D — COMMON ERRORS & FIXES

รวบรวมความผิดที่ Reviewer JPSPA พบบ่อยที่สุด พร้อมการแก้

### D.1 Format Errors (Pillar 1)

## PART E — PRE-SUBMISSION CITATION CHECKLIST

ใช้ Checklist นี้เป็น Last Mile ก่อน Submit ผ่าน ThaiJO ตอบ "ใช่" ทุกข้อจึงผ่าน Gate 3

### E.1 Format & Style (Pillar 1)

- [ ] ทุกรายการในท้ายเล่มตรง Master Template (A.2.1-A.2.12 / A.3.1-A.3.9)
- [ ] เครื่องหมายวรรคตอน . , : สม่ำเสมอทั่วเอกสาร
- [ ] เว้นวรรคหลัง . และ , ทุกจุด
- [ ] ปี พ.ศ. ในรายการไทย / ค.ศ. ในรายการอังกฤษ ไม่ปนกัน
- [ ] สถานที่พิมพ์ระบุเมือง ไม่ใช่ประเทศ
- [ ] เลขหน้าวารสารไม่มีคำว่า "หน้า" / "p." (รูปแบบ pp-pp เท่านั้น)
- [ ] ครั้งที่พิมพ์ใส่เฉพาะ ≥ 2 (พิมพ์ครั้งแรกไม่ระบุ)
- [ ] URL ไม่ตัด สั้นด้วย bit.ly หรือ goo.gl

### E.2 Reciprocity (Pillar 2)

- [ ] Set A (In-text) ⊆ Set B (Reference List) — ไม่มี Ghost
- [ ] Set B ⊆ Set A — ไม่มี Orphan
- [ ] ปีในเนื้อหาตรงกับปีในท้ายเล่มทุกราย
- [ ] ชื่อผู้แต่งสะกดตรงกันทุกครั้งที่ปรากฏ
- [ ] ลำดับชื่อ-นามสกุลของผู้แต่งร่วมตรงกัน

### E.3 Authority (Pillar 3)

- [ ] ทุกการอ้างพระไตรปิฎกใช้ฉบับ มจร 2539 / ค.ศ. 1996
- [ ] เลขย่อคัมภีร์ตรงตารางใน A.4 หรือพจนานุกรมพุทธศาสตร์
- [ ] เลขเล่ม-ข้อ-หน้าสืบค้นได้จริงในเล่ม
- [ ] Recency 5 ปีล่าสุด ≥ 50%
- [ ] International Citation 30-50%
- [ ] ไม่มี Hallucinated Author / Year / Page / URL
- [ ] URL ทุกรายการเปิดได้จริง ณ วัน Submit
- [ ] DOI/ISBN ระบุเมื่อมี (Recommended)

### E.4 Pattern (Pillar 4)

- [ ] ผู้แต่ง 1 ราย ใช้ (ชื่อ, ปี) / (Lastname, Year)
- [ ] ผู้แต่ง 2 ราย ภาษาไทยใช้ "และ" / ภาษาอังกฤษใช้ "&" ใน Parenthetical
- [ ] ผู้แต่ง > 2 ราย ภาษาไทยใช้ "และคณะ" / ภาษาอังกฤษใช้ "et al."
- [ ] "et al." มีจุดท้าย al ทุกครั้ง — ไม่ Italic
- [ ] Narrative ใช้ "and" / Parenthetical ใช้ "&" ในภาษาอังกฤษ
- [ ] เนื้อความเดียวกันที่มีหลายแหล่งอ้างใช้แหล่งใกล้ปีปัจจุบันที่สุด

### E.5 Final Verification

- [ ] รัน CopyCat (ThaiJO) ≤ 15%
- [ ] รัน thesis-ai-det-col Mode 1 — AI Score ≤ 20%
- [ ] Citation Audit Report ลงนามผู้ตรวจ + วันที่
- [ ] ผ่านทั้ง ๔ Pillar (PASS) — verdict "READY FOR SUBMISSION"
