## PART A — สรุปองค์ความรู้ปัจจุบัน (Current Knowledge)

จากวรรณกรรมที่อ่าน (ผ่าน Stage 2) ระบุ: วงวิชาการ *รู้อะไรแล้ว* · ประเด็นที่ข้อค้นพบ *สอดคล้องกัน* · ประเด็นที่ *ขัดแย้งกัน* · ประเด็นที่ *หลักฐานยังไม่พอ*. ข้อค้นพบที่ขัดแย้งกัน = แหล่งของ Research-Gap ที่ดี (ระบุชัดว่าเป็น gap หรือไม่ เพราะอะไร)

<a id="part-b"></a>

## PART B — เช็คลิสต์ช่องว่างการวิจัย 7 ชนิด (เลือก 1–2 ที่ใช่ที่สุด)

ถามว่า gap ของบทความเข้าชนิดใด — *เลือกเท่าที่มีหลักฐานจริง ไม่ครบ 7 ก็ได้*

| ชนิด Gap | คำถามวินิจฉัย |
|---|---|
| **Theoretical** | ทฤษฎีเดิมอธิบายปรากฏการณ์นี้ไม่ครบ/ขัดกัน? |
| **Conceptual** | นิยาม/มโนทัศน์ยังคลุมเครือ/ไม่ลงรอย? |
| **Methodological** | วิธีศึกษาเดิมมีข้อจำกัด? |
| **Empirical/Contextual** | ยังไม่ศึกษาในบริบทไทย/พื้นที่/ช่วงเวลานี้? |
| **Population** | กลุ่มประชากร/หน่วยศึกษานี้ถูกศึกษาน้อย? |
| **Practical/Policy** | ยังขาดข้อเสนอเชิงปฏิบัติ/นโยบายที่ใช้ได้จริง? |
| **Temporal** | องค์ความรู้เก่า/ไม่ทันบริบทปัจจุบัน? |

## PART C — Novelty Statement + Counterargument (เกณฑ์ TCI Tier 1)

JPSPA เป็น TCI Tier 1 ต้องการ **Synthesis + Counterargument** (RULE 10) — gap ที่ระบุจึงต้องนำไปสู่ 2 สิ่งนี้:

1. **Novelty Statement (สั้น)** — "บทความนี้เติมช่องว่าง [ชนิด gap] โดย [มุม/กรอบที่ใหม่]" → เป็นแกนของ **องค์ความรู้ใหม่ §9**
2. **Counterargument (≥1 จุด)** — มุมมองที่อาจโต้แย้ง gap/ข้อเสนอนี้ + การตอบโต้ → กระจายในเนื้อหา §8 (Analysis Depth)

> *Light+ : ไม่ทำ Top-5/Matrix — แต่ Novelty + Counterargument คือสิ่งที่ TCI Tier 1 บังคับ ต้องมี*

## PART D — Bridge เข้า Lifecycle / โครง 9 ส่วน + Guard

**Bridge → Lifecycle + โครงบทความ 9 ส่วน (บังคับ — ห้ามข้าม):**

| ผลจาก Research-Gap | ป้อนเข้าตรงไหน |
|---|---|
| gap ที่เลือก (1–2 ชนิด) | **Stage 1 Topic & Framing** (Gate 1) → เขียนใน **บทนำ §7** (ทบทวนแนวคิด + ชี้ช่องว่าง) |
| Novelty Statement | หนุน **องค์ความรู้ใหม่ §9** (โมเดล/แผนภาพ + คำอธิบาย) |
| Counterargument | กระจายในเนื้อหา **§8** (≥1 จุดต่อประเด็น ตาม TCI Tier 1) |

**Guard (Anti-Hallucination — ผูกกับ governance jpspa):**
- **วิเคราะห์เฉพาะข้อมูล/วรรณกรรมที่มี** (ผ่าน Stage 2) — ห้ามแต่งชื่องานวิจัย/บทความ/ผลการวิจัย (RULE 2 Source Integrity)
- ข้อมูลไม่พอ/สรุปไม่ได้ → ระบุ **"ต้องตรวจโดยผู้วิจัย"** หรือ **"ข้อมูลไม่ปรากฏในเอกสาร"**
- **แยก 3 ชั้น:** ข้อค้นพบจากวรรณกรรม · การตีความเชิงวิเคราะห์ (ของ Claude) · ข้อเสนอสำหรับวิจัยอนาคต
- gap/Novelty ต้องสอดคล้องเกณฑ์ Citation (RULE 13 4-Pillar + Recency ≥ 50% / International 30–50%)
