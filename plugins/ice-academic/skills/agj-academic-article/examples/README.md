# Examples — ตัวอย่างบทความจากวารสาร AGJ

วาง PDF บทความตัวอย่างจากวารสารบัณฑิตศึกษาวิชาการในโฟลเดอร์นี้
เพื่อใช้เป็นแหล่งอ้างอิงรูปแบบและเนื้อหาเมื่อเขียนบทความใหม่

## วิธีใช้

- เมื่อต้องการดูตัวอย่างการเขียนจริง → อ่านไฟล์ในโฟลเดอร์นี้
- ใช้เป็นแหล่งอ้างอิงรูปแบบ ไม่ใช่คัดลอก
- ทฤษฎีและอ้างอิงที่ปรากฏในตัวอย่าง สามารถนำไปใช้อ้างอิงในบทความใหม่ได้

## ⚠️ กฎ
- ห้ามค้นหาอ้างอิงจากภายนอกโดยไม่ถามผู้ใช้ก่อน
- หากต้องการอ้างอิงเพิ่ม → ใช้จากไฟล์ในโฟลเดอร์นี้ หรือจาก references/ ก่อน
- หากไม่พบ → ใส่ `[ต้องเติมอ้างอิงจริง]` แทน
