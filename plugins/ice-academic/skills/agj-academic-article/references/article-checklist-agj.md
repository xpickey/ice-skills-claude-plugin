# Checklist ก่อนส่งวารสาร AGJ

> **Checklist นี้ = เกณฑ์ "อะไรต้องมี" ของ AGJ (source-of-truth ของวารสารนี้).**
> **การตรวจ "เชิงเทคนิคก่อนส่ง" (AI signature · citation cross-check 2 ทิศ · font/PDF · ข้อความซ้ำ · negative→positive wording)** → ใช้ **Thai Academic Audit Engine** ที่ `~/.claude/skills/thesis-ai-det-col/references/10_academic_audit_engine.md` (ผ่าน qa-master Academic QA Mode หรือ ผู้ทรง/สมนึก). Engine จะ "Resolve Standard" มาดึงเกณฑ์จาก checklist นี้เป็น L1 ก่อนตรวจ — เกณฑ์ตัวเลขจริง (≤8000 คำ, บทคัดย่อ 300-400, ≥15 อ้างอิง, TH Sarabun PSK) อยู่ในไฟล์นี้ ไม่ได้ฝังใน engine.

## A. โครงสร้างบทความวิชาการ

- [ ] ชื่อเรื่องภาษาไทย สั้น ชัด ตรงเนื้อหา
- [ ] ชื่อเรื่องภาษาอังกฤษ (ตัวพิมพ์ใหญ่ทั้งหมด)
- [ ] ชื่อผู้แต่ง + สังกัด (ไทย+อังกฤษ) + Corresponding Author Email
- [ ] บทคัดย่อภาษาไทย (300-400 คำ) ครอบคลุม 4 ส่วน
- [ ] คำสำคัญ 3-5 คำ
- [ ] Abstract (แปลตามภาษาไทย)
- [ ] Keywords 3-5 words
- [ ] บทนำ — ชี้ความสำคัญ ช่องว่าง วัตถุประสงค์ ขอบเขต
- [ ] เนื้อหา — มีการวิเคราะห์ของผู้เขียน ไม่ใช่แค่รวบรวม
- [ ] องค์ความรู้ใหม่ — โมเดล/แผนภาพ + คำอธิบาย
- [ ] สรุป — ตอบประเด็นหลักของเรื่อง สังเคราะห์ภาพรวม
- [ ] เอกสารอ้างอิง — ≥15 รายการ ไทยก่อนอังกฤษ

## B. รูปแบบเอกสาร

- [ ] กระดาษ A4 ขาว
- [ ] ขอบ 1 นิ้ว ทุกด้าน
- [ ] Font: TH Sarabun PSK ตลอดเอกสาร
- [ ] Single spacing
- [ ] ย่อหน้า เว้น 5 ตัวอักษร
- [ ] ไม่ใส่เลขหน้า
- [ ] ความยาว 8-15 หน้า (รวมทุกอย่าง)
- [ ] ≤ 8,000 คำ
- [ ] ไฟล์ .docx (Word 2010+)
- [ ] ขนาดตัวอักษรตรงตาม template

## C. เอกสารอ้างอิง

- [ ] รูปแบบ APA นาม-ปี ถูกต้อง
- [ ] อ้างในเนื้อหา ↔ ท้ายบทความ ตรงกันทุกรายการ
- [ ] ไม่มีรายการในเนื้อหาที่ไม่อยู่ท้ายบทความ
- [ ] ไม่มีรายการท้ายบทความที่ไม่ถูกอ้างในเนื้อหา
- [ ] อ้างอิงพระไตรปิฎกถูกรูปแบบ (ถ้ามี)
- [ ] เรียง: ไทยก่อนอังกฤษ ตามลำดับอักษร
- [ ] ชื่อผู้แต่ง ปี ชื่อเรื่อง ถูกต้อง

## D. คุณภาพเนื้อหา (7 Cs)

- [ ] **Correctness**: ข้อมูล ตัวเลข ทฤษฎี ชื่อ ปี ถูกต้อง
- [ ] **Cogency**: ข้อสรุปทุกข้อมีเหตุผลรองรับ
- [ ] **Clarity**: ไม่มีประโยคกำกวม อ่านเข้าใจง่าย
- [ ] **Completeness**: โครงสร้างครบ เนื้อหาครบ
- [ ] **Concise**: ไม่มีคำหรือประโยคที่ตัดได้โดยไม่เสียความหมาย
- [ ] **Consistency**: คำสำคัญใช้เหมือนกันตลอด
- [ ] **Concatenation**: ทุกย่อหน้าเชื่อมกับคำถามหลักของบทความ

## E. จริยธรรม

- [ ] ไม่เคยตีพิมพ์ที่ใดมาก่อน
- [ ] ไม่คัดลอกผลงานผู้อื่น
- [ ] CopyCat ≤ 25%
- [ ] ข้อมูลเป็นจริง ไม่สร้างข้อมูลเท็จ
- [ ] ระบุแหล่งทุนสนับสนุน (ถ้ามี)
- [ ] ยินยอมให้เป็นลิขสิทธิ์วารสาร (CC BY)

## F. การส่ง

- [ ] ส่งผ่าน ThaiJO: https://so18.tci-thaijo.org/index.php/A_GJ/about/submissions
- [ ] กรอก Metadata ครบ (ชื่อ นามสกุล สังกัด อีเมล)
- [ ] แจ้งเพิ่มเติมทาง Email: journal.agj@mcu.ac.th
- [ ] ชำระค่าธรรมเนียม 3,000 บาท **หลัง accepted เท่านั้น**
