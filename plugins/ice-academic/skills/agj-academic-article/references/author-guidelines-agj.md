# คำแนะนำผู้เขียน + จริยธรรม — วารสาร AGJ

## โครงสร้างบทความวิชาการ (เฉพาะ)

บทความวิชาการให้เรียงลำดับสาระดังนี้:
1. ชื่อเรื่องภาษาไทย ภาษาอังกฤษ
2. ชื่อผู้แต่ง หน่วยงานต้นสังกัด และอีเมล
3. บทคัดย่อภาษาไทย
4. คำสำคัญ
5. Abstract
6. Keywords
7. บทนำ
8. เนื้อหา
9. สรุป
10. เอกสารอ้างอิง

---

## ไฟล์ที่ส่ง

- ต้นฉบับ: **.docx** (Microsoft Word 2010+)
- หากมีภาพ/ตาราง/สมการ: ส่งแยกเป็นไฟล์ *.PDF / *.JPG / *.GIF / *.bmp
- ความยาว: ≤ 15 หน้า A4 (ไม่รวมเอกสารอ้างอิง)

## ภาพและตาราง

### ภาพ
```
ภาพที่ X ชื่อภาพ (16pt)
ที่มา: (ถ้ามี) (16pt)
คำอธิบายภาพ (ถ้ามี) (16pt)
```

### ตาราง
```
ตารางที่ X ชื่อตาราง (16pt)
[ตาราง]

```

## จริยธรรมผู้เขียน

1. ผู้เขียนต้องรับรองว่าผลงาน **ไม่เคยตีพิมพ์หรือเผยแพร่ที่ใดมาก่อน**
2. ต้อง **ไม่ละเมิดลิขสิทธิ์หรือคัดลอก** ผลงานผู้อื่น
3. ผ่านการตรวจ CopyCat (ThaiJO) **ไม่เกิน 25%**
4. ข้อมูลต้องเป็นจริง **ไม่สร้างข้อมูลเท็จ ไม่ปลอมแปลง บิดเบือน**
5. ทัศนะในบทความเป็นความรับผิดชอบของผู้เขียน ไม่ใช่ของกองบรรณาธิการ
6. ต้องปรับแก้ตามผลประเมินจากผู้ทรงคุณวุฒิ
7. ระบุแหล่งทุนสนับสนุน (ถ้ามี)
8. ยินยอมให้บทความเป็นลิขสิทธิ์ของวารสาร (CC BY)
9. ต้องกรอก Metadata (ชื่อ นามสกุล สังกัด อีเมล) ให้ครบถ้วนถูกต้อง

## กระบวนการพิจารณา

1. กองบรรณาธิการพิจารณาเบื้องต้น (รูปแบบ)
2. ไม่ผ่าน → ส่งกลับแก้ไข
3. ผ่าน → ส่งผู้ทรงคุณวุฒิ ≥ 3 ท่าน (Double-Blind)
4. ผลประเมิน: ผ่าน / ไม่ผ่าน / แก้ไข
5. ยอมรับ (accepted) → เก็บค่าธรรมเนียม → ตีพิมพ์
6. ได้รับลิงก์ฉบับตีพิมพ์ + หนังสือรับรอง
