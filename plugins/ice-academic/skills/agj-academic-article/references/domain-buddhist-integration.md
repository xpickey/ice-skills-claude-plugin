## ตัวอย่าง Theory-to-Dhamma Mapping

| ทฤษฎี PA/PolSci | หลักธรรมที่จับคู่ | เหตุผล |

## วิธีเขียนเชิงพุทธบูรณาการ (ไม่ใช่เชิงเทศนา)

### ควรทำ:
- ใช้หลักธรรมเป็น **กรอบวิเคราะห์** (Analytical Framework) ไม่ใช่เนื้อหาเทศนา
- อธิบายว่าหลักธรรมนั้น **สอดคล้องหรือขยายความ** ทฤษฎีสากลอย่างไร
- เปรียบเทียบจุดแข็ง-จุดอ่อนของกรอบพุทธกับกรอบสากล
- อ้างอิงพระไตรปิฎกอย่างเป็นระบบ

### ไม่ควรทำ:
- เขียนในน้ำเสียงสอนธรรมะหรือเทศนา
- สรุปว่าพุทธดีกว่าทุกอย่างโดยไม่วิเคราะห์
- ใช้ภาษาศาสนาล้วนๆ โดยไม่เชื่อมกับภาษาวิชาการ

## การอ้างอิงพระไตรปิฎก

### ในเนื้อหา:
```
"ข้อความจากพระไตรปิฎก" (องฺ.จตุกก. 21/31/37) (มหาจุฬาลงกรณราชวิทยาลัย, 2539)
```

### ท้ายบทความ:
```
มหาจุฬาลงกรณราชวิทยาลัย. (2539). พระไตรปิฎกฉบับภาษาไทย ฉบับมหาจุฬาลงกรณราชวิทยาลัย. กรุงเทพฯ: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย.
```

### ย่อชื่อคัมภีร์ที่พบบ่อย:
- องฺ.จตุกก. = อังคุตตรนิกาย จตุกกนิบาต
- ที.ปา. = ทีฆนิกาย ปาฏิกวรรค
- ม.มู. = มัชฌิมนิกาย มูลปัณณาสก์
- สํ.ม. = สังยุตตนิกาย มหาวารวรรค
- ขุ.ธ. = ขุททกนิกาย ธรรมบท
