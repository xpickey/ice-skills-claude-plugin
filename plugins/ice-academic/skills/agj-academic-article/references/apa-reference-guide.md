- ใช้ระบบ **นาม-ปี** ตาม American Psychological Association (APA)

## 2. เอกสารอ้างอิงท้ายเล่ม (ภาษาไทย)

### 2.1 พระไตรปิฎก อรรถกถา
```
ผู้แต่ง. (ปีที่พิมพ์). ชื่อพระไตรปิฎก อรรถกถา. สถานที่พิมพ์: สำนักพิมพ์.
```
ตัวอย่าง:
```
มหาจุฬาลงกรณราชวิทยาลัย. (2539). พระไตรปิฎกฉบับภาษาไทย ฉบับมหาจุฬาลงกรณราชวิทยาลัย. กรุงเทพฯ: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย.
```

### 2.2 หนังสือ
```
ผู้แต่ง. (ปีที่พิมพ์). ชื่อหนังสือ (ครั้งที่พิมพ์). สถานที่พิมพ์: สำนักพิมพ์.
```

### 2.3 บทความในหนังสือ
```
ผู้แต่ง. (ปีที่พิมพ์). ชื่อบทความ. ใน ชื่อบรรณาธิการ, ชื่อเรื่อง. (น. เลขหน้า). สถานที่พิมพ์: สำนักพิมพ์.
```

### 2.4 บทความในวารสาร
```
ผู้แต่ง. (ปีที่พิมพ์). ชื่อบทความ. ชื่อวารสาร, ปีที่(ฉบับที่), หน้าแรก-หน้าสุดท้าย.
```
ตัวอย่าง:
```
ธิติวุฒิ หมั่นมี. (2557). การวางแผนและการติดต่อประสานงานเชิงพุทธ. วารสารบัณฑิตศึกษาวิชาการ, 3(1), 25-31.
```

### 2.5 สารานุกรม
```
ผู้แต่ง. (ปีที่พิมพ์). ชื่อบทความ. ใน ชื่อสารานุกรม (เล่มที่อ้าง, หน้า เลขหน้าที่อ้าง).
```

### 2.6 หนังสือพิมพ์
```
ผู้แต่ง. (ปีที่พิมพ์, วันที่ เดือน). ชื่อบทความ. ชื่อหนังสือพิมพ์. น. เลขหน้า.
```

### 2.7 สารนิพนธ์ / วิทยานิพนธ์ / ดุษฎีนิพนธ์
```
ผู้แต่ง. (ปีที่พิมพ์). ชื่อวิทยานิพนธ์ (ระดับวิทยานิพนธ์ สาขาวิชา). สถานที่พิมพ์: สำนักพิมพ์.
```

### 2.8 รายงานการวิจัย
```
ผู้แต่ง. (ปีที่พิมพ์). ชื่อเรื่อง (รายงานผลการวิจัย). สถานที่พิมพ์: สำนักพิมพ์.
```

### 2.9 สัมภาษณ์
```
ชื่อผู้สัมภาษณ์. (ปีที่สัมภาษณ์, วัน เดือน). ตำแหน่ง [บทสัมภาษณ์].
```

### 2.10 สื่ออิเล็กทรอนิกส์
```
ผู้แต่ง. (ปีที่เผยแพร่). ชื่อเรื่อง. สืบค้นเมื่อ วัน เดือน ปี จาก URL.
```

### 2.11 ราชกิจจานุเบกษา
```
ชื่อเรื่อง. (พ.ศ., วันที่ เดือนที่ประกาศ). ราชกิจจานุเบกษา. เล่ม ตอน. หน้า.
```

## 3. เอกสารอ้างอิงท้ายเล่ม (ภาษาอังกฤษ)

### 3.1 Tripitaka
```
Institution Name. (Year). Title. Location: Publisher.
```

### 3.2 Books
```
Author. (Year). Title (Edition ed.). Location: Publisher.
```

### 3.3 Book Articles
```
Author. (Year). Title of chapter. In Editor (Eds.), Title of book (xth ed.). (pp-pp). Publisher.
```

### 3.4 Journal Articles
```
Author. (Year). Title of Article. Title of Journal, Volume(Issue), XX-XX.
```

### 3.5 Newspaper
```
Author. (Year, Month Day). Column Name. Title of Newspaper. p. xx.
```

### 3.6 Thesis/Dissertations
```
Author. (Year). Title (Doctoral/Master's Thesis). Location: Publisher.
```

### 3.7 Interview
```
Key Informant. (Year, Month Day). Position. (Interview).
```

### 3.8 Website
```
Author. (Year). Title. Retrieved Month Day, Year, from URL.
```

## การจัดเรียงเอกสารอ้างอิง

1. **ภาษาไทยก่อน** → ภาษาอังกฤษ
2. เรียงตามลำดับอักษร (ก-ฮ / A-Z)
3. ผู้แต่งคนเดียวกันหลายเรื่อง: เรียงตามปีจากเก่าไปใหม่
4. ผู้แต่งคนเดียวกันปีเดียวกัน: ใส่ ก ข ค... หรือ a b c... ต่อท้ายปี
