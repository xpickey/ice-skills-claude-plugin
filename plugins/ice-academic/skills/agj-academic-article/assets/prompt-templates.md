# Prompt Templates สำเร็จรูป — 5 ชุด

อ้างอิง: พระมหาสมชาย ขนฺติสรโณ (มะลิซ้อน), ดร.
ปรับสำหรับ: วารสารบัณฑิตศึกษาวิชาการ (AGJ) — บทความวิชาการเท่านั้น

---

## ชุดที่ 1: Prompt ตั้งต้น

### 1.1 ตั้งบทบาท AI
```
ให้คุณทำหน้าที่เป็นผู้ช่วยเขียนบทความวิชาการภาษาไทยในสาขา [สาขาวิชา]
สำหรับวารสารบัณฑิตศึกษาวิชาการ (AGJ, TCI Tier 2)
ใช้น้ำเสียงแบบวิชาการ เน้นการเขียนที่ชัดเจน กระชับ เป็นระบบ มีการวิเคราะห์ มีข้อสังเคราะห์ของผู้เขียน
เงื่อนไข: ห้ามสร้างข้อมูลเท็จ ห้ามแต่งอ้างอิงปลอม หากข้อมูลไม่พอให้ระบุว่าต้องเติมข้อมูลจริง
ทุกคำตอบต้องคำนึงถึงหลัก 7 Cs
```

### 1.2 ตั้งหัวข้อและขอบเขตงาน
```
ฉันต้องการเขียนบทความวิชาการเรื่อง [ชื่อเรื่องเบื้องต้น]
สาขา [สาขาวิชา]
เน้นประเด็น [ประเด็นหลัก]
ใช้กรอบแนวคิด [กรอบแนวคิด/ทฤษฎี]
กลุ่มผู้อ่านเป้าหมายคือ [เช่น นักวิชาการ / ครู / นักบริหาร / พระสงฆ์]
วารสารเป้าหมายคือ วารสารบัณฑิตศึกษาวิชาการ (AGJ)
ความยาวประมาณ [จำนวนหน้า] หน้า A4
กรุณาวิเคราะห์ความเหมาะสมของหัวข้อนี้ พร้อมเสนอแนวทางพัฒนาให้เป็นบทความวิชาการที่ชัดเจนและตีพิมพ์ได้
```

## ชุดที่ 2: Prompt สร้างบทความทีละส่วน

### 2.1 คิดชื่อบทความ
```
จากประเด็นต่อไปนี้:
- เรื่องหลัก: [เรื่องหลัก]
- ประเด็นย่อย: [ประเด็นย่อย]
- กรอบแนวคิด: [กรอบแนวคิด]
- บริบท: [บริบทหรือพื้นที่ศึกษา]
ช่วยตั้งชื่อบทความวิชาการภาษาไทยจำนวน 10 ชื่อ และชื่อภาษาอังกฤษที่สอดคล้องกัน
เงื่อนไข: สั้นกระชับ ชัดเจน สะท้อนแก่นบทความ เหมาะกับวารสาร AGJ
```

### 2.2 วางโครงบทความทั้งฉบับ
```
ช่วยวางโครงบทความวิชาการทั้งฉบับ เรื่อง "[ชื่อเรื่อง]"
สาขา [สาขาวิชา] ใช้กรอบแนวคิด [กรอบแนวคิด]
ความยาวประมาณ [จำนวนหน้า] หน้า
ขอให้ประกอบด้วย: ชื่อบทความไทย/อังกฤษ, บทคัดย่อ, คำสำคัญ, บทนำ, หัวข้อเนื้อหาหลัก [จำนวน] หัวข้อ, องค์ความรู้ใหม่, สรุป, ประเภทเอกสารอ้างอิงที่ควรใช้
โครงสร้างเรียงจากพื้นฐานไปสู่การวิเคราะห์และข้อเสนอ
```

### 2.3 เขียนบทคัดย่อ
```
ช่วยเขียนบทคัดย่อบทความวิชาการภาษาไทยเรื่อง "[ชื่อเรื่อง]"
ความยาว 300-400 คำ ภาษาวิชาการ
ให้มี 4 ส่วน: ความสำคัญ วัตถุประสงค์ วิธีวิเคราะห์ ข้อสรุป
ต่อท้ายด้วยคำสำคัญ 3-5 คำ
```

### 2.4 แปล Abstract
```
ช่วยแปลบทคัดย่อภาษาไทยต่อไปนี้เป็นภาษาอังกฤษเชิงวิชาการ
ให้สำนวนเหมาะกับบทความวารสาร และรักษาความหมายเดิมให้ครบถ้วน
[วางบทคัดย่อภาษาไทย]
```

### 2.5 เขียนบทนำ
```
ช่วยเขียนบทนำบทความวิชาการเรื่อง "[ชื่อเรื่อง]"
สาขา [สาขาวิชา] โดยใช้กรอบแนวคิด [กรอบแนวคิด]
ขอให้ประกอบด้วย: ความเป็นมาและความสำคัญ, บริบททางวิชาการ, ช่องว่างขององค์ความรู้, วัตถุประสงค์, แนวทางการนำเสนอ
ความยาวประมาณ [จำนวน] ย่อหน้า ใช้น้ำเสียงแบบวารสาร TCI
```

### 2.6 เขียนเนื้อหาแบบวิเคราะห์
```
ช่วยเขียนหัวข้อ "[ชื่อหัวข้อย่อย]" ของบทความเรื่อง "[ชื่อเรื่อง]"
โดยใช้กรอบแนวคิด [กรอบแนวคิด] และข้อมูลพื้นฐาน [ใส่ข้อมูล]
ขอให้เขียนแบบวิเคราะห์ ไม่ใช่เพียงสรุปข้อมูล
แต่ละย่อหน้ามีโครงสร้าง: เปิดประเด็น → อธิบายข้อมูลสนับสนุน → วิเคราะห์ → เชื่อมไปประเด็นถัดไป
ความยาวประมาณ [จำนวนคำ] คำ
```

### 2.7 เขียนบทสรุป
```
ช่วยเขียนบทสรุปบทความวิชาการเรื่อง "[ชื่อเรื่อง]"
จากเนื้อหาหลักดังนี้ [วางสรุปประเด็นสำคัญ]
เงื่อนไข: ไม่คัดลอกจากเนื้อหาซ้ำ, เป็นการสังเคราะห์ภาพรวม, ชี้ข้อค้นพบทางความคิด, ระบุคุณค่าทางวิชาการ, ปิดท้ายด้วยข้อเสนอ
```

## ชุดที่ 3: Prompt ตรวจคุณภาพตาม 7 Cs

### ตรวจ 7 Cs แบบภาพรวม
```
ช่วยตรวจต้นฉบับบทความวิชาการนี้ตามหลัก 7 Cs ได้แก่
1) Correctness 2) Cogency 3) Clarity 4) Completeness 5) Concise 6) Consistency 7) Concatenation
กรุณาประเมินทีละข้อ โดยระบุ: จุดแข็ง, จุดอ่อน, ตัวอย่างประโยคที่ควรแก้, แนวทางปรับปรุง
แล้วสรุปภาพรวมของบทความตอนท้าย
[วางต้นฉบับ]
```

### ตรวจเฉพาะ Clarity + Concise
```
ช่วยตรวจข้อความนี้ในมิติ Clarity (ความชัดเจน) และ Concise (ความกระชับ)
โดยทำตารางดังนี้: ข้อความเดิม | ปัญหาที่พบ | ข้อความที่ปรับแล้ว | เหตุผลที่ปรับ
[วางข้อความ]
```

### ตรวจ Concatenation ทั้งบทความ
```
ช่วยตรวจว่าบทความนี้มีความเชื่อมโยงทั้งเรื่องหรือไม่ โดยวิเคราะห์ว่า
1) บทนำเปิดประเด็นอะไร
2) เนื้อหาแต่ละหัวข้อตอบประเด็นนั้นหรือไม่
3) มีส่วนใดหลุดประเด็นหรือซ้ำซ้อนหรือไม่
4) บทสรุปสอดรับกับบทนำหรือไม่
จากนั้นช่วยเสนอแผนปรับโครงสร้าง
[วางต้นฉบับ]
```

## ชุดที่ 4: Prompt เกลาภาษาและเตรียมส่งวารสาร

### เกลาภาษาให้เป็นวารสาร
```
ช่วยเกลาภาษาบทความวิชาการนี้ให้เป็นสำนวนวารสารวิชาการ
โดยคงสาระเดิมไว้ ปรับเฉพาะ:
1) ความเป็นทางการของภาษา 2) ความกระชับ 3) ความชัดเจน 4) ความลื่นไหลของย่อหน้า 5) ความสม่ำเสมอของคำศัพท์
ห้ามตัดสาระสำคัญ
[วางต้นฉบับ]
```

### ตรวจรูปแบบก่อนส่ง
```
ฉันจะส่งบทความเรื่อง "[ชื่อเรื่อง]" ไปยังวารสารบัณฑิตศึกษาวิชาการ (AGJ)
ช่วยทำ checklist ตรวจต้นฉบับก่อนส่ง ครอบคลุม:
1) ชื่อเรื่อง 2) บทคัดย่อ 3) คำสำคัญ 4) บทนำ 5) เนื้อหา 6) องค์ความรู้ใหม่
7) สรุป 8) เอกสารอ้างอิง 9) ความสอดคล้องอ้างอิง 10) ภาษา 11) รูปแบบ template
แล้วสรุปว่า "ผ่าน / ควรแก้ / ต้องแก้มาก" พร้อมเหตุผล
[วางต้นฉบับ]
```

## ชุดที่ 5: Prompt เฉพาะสาขา

### A) รัฐประศาสนศาสตร์
```
ให้คุณทำหน้าที่เป็นผู้เชี่ยวชาญด้านรัฐประศาสนศาสตร์และการเขียนบทความวิชาการสำหรับ AGJ
ฉันต้องการเขียนบทความเรื่อง "[ชื่อเรื่อง]"
ประเด็นหลักคือ [ประเด็น]
กรอบแนวคิดคือ [เช่น ธรรมาภิบาล / Digital Governance / NPM]
บริบทคือ [ประเทศไทย / อปท. / หน่วยงานรัฐ]
กรุณาช่วย: วิเคราะห์หัวข้อ, เสนอชื่อเรื่อง, วางโครง, เขียนบทนำ, เสนอหัวข้อเนื้อหา, เขียนเนื้อหาแบบวิเคราะห์เชิงนโยบาย, เขียนบทสรุป, ตรวจ 7 Cs
ห้ามแต่งอ้างอิงปลอม
```

### B) พุทธบูรณาการ
```
ให้คุณทำหน้าที่เป็นผู้เชี่ยวชาญด้านพุทธบูรณาการและการเขียนบทความวิชาการสำหรับ AGJ
ฉันต้องการเขียนบทความเรื่อง "[ชื่อเรื่อง]"
ประเด็นหลักคือ [ประเด็น]
หลักธรรมที่คาดว่าจะใช้คือ [เช่น อิทธิบาท 4 / พรหมวิหาร 4 / สังคหวัตถุ 4 / ไตรสิกขา]
บริบทคือ [องค์กร / ชุมชน / วัด / ผู้สูงอายุ / การศึกษา / การบริหาร]
กรุณาช่วย: วิเคราะห์หัวข้อ, เสนอหลักธรรมที่เหมาะ, วางโครง, เขียนเนื้อหาแบบวิเคราะห์ (ไม่ใช่เชิงเทศนา), สังเคราะห์ข้อเสนอเชิงพุทธบูรณาการ
ห้ามสร้างข้อมูลเท็จ ห้ามแต่งอ้างอิงปลอม
```
