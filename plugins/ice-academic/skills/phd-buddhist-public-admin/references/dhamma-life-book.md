# ธรรมนูญชีวิต — พุทธจริยธรรมเพื่อชีวิตที่ดีงาม
## แหล่งอ้างอิงการประยุกต์หลักธรรมกับบทบาทสังคมและการปกครอง

**บรรณานุกรม (ใช้อ้างในงานวิจัย):**
พระพรหมคุณาภรณ์ (ป. อ. ปยุตฺโต). *ธรรมนูญชีวิต พุทธจริยธรรมเพื่อชีวิตที่ดีงาม*. (กรุงเทพมหานคร: วัดญาณเวศกวัน จ.นครปฐม, ๒๕๔๑). ISBN 974-8239-004

**ชื่อภาษาอังกฤษ:** A Constitution for Living: Buddhist Principles for a Fruitful and Harmonious Life. Translated by Bruce Evans.

**โครงสร้างหนังสือ:**

- **ภาค ๒ ธรรมนูญชีวิต** — ๒๒ บทบาทหน้าที่ใน ๔ หมวด

## ภาค ๑: วินัยชาวพุทธ (The Buddhist's Discipline)

### หมวดหนึ่ง: วางฐานชีวิตให้มั่น

**กฎ ๑: เว้นชั่ว ๑๔ ประการ**

**ก. เว้นกรรมกิเลส ๔ (กรรมที่ทำให้ชีวิตมัวหมอง):**
1. ไม่ทำร้ายร่างกายทำลายชีวิต (เว้นปาณาติบาต)
2. ไม่ลักทรัพย์ละเมิดกรรมสิทธิ์ (เว้นอทินนาทาน)
3. ไม่ประพฤติผิดทางเพศ (เว้นกาเมสุมิจฉาจาร)
4. ไม่พูดเท็จโกหกหลอกลวง (เว้นมุสาวาท)
*อ้างอิง:* ที.ปา. ๑๑/๑๗๕–๑๗๖/๑๙๖ (D.III.181)

**ข. เว้นอคติ ๔:**
1. ไม่ลำเอียงเพราะชอบ (เว้นฉันทาคติ)
2. ไม่ลำเอียงเพราะชัง (เว้นโทสาคติ)
3. ไม่ลำเอียงเพราะเขลา (เว้นโมหาคติ)
4. ไม่ลำเอียงเพราะขลาด (เว้นภยาคติ)
*อ้างอิง:* ที.ปา. ๑๑/๑๗๖/๑๙๖ (D.III.182)

**ค. เว้นอบายมุข ๖** (ช่องทางแห่งความเสื่อม): ดื่มสุรา / เที่ยวกลางคืน / ดูการละเล่น / เล่นการพนัน / คบคนชั่ว / เกียจคร้านในการทำงาน
*อ้างอิง:* ที.ปา. ๑๑/๑๗๕/๑๙๖ (D.III.182)

**กฎ ๒: เตรียมทุนชีวิต ๒ ด้าน**

**ก. เลือกสรรคนที่จะเสวนา:**
- มิตรเทียม ๔ ประเภท: คนปอกลอก / คนดีแต่พูด / คนหัวประจบ / คนชักชวนในทางฉิบหาย
- มิตรแท้ ๔ ประเภท: มิตรอุปการะ / มิตรร่วมสุขร่วมทุกข์ / มิตรแนะประโยชน์ / มิตรมีความรักใคร่
*อ้างอิง:* ที.ปา. ๑๑/๑๗๗–๑๗๙/๑๙๗ (D.III.185–187)

**ข. จัดสรรทรัพย์ที่หามาได้:** แบ่ง ๔ ส่วน (โภควิภาค ๔)
- ส่วนที่ ๑: ใช้จ่ายเลี้ยงตนและครอบครัว
- ส่วนที่ ๒: เก็บออม สำรองฉุกเฉิน
- ส่วนที่ ๓: ลงทุนงาน
- ส่วนที่ ๔: ทำบุญ สงเคราะห์
*อ้างอิง:* องฺ.อฏก. ๒๓/๑๔๔/๒๕๗–๒๕๘ (A.IV.281)

**กฎ ๓: รักษาความสัมพันธ์ ๖ ทิศ (สิงคาลกสูตร)**
ทิศที่ ๑ บิดามารดา / ทิศที่ ๒ ครูอาจารย์ / ทิศที่ ๓ ภรรยา/สามี / ทิศที่ ๔ มิตรสหาย / ทิศที่ ๕ คนรับใช้และคนงาน / ทิศที่ ๖ พระสงฆ์
*อ้างอิง:* ที.ปา. ๑๑/๑๘๖–๑๘๙/๑๙๙–๒๐๒ (D.III.189)

## ภาค ๒: ธรรมนูญชีวิต (A Constitution for Living)

### หมวดนำ: คนกับความเป็นคน

#### บทที่ ๑ คนผู้เป็นสัตว์ประเสริฐ
ผู้มีความสมบูรณ์ ๓ ประการ: รูปสมบัติ + ทรัพย์สมบัติ + คุณสมบัติ
*อ้างอิง:* องฺ.ติก. ๒๐/๔๓๔/๑๓๒

#### บทที่ ๒ คนสมบูรณ์แบบ
สมบูรณ์ด้วย **อัตตสัมปทา ๔**: ความสำเร็จแห่งตน / **ปรสัมปทา ๔**: ความสำเร็จเพื่อผู้อื่น

---

### หมวดหนึ่ง: คนกับสังคม ⭐ (สำคัญสำหรับงานวิจัยรัฐศาสตร์/รัฐประศาสนศาสตร์)

#### บทที่ ๓ คนมีศีลธรรม (สมาชิกในหมู่อารยชน)
**ก.** มีสุจริต ๓ (กาย–วาจา–ใจ): ดำเนินชีวิตโดยสุจริต
**ข.** ประพฤติตามอารยธรรม ๕ ประการ
**ค.** อย่างต่ำมีศีล ๕
*อ้างอิง:* ที.ปา. ๑๑/๒๒๗/๒๓๑

#### บทที่ ๔ คนมีคุณแก่ส่วนรวม (สมาชิกที่ดีของสังคม)
**ก.** มีพรหมวิหาร ๔ (เมตตา–กรุณา–มุทิตา–อุเบกขา)
**ข.** บำเพ็ญการสงเคราะห์ด้วย **สังคหวัตถุ ๔** (ทาน–ปิยวาจา–อัตถจริยา–สมานัตตตา)

> 💡 **ใช้ในงานวิจัย:** เชื่อมกับ: ภาวะผู้นำ / CSR / การบริการสาธารณะ / ความพึงพอใจ

*อ้างอิง:* ที.ปา. ๑๑/๑๔๐/๑๖๗; ๒๓๓/๒๓๓ (D.III.152; 223)

#### บทที่ ๕ คนผู้เป็นส่วนร่วมที่ดีของหมู่ชน (สมาชิกที่ดีของชุมชน)
**ก.** พึ่งตนเองได้ด้วย **นาถกรณธรรม ๑๐**:
ศีล / พาหุสัจจะ / กัลยาณมิตตตา / โสวจัสสตา / กิงกรณีเยสุ ทักขตา / ธรรมกามตา / วิริยารัมภะ / สันตุฏฐี / สติ / ปัญญา
*อ้างอิง:* ที.ปา. ๑๑/๓๕๗/๒๘๑ (D.III.266,290)

**ข.** อยู่ร่วมในหมู่ด้วยดีตาม **สาราณียธรรม ๖**:
เมตตากายกรรม / เมตตาวจีกรรม / เมตตามโนกรรม / สาธารณโภคี / สีลสามัญญตา / ทิฏฐิสามัญญตา
*อ้างอิง:* ที.ปา. ๑๑/๓๑๗/๒๕๗ (D.III.245)

> 💡 **ใช้ในงานวิจัย:** เชื่อมกับ: การมีส่วนร่วม / OD / ธรรมาภิบาล / สังคมประชาธิปไตย

#### บทที่ ๖ คนมีส่วนร่วมในการปกครองที่ดี (สมาชิกที่ดีของรัฐ) ⭐
**ก.** รู้หลัก **อธิปไตย ๓**:
- อัตตาธิปไตย = ถือตนเป็นใหญ่ (Attādhipateyya)
- โลกาธิปไตย = ถือโลกเปนใหญ่ (Lokādhipateyya)
- **ธรรมาธิปไตย** = ถือธรรมเป็นใหญ่ (Dhammādhipateyya) — **พึงยึดข้อนี้**
*อ้างอิง:* ที.ปา. ๑๑/๒๒๘/๒๓๑ (D.III.220)

**ข.** มีส่วนร่วมในการปกครองตาม **อปริหานิยธรรม ๗**:
1. หมั่นประชุมกันเนืองนิตย์
2. พร้อมเพรียงกันประชุม เลิกประชุม ทำกิจร่วม
3. ไม่ถืออำเภอใจบัญญัติกฎเกณฑ์ที่ไม่ได้ตกลงกัน
4. ให้เกียรติเคารพนับถือผู้ใหญ่มีประสบการณ์
5. ให้เกียรติและคุ้มครองสตรี
6. เคารพบูชาสักการะสถานที่สำคัญ
7. อารักขาบรรพชิตผู้มีศีลธรรม
*อ้างอิง:* ที.ม. ๑๐/๖๘/๘๖ (D.II.73)

> 💡 **ใช้ในงานวิจัย:** เชื่อมกับ: การมีส่วนร่วมทางการเมือง / ธรรมาภิบาล / ประชาธิปไตย / อปท.

#### บทที่ ๗ คนผู้นำรัฐ (พระมหากษัตริย์หรือผู้ปกครองบ้านเมือง) ⭐
**ก.** ทรง **ทศพิธราชธรรม ๑๐**:
ทาน–ศีล–ปริจจาค–อาชชว–มัททว–ตป–อักโกธ–อวิหิงส–ขันติ–อวิโรธน
*อ้างอิง:* Khp.9; ขุ.ขุ.25/9/4

**ข.** บำเพ็ญกรณียของจักรพรรดิ (จักรวัตติวัตร ๑๒ ข้อ): ให้ความคุ้มครองโดยธรรมแก่ทุกหมู่ชน
*อ้างอิง:* ที.ปา. ๑๑/๓๗/๗๒ (D.III.61)

**ค.** ประกอบ **ราชสังคหวัตถุ ๔**:
1. สัสสเมธะ = ฉลาดส่งเสริมการเกษตร/เศรษฐกิจ
2. ปุริสเมธะ = ฉลาดบำรุงข้าราชการคนดีมีความสามารถ
3. สัมมาปาสะ = ผูกประสานปวงประชาด้วยนโยบายส่งเสริมอาชีพ
4. วาชเปยยะ = มีวาทะดูดดื่มใจ พูดไพเราะ มีประโยชน์ สร้างสามัคคี
*อ้างอิง:* สํ.ส. ๑๕/๓๕๑/๑๑๐ (S.I.76)

**ง.** ละเว้น **อคติ ๔**: ฉันทาคติ–โทสาคติ–โมหาคติ–ภยาคติ
*อ้างอิง:* ที.ปา. ๑๑/๑๗๖/๑๙๖ (D.III.182,288)

> 💡 **ใช้ในงานวิจัย:** เชื่อมกับ: ภาวะผู้นำทางการเมือง / ผู้นำทางการเมือง / ธรรมาภิบาล / ผู้บริหารองค์กร

### หมวดสอง: คนกับชีวิต

#### บทที่ ๘ คนมีชีวิตอยู่อย่างมั่นใจ
ดำเนินชีวิตให้บรรลุ **อัตถะ ๓**:
1. **ทิฏฐธัมมิกัตถะ** = ประโยชน์ปัจจุบัน (สุขภาพ / ทรัพย์ / สถานภาพ / ครอบครัวผาสุก)
2. **สัมปรายิกัตถะ** = ประโยชน์เบื้องหน้า (ศรัทธา / ศีล / จาค / ปัญญา)
3. **ปรมัตถะ** = ประโยชน์สูงสุด (ความเป็นอิสระทางจิต)

*ยังแบ่งเป็น ๓ ด้าน:* อัตตัตถะ (ประโยชน์ตน) / ปรัตถะ (ประโยชน์ผู้อื่น) / อุภยัตถะ (ประโยชน์ร่วม)
*อ้างอิง:* ที.ปา. ๑๑/๓๓–๓๕ (D.III.233)

> 💡 **ใช้ในงานวิจัย:** เชื่อมกับ: คุณภาพชีวิต / ความต้องการ / ความพึงพอใจ / การพัฒนาคุณภาพชีวิต

#### บทที่ ๙ คนประสบความสำเร็จ
**ก.** หลักความเจริญ — **วุฑฒิธรรม ๔** (สัปปุริสสังสวะ–สัทธัมมัสสวนะ–โยนิโสมนสิการ–ธัมมานุธัมมปฏิปัตติ)
**ข.** หลักความสำเร็จ — **อิทธิบาท ๔** (ฉันทะ–วิริยะ–จิตตะ–วิมังสา)
**ค.** หลักเผล็ดโพธิญาณ — **โพธิปักขิยธรรม** ๓๗ ประการ

> 💡 **ใช้ในงานวิจัย:** เชื่อมกับ: ประสิทธิผล / ประสิทธิภาพ / สมรรถนะ / แรงจูงใจ

#### บทที่ ๑๐ คนรู้จักทำมาหาเลี้ยงชีพ
**ก.** หาและรักษาทรัพย์ (อุฏฐาน–อารักขะ–กัลยาณมิตตตา–สมชีวิตา — ทิฏฐธัมมิกัตถประโยชน์ ๔)
**ข.** จัดสรรทรัพย์ (โภควิภาค ๔)
**ค.** ใช้ทรัพย์ ๑๐ แบบ (จาก อวณฺณารห บัคคล ๑๐)

#### บทที่ ๑๑ คนครองเรือนที่เลิศล้ำ ⭐
**ก.** มีความสุข ๔ (อัตถิสุข–โภคสุข–อนณสุข–อนวัชชสุข)
*อ้างอิง:* องฺ.จตุกฺก. ๒๑/๖๒/๙๒ (A.II.65)

**ข.** เป็นชาวบ้านแบบฉบับ (คฤหัสถ์ที่พระพุทธเจ้าสรรเสริญ ประเภทที่ ๑๐)
*อ้างอิง:* องฺ.ทสก. ๒๔/๙๑/๑๘๘

**ค.** กำกับชีวิตด้วย **ฆราวาสธรรม ๔** (สัจจะ–ทมะ–ขันติ–จาคะ)
*อ้างอิง:* ขุ.สุ. ๒๕/๓๑๑/๓๖๑

**ง.** รับผิดชอบชีวิตที่เกี่ยวข้อง ช่วยผู้ใกล้ชิดเจริญด้วย **อาริยวัฑฒิ ๕**:
1. งอกงามด้วยศรัทธา
2. งอกงามด้วยศีล
3. งอกงามด้วยสุตะ (ความรู้)
4. งอกงามด้วยจาคะ (การเสียสละ)
5. งอกงามด้วยปัญญา
*อ้างอิง:* องฺ.ปญฺจก. ๒๒/๔๐/๔๗ (A.III.80)

**จ.** ครองตนเป็นพลเมืองที่ดี ๑๐ ข้อ (ไม่คบชู้ / ไม่ใจแคบ / มีศีล ๕ / ปฏิบัติหน้าที่สม่ำเสมอ / ไม่ประมาท / มีวิจารณญาณ / สุภาพ / เสงี่ยมงาม / พูดนาฟัง ฯลฯ)

> 💡 **ใช้ในงานวิจัย:** เชื่อมกับ: คุณภาพชีวิต / ความผูกพันต่อองค์กร / CSR / สุขภาวะ

### หมวดสาม: คนกับคน

#### บทที่ ๑๗ คนงาน–นายงาน (ลูกจ้าง–นายจ้าง)
**ก. นายจ้างพึงบำรุงคนรับใช้และคนงาน ๕ ข้อ:**
1. จัดงานให้เหมาะสมกับกำลังความสามารถ
2. ให้ค่าตอบแทนที่เหมาะสม
3. รักษาพยาบาลเมื่อเจ็บป่วย
4. แบ่งปันของดีๆ ให้ได้รับรส
5. ปล่อยให้พักผ่อนตามโอกาส

**ข. ลูกจ้างพึงปฏิบัติต่อนายจ้าง ๕ ข้อ:**
1. ลุกขึ้นทำงานก่อนนาย
2. เลิกงานทีหลังนาย
3. ถือเอาแต่ของที่นายให้
4. ทำงานให้ดีขึ้นเสมอ
5. นำชื่อเสียงที่ดีมาให้นาย
*อ้างอิง:* ที.ปา. ๑๑/๑๘๘/๒๐๑ (D.III.191)

> 💡 **ใช้ในงานวิจัย:** เชื่อมกับ: HRM / ความพึงพอใจ / แรงจูงใจ / ความผูกพันต่อองค์กร

### หมวดสี่: คนกับมรรคา

#### บทที่ ๑๘ คนผู้สั่งสอนหรือให้การศึกษา (ครู อาจารย์)
**ก.** เป็น **กัลยาณมิตร** ๗ คุณสมบัติ (ปิโย–ครุ–ภาวนีโย–วัตตา–วจนักขโม–คัมภีรัญจ–โน จัตถาเน)
**ข.** ตั้งใจประสิทธิ์ความรู้
**ค.** มีลีลาครูครบทั้ง ๔ (สันทัสสนา–สมาทปนา–สมุตเตชนา–สัมปหังสนา)
**ง.** มีหลักตรวจสอบ ๓ (อดีต–อนาคต–ปัจจุบัน)
**จ.** ทำหน้าที่ครูต่อศิษย์ ๕ ข้อ
*อ้างอิง:* องฺ.สตฺตก. ๒๓/๖๔/๑๑๙

#### บทที่ ๑๙ คนผู้เล่าเรียนศึกษา
**ก.** รู้หลักบุพภาคของการศึกษา (**ปัจจัยให้เกิดสัมมาทิฏฐิ ๒**: ปรโตโฆสะ–โยนิโสมนสิการ)
**ข.** มีหลักประกันชีวิตที่พัฒนา (**นาถกรณธรรม ๑๐**)
**ค.** ทำตามหลักเสริมสร้างปัญญา (**โยนิโสมนสิการ**)
**ง.** ศึกษาให้เป็นพหูสูต (**พหูสูต ๕ ประการ**: ได้ยินมาก / จำไว้ได้ / สะสมไว้ / คล่องปาก / เข้าใจด้วยปัญญา)

> 💡 **ใช้ในงานวิจัย:** เชื่อมกับ: HRD / การพัฒนาบุคลากร / สมรรถนะ

## วิธีอ้างอิงธรรมนูญชีวิตในงานวิจัย

### เชิงอรรถ (ครั้งแรก):
```
พระพรหมคุณาภรณ์ (ป. อ. ปยุตฺโต), ธรรมนูญชีวิต พุทธจริยธรรมเพื่อชีวิตที่ดีงาม, (กรุงเทพมหานคร: วัดญาณเวศกวัน จ.นครปฐม, ๒๕๔๑), หน้า ๒๘–๓๒.
```

### เชิงอรรถ (อ้างซ้ำ มีเอกสารคั่น):
```
พระพรหมคุณาภรณ์, ธรรมนูญชีวิต, หน้า ๔๓–๔๗.
```

### บรรณานุกรม:
```
พระพรหมคุณาภรณ์ (ป. อ. ปยุตฺโต). ธรรมนูญชีวิต พุทธจริยธรรมเพื่อชีวิตที่ดีงาม. กรุงเทพมหานคร: วัดญาณเวศกวัน จ.นครปฐม, ๒๕๔๑.
```

### หมายเหตุการใช้:
- หนังสือนี้ **อ้างอิงพระไตรปิฎกทุกข้อ** — สามารถไล่ดูเลขพระไตรปิฎกที่ท้ายแต่ละบทแล้วอ้างพระไตรปิฎกโดยตรงได้
- เหมาะเป็น **แหล่งอ้างอิงรอง** ขยายความหลักธรรมที่พระไตรปิฎกกล่าวไว้
- มีทั้งภาษาไทยและอังกฤษ เหมาะสำหรับงานวิจัยที่ต้องการศัพท์บาลี–อังกฤษ
