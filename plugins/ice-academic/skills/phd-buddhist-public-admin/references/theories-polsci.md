## 1. กลุ่มผลประโยชน์ (Interest Group)
- ประเภทกลุ่ม / วิธีการดำเนินงาน / บทบาทต่อนโยบาย / ความสัมพันธ์กับรัฐ

## 2. การกล่อมเกลาทางการเมือง (Political Socialization)
- ด้านครอบครัว / โรงเรียน / กลุ่มเพื่อน / สื่อมวลชน / ประสบการณ์การเมือง
- **แหล่งที่มา:** Greenstein, F.I., *Children and Politics*, (New Haven: Yale University Press, 1965)

## 3. การจัดการความขัดแย้ง (Conflict Management)
**3.1 Thomas-Kilmann (5 รูปแบบ):** ① แข่งขัน ② ร่วมมือ ③ ประนีประนอม ④ หลีกเลี่ยง ⑤ ยอมรับ

**3.2 การแก้ไขความขัดแย้ง:** ① เจรจา ② ไกล่เกลี่ย ③ อนุญาโตตุลาการ ④ การไกล่เกลี่ยเชิงแปลงสภาพ

## 4. การตรวจสอบทางการเมือง (Political Investigation)
- ด้านองค์กรตรวจสอบ / กลไกการตรวจสอบ / ประสิทธิภาพ / ความโปร่งใส

## 5. การตัดสินใจไปใช้สิทธิออกเสียงเลือกตั้ง (Decision on Election Voting)
**5.1 ปัจจัยที่มีผลต่อการตัดสินใจ:** ① ผู้สมัคร ② นโยบายพรรค ③ ความภักดีพรรค ④ อิทธิพลกลุ่ม ⑤ การมีส่วนร่วมทางการเมือง
- **แหล่งที่มา:** ผู้วิจัยท้องถิ่น + เอกสารรัฐศาสตร์ มจร

## 6. การตื่นตัวทางการเมือง (Political Awareness)
**6.1 Political Awakening:** ① ความรู้ทางการเมือง ② การมีส่วนร่วม ③ การรับรู้ข้อมูล

**6.2** ด้านทัศนคติ / ความสนใจ / การรับรู้ / การมีส่วนร่วม

## 7. การปกครองคณะสงฆ์ (Sangha Governance)
- อำนาจ / หน้าที่ / พระธรรมวินัย / กฎมหาเถรสมาคม / จริยาสงฆ์ (ดูส่วนที่ 1 เพิ่มเติม)

## 8. การปกครองท้องถิ่น (Local Government)
- โครงสร้างอปท. / อำนาจหน้าที่ / งบประมาณ / การมีส่วนร่วมประชาชน

## 9. การปกครองท้องที่ (Area Government)
- กำนัน / ผู้ใหญ่บ้าน / หน้าที่ / อำนาจ / ความสัมพันธ์กับ อปท.

## 10. การพัฒนาทางการเมือง (Political Development)
**10.1 Pye (5 ลักษณะ):** ① การมีส่วนร่วม ② การรวมอำนาจ/กระจายอำนาจ ③ ความเท่าเทียม ④ ขีดความสามารถ ⑤ ความแตกต่างทางโครงสร้าง

**10.2 Almond & Powell:** ① เสถียรภาพทางการเมือง ② ความชอบธรรม ③ การมีส่วนร่วม ④ การรวมชาติ

## 11. การมีส่วนร่วมทางการเมือง (Political Participation)
**11.1 รูปแบบ (Verba & Nie):** ① ลงคะแนนเสียง ② กิจกรรมหาเสียง ③ ติดต่อเจ้าหน้าที่ ④ กิจกรรมสหกรณ์

**11.2 Cohen & Uphoff (4 ด้าน):**
① การมีส่วนร่วมในการตัดสินใจ ② การดำเนินงาน ③ รับผลประโยชน์ ④ การประเมินผล

**11.3 ระดับการมีส่วนร่วม (Arnstein):**
① การจัดการ ② การรักษา ③ การแจ้งข้อมูล ④ การปรึกษา ⑤ การเอาใจ ⑥ การเป็นหุ้นส่วน ⑦ มอบอำนาจ ⑧ ควบคุมโดยพลเมือง

**11.4 Erwin (4 ด้าน):** กระจาย / ปรึกษา / เผยแพร่ / ร่วมคิด

**11.5 Andrew (4 ด้าน):** ข้อมูล / ปรึกษา / ร่วมมือ / มอบอำนาจ

## 12. การเมืองภาคพลเมือง (Civil Politics)
**12.1** ① ด้านข้อมูล ② ด้านการมีส่วนร่วมทางการเมือง ③ ด้านกฎหมาย ④ ด้านสิทธิพลเมือง

## 13. การรับรู้ทางการเมือง (Political Perception)
- ด้านความรู้ / ทัศนคติ / ความเชื่อ / คุณค่า

## 14. การเลือกตั้ง (Election)
**14.1** ① ระบบการเลือกตั้ง ② กฎหมายเลือกตั้ง ③ การดำเนินการ ④ ผลการเลือกตั้ง

**14.2 ปัจจัยที่มีผลต่อการตัดสินใจ:** ตัวผู้สมัคร / พรรคการเมือง / นโยบาย / ระบบอุปถัมภ์ / สื่อมวลชน

## 15. การสมรู้ร่วมคิดทางการเมือง (Political Collusion)
- รูปแบบ / ลักษณะ / ผลกระทบ / การป้องกัน

## 16. การสื่อสาร (Communication)
**16.1 Shannon-Weaver Model:** ผู้ส่งสาร → ข้อความ → ช่องทาง → ผู้รับสาร → Feedback

**16.2 Berlo SMCR:** Source / Message / Channel / Receiver

**16.3 ประสิทธิภาพการสื่อสาร:** ① ความชัดเจน ② ความถูกต้อง ③ ความทันเวลา ④ ครบถ้วน ⑤ กระชับ

## 17. การสื่อสารทางการเมือง (Political Communications)
**17.1 Lasswell (5W):** Who / Says What / In Which Channel / To Whom / With What Effect

**17.2 Nimmo:** ① ผู้สื่อสาร ② เนื้อหา ③ ช่องทาง ④ ผู้รับสาร ⑤ ผล

**17.3 McNair:** ① ข่าว ② โฆษณา ③ การประชาสัมพันธ์ ④ การโน้มน้าวใจ

## 18. ความปรองดองทางการเมือง (Political Reconciliation)
**18.1** ① ยอมรับความจริง ② ความยุติธรรม ③ การรักษาบาดแผล ④ ความไม่ทำซ้ำ

**18.2 Lederach:** ① ความจริง ② ความเมตตา ③ ความยุติธรรม ④ สันติภาพ

## 19. ความปรองดองสมานฉันท์ (Reconciliation)
- ด้านยอมรับความแตกต่าง / เจรจา / ร่วมมือ / ให้อภัย / สร้างความไว้วางใจ

## 20. ความเป็นพลเมืองในระบอบประชาธิปไตย (Citizen in Democratic System)
**20.1** ① ด้านความรู้ ② ด้านทักษะ ③ ด้านคุณลักษณะ ④ ด้านการมีส่วนร่วมทางการเมือง

## 21. ความรู้ความเข้าใจทางการเมือง (Political Understanding)
- ความรู้เรื่องระบบการเมือง / กระบวนการนโยบาย / สิทธิเสรีภาพ / การมีส่วนร่วม

## 22. ความไว้วางใจทางการเมือง (Political Trust)
**22.1** ① ไว้วางใจในรัฐบาล ② ไว้วางใจในนักการเมือง ③ ไว้วางใจในสถาบัน ④ ไว้วางใจทางสังคม

## 23. ความเสมอภาคทางสังคม (Social Legitimacy/Equity)
- ด้านโอกาส / ทรัพยากร / สิทธิ / กระบวนการ

## 24. ค่านิยมทางการเมือง (Political Values)
- เสรีภาพ / เสมอภาค / ยุติธรรม / ความสงบสุข / ประสิทธิภาพ

## 25. คุณลักษณะผู้นำทางการเมือง (Political Leaders Characteristics)
- ความรู้/ประสบการณ์ / ทักษะการสื่อสาร / วิสัยทัศน์ / ความซื่อสัตย์ / บุคลิกภาพ

## 26. จริยธรรมทางการเมือง (Political Ethics)
- ความซื่อสัตย์ / ความรับผิดชอบ / ความโปร่งใส / ความยุติธรรม / การเคารพสิทธิ

## 27. จิตสำนึกทางการเมือง (Political Consciousness)
- ด้านความรู้ / ทัศนคติ / ค่านิยม / การมีส่วนร่วม

## 28. บทบาทของนักการเมือง (Politician Roles)
**28.1** ① บทบาทด้านนิติบัญญัติ ② ด้านตรวจสอบ ③ ด้านผู้แทนราษฎร ④ ด้านการบริหาร

## 29. บทบาทของสมาชิกวุฒิสภา (Senator's Roles)
- กลั่นกรองกฎหมาย / ตรวจสอบถ่วงดุล / การเลือกบุคคลดำรงตำแหน่ง / พิจารณาสนธิสัญญา

## 30. บทบาทของสมาชิกสภาผู้แทนราษฎร (Representative Roles)
- ออกกฎหมาย / ควบคุมรัฐบาล / ตรวจสอบงบประมาณ / แสดงเจตนารมณ์ประชาชน

## 31. ผู้นำทางการเมือง (Political Leader)
- คุณสมบัติ / ประเภท / บทบาท / อิทธิพล

## 32. พรรคการเมือง (Political Party)
- โครงสร้าง / หน้าที่ / ระบบพรรค / การสรรหาผู้สมัคร / นโยบายพรรค

## 33. พฤติกรรมทางการเมือง (Political Behavior)
**33.1** ① ด้านทัศนคติและความเชื่อ ② ด้านการมีส่วนร่วม ③ ด้านการลงคะแนนเสียง ④ ด้านการเคลื่อนไหว

## 34. ภาวะผู้นำทางการเมือง (Political Leadership)
**34.1 Gastil — เปิดกว้าง / สร้างแรงบันดาลใจ / ให้คำมั่น / ร่วมคิด

**34.2 Lussier & Achua:**
① ภาวะผู้นำเชิงคุณลักษณะนิสัย ② เชิงพฤติกรรม ③ เชิงสถานการณ์ ④ เชิงการเปลี่ยนแปลง

**34.3 ลักษณะทางกายภาพ (Bryman):** ความสูง / หน้าตา / ความมั่นใจ ฯลฯ

**34.4 Rensis Likert (4 ระบบ):** เผด็จการ / เผด็จการเชิงเมตตา / ปรึกษาหารือ / มีส่วนร่วม

**34.5 ภาวะผู้นำการเปลี่ยนแปลง (Bass & Daft):** ผู้ยิ่งใหญ่ / เชิงวิสัยทัศน์ / เชิงประทับใจ / การบริหารเชิงเปลี่ยนแปลง

## 35. แรงจูงใจทางการเมือง (Political Motivation)
