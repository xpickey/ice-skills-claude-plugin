## หมวด ๒

### 1. ธรรมเป็นโลกบาล (คุ้มครองโลก) ๒
- **หิริ** = ความละอายบาป (ความละอายต่อบาป)
- **โอตตัปปะ** = ความเกรงกลัวบาป (ความกลัวผลของบาป)
- *ใช้กับ:* จริยธรรม, ธรรมาภิบาล, จรรยาบรรณ, พฤติกรรมทางการเมือง

### 2. ธรรมมีอุปการะ ๒
- **สติ** = ความระลึกรู้
- **สัมปชัญญะ** = ความรู้ตัวทั่วพร้อม
- *ใช้กับ:* ผู้นำ, การตัดสินใจ, การบริหารงาน

### 3. ธรรมให้เกิดสัมมาทิฏฐิ ๒
- **ปรโตโฆสะ** = เสียงจากผู้อื่น (ข้อมูลจากผู้ให้สัมภาษณ์)
- **โยนิโสมนสิการ** = ความคิดโดยแยบคาย (การวิเคราะห์/สังเคราะห์ของผู้วิจัย)
- *ใช้กับ:* บทที่ 4 การนำเสนอผลเชิงคุณภาพ, การคิดวิเคราะห์

### 4. ธรรมอันทำให้งาม ๒
- **ขันติ** = ความอดทน
- **โสรัจจะ** = ความสงบเสงี่ยม
- *ใช้กับ:* คุณลักษณะผู้นำ, การบริการ
