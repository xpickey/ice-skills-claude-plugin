## ก. การวิจัยแบบผสานวิธี (Mixed Methods Research)

### ๓.๑ รูปแบบการวิจัย
- ระบุชัดว่าเป็น "การวิจัยแบบผสานวิธี" (Mixed Methods)
- ระบุว่า **อะไรเป็นตัวนำ** (เชิงคุณภาพหรือเชิงปริมาณ) และอะไรเป็นตัวสนับสนุน

### ๓.๒ การวิจัยเชิงคุณภาพ (Qualitative)

#### ผู้ให้ข้อมูลสำคัญ (สัมภาษณ์เชิงลึก)
- **จำนวน: ≥ 17 รูป/คน** (สำหรับวิจัยแบบผสานวิธี)
- ต้องมีความหลากหลาย ครอบคลุม และมีส่วนเกี่ยวข้องกับเรื่องที่วิจัยแท้จริง
- **กลุ่มที่ควรมี:**
  - กลุ่มผู้กำหนดนโยบาย (Policy Makers)
  - กลุ่มผู้ปฏิบัติงาน (Practitioners)
  - กลุ่มผู้เกี่ยวข้องหรือผู้รับบริการ (Stakeholders/Beneficiaries)
  - กลุ่มนักวิชาการในประเด็นที่วิจัย (Subject Matter Experts)
  - กลุ่มนักวิชาการด้านพระพุทธศาสนา (Buddhist Scholars)

#### ผู้ทรงคุณวุฒิ (สนทนากลุ่มเฉพาะ — Focus Group Discussion)
- **จำนวน: 9–12 รูป/คน**
- **ห้ามซ้ำ** กับผู้ให้สัมภาษณ์เชิงลึก
- **ทุกครั้ง ต้องมีอาจารย์ประจำหลักสูตร ≥ 1 รูป/คน** เข้าร่วม

#### การตรวจสอบคุณภาพเครื่องมือ
- ผู้เชี่ยวชาญ **5 รูป/คน** (ภายนอก มจร **≥ 2 รูป/คน**)
- ผู้เชี่ยวชาญต้องครอบคลุม **≥ 3 ด้าน:**
  1. ด้านเนื้อหา/ทฤษฎี
  2. ด้านระเบียบวิธีวิจัย
  3. ด้านพระพุทธศาสนา หรือด้านการปฏิบัติงานที่เกี่ยวข้อง
- หาค่า IOC (Index of Item-Objective Congruence); เกณฑ์ ≥ 0.5

### ๓.๓ การวิจัยเชิงปริมาณ (Quantitative)

#### ประชากรและกลุ่มตัวอย่าง
- ประชากร: ระบุชัดเจนพร้อมอ้างอิงแหล่งที่มา
- กลุ่มตัวอย่าง: แสดงสูตรหรือตาราง Krejcie & Morgan; ทำตารางสัดส่วน

**สูตร Taro Yamane:** n = N/(1+Ne²)

#### การตรวจสอบคุณภาพเครื่องมือ 2 ขั้นตอน
**ขั้นที่ 1 Validity:** IOC โดยผู้เชี่ยวชาญ 5 ท่าน (เหมือนเชิงคุณภาพ)
**ขั้นที่ 2 Reliability:** Try out **30 ชุด** → Cronbach's Alpha ≥ 0.7

#### สถิติที่ใช้

**สถิติพรรณนา:** ความถี่ / ร้อยละ / ค่าเฉลี่ย (X̄) / ส่วนเบี่ยงเบนมาตรฐาน (S.D.)

**เกณฑ์แปลค่าเฉลี่ย Likert 5 ระดับ:**
- 4.21–5.00 = มากที่สุด
- 3.41–4.20 = มาก
- 2.61–3.40 = ปานกลาง
- 1.81–2.60 = น้อย
- 1.00–1.80 = น้อยที่สุด

**สถิติอ้างอิง:**

| วัตถุประสงค์ | สถิติที่ใช้ |
|---|---|
| เปรียบเทียบ 2 กลุ่ม | t-test |
| เปรียบเทียบ 3 กลุ่มขึ้นไป | F-test (One-way ANOVA) |
| ศึกษาความสัมพันธ์ | Pearson Correlation |
| ศึกษาปัจจัยที่ส่งผล | Multiple Regression |
| โมเดลโครงสร้าง | SEM |

## ข. การวิจัยเชิงคุณภาพ (อย่างเดียว)

- ผู้ให้ข้อมูลสำคัญ: **≥ 25 รูป/คน**
- สนทนากลุ่มเฉพาะ: 9–12 รูป/คน (ไม่ซ้ำ; อาจารย์หลักสูตร ≥ 1 ทุกครั้ง)
- เครื่องมือ: IOC โดยผู้เชี่ยวชาญ 5 ท่าน (ไม่ต้อง Try out)

---

## ค. ตารางสรุปตัวเลขขั้นต่ำ

| รายการ | แบบผสานวิธี | เชิงคุณภาพล้วน |
|---|---|---|
| ผู้ให้ข้อมูลสำคัญ (สัมภาษณ์) | ≥ 17 | ≥ 25 |
| ผู้ทรงคุณวุฒิ (FGD) | 9–12 | 9–12 |
| ผู้เชี่ยวชาญตรวจเครื่องมือ | 5 (นอก มจร ≥ 2) | 5 (นอก มจร ≥ 2) |
| Try out (เชิงปริมาณ) | 30 ชุด | ไม่มี |
| อาจารย์ประจำหลักสูตรใน FGD | ≥ 1 ทุกครั้ง | ≥ 1 ทุกครั้ง |

## ง. กระบวนการวิจัยแบบผสานวิธี

```
1. กำหนดตัวแปร กรอบแนวคิด
2. สร้างเครื่องมือวิจัย (แบบสัมภาษณ์ + แบบสอบถาม)
3. ตรวจสอบคุณภาพเครื่องมือ (IOC 5 ผู้เชี่ยวชาญ + Try out 30 ชุด)
4. เก็บข้อมูลเชิงคุณภาพ (สัมภาษณ์ ≥17 + FGD 9-12)
5. เก็บข้อมูลเชิงปริมาณ (แบบสอบถามกลุ่มตัวอย่าง)
6. วิเคราะห์ข้อมูล (Content Analysis + สถิติพรรณนา + สถิติอ้างอิง)
7. สรุปผล → องค์ความรู้ใหม่
```
