## หลักธรรมที่ใช้บ่อยที่สุดในงานวิจัย มจร

### 1. อิทธิบาท ๔ (ใช้บ่อยที่สุด)
ฉันทะ / วิริยะ / จิตตะ / วิมังสา
→ ประสิทธิภาพ, ประสิทธิผล, สมรรถนะ, แรงจูงใจ

### 2. พรหมวิหาร ๔
เมตตา / กรุณา / มุทิตา / อุเบกขา
→ ภาวะผู้นำ, คุณลักษณะผู้นำ, การบริการ

### 3. สังคหวัตถุ ๔
ทาน / ปิยวาจา / อัตถจริยา / สมานัตตตา
→ การบริการสาธารณะ, ความพึงพอใจ, CSR

### 4. ทศพิธราชธรรม ๑๐
ทาน / สีล / ปริจจาค / อาชชว / มัททว / ตป / อักโกธ / อวิหิงส / ขันติ / อวิโรธน
→ ธรรมาภิบาล, ผู้นำ, นักการเมือง

### 5. ไตรสิกขา
ศีล / สมาธิ / ปัญญา
→ การบริหาร, HRD, คุณภาพชีวิต

### 6. สัปปุริสธรรม ๗
ธัมมัญญุตา / อัตถัญญุตา / อัตตัญญุตา / มัตตัญญุตา / กาลัญญุตา / ปริสัญญุตา / ปุคคลัญญุตา
→ ภาวะผู้นำ, ธรรมาภิบาล

### 7. ภาวนา ๔
กายภาวนา / สีลภาวนา / จิตภาวนา / ปัญญาภาวนา
→ HRD, คุณภาพชีวิต, สุขภาวะผู้สูงอายุ

### 8. ธรรมเป็นโลกบาล ๒
หิริ / โอตตัปปะ
→ จริยธรรม, ธรรมาภิบาล

### 9. อคติ ๔
ฉันทาคติ / โทสาคติ / โมหาคติ / ภยาคติ
→ จริยธรรม, ธรรมาภิบาล, พฤติกรรมทางการเมือง

### 10. อปริหานิยธรรม ๗ (วัชชี)
→ การมีส่วนร่วม, OD, ธรรมาภิบาล

## การอ้างอิงพระไตรปิฎก

**ฉบับหลัก:** พระไตรปิฎก ฉบับมหาจุฬาลงกรณราชวิทยาลัย (กรุงเทพมหานคร: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย, ๒๕๓๙)

**รูปแบบเชิงอรรถ:**
```
ขุ.ธ. (บาลี) ๒๕/๑๒/๒, ขุ.ธ. (ไทย) ๒๕/๑๒/๒.
```

**ค้นหาออนไลน์:** https://84000.org

**คำอ้างอิงหลักสำหรับขยายความ:**
- พระพรหมคุณาภรณ์ (ป.อ. ปยุตฺโต), *พุทธธรรม ฉบับปรับขยาย*, พิมพ์ครั้งที่ ๑๖, (กรุงเทพมหานคร: สหธรรมิก, ๒๕๕๘)
- พระพรหมคุณาภรณ์ (ป.อ. ปยุตฺโต), *พจนานุกรมพุทธศาสตร์ ฉบับประมวลธรรม*, พิมพ์ครั้งที่ ๒๓, (กรุงเทพมหานคร: ผลิธัมม์, ๒๕๖๐)
