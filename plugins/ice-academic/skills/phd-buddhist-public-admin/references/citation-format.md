## ๑. ระบบเชิงอรรถ (Footnote System)

### การพิมพ์เชิงอรรถ
- พิมพ์ที่ส่วนล่างของแต่ละหน้าที่อ้างอิง
- แยกจากเนื้อหาด้วยเส้นคั่น ยาว ~20 ตัวพิมพ์ จากขอบซ้าย
- Before 12 pt (จากบรรทัดสุดท้ายเนื้อหา) + After 6 pt (ก่อนบรรทัดแรกของเชิงอรรถ)
- ขนาดอักษร: **14 pt**; indent บรรทัดแรก: 0.7 นิ้ว (1.75 ซม.)
- บรรทัดต่อมา: ชิดขอบซ้าย; บรรทัดสุดท้ายห่างจากขอบล่าง 1 นิ้ว
- ลงเลขท้ายข้อความที่อ้างอิง; เริ่ม ๑ ใหม่ทุกบท
- ใช้ Word → References → Insert Footnote

## ๒. รูปแบบเชิงอรรถแต่ละประเภท

### ก. คัมภีร์พระไตรปิฎก (อ้างครั้งแรก)
**รูปแบบ:** ชื่อย่อคัมภีร์ (บาลี/ไทย) เล่ม/ข้อ/หน้า.

**ตัวอย่าง:**
```
๑ ขุ.ธ. (บาลี) ๒๕/๑๒/๒, ขุ.ธ. (ไทย) ๒๕/๑๒/๒.
```

**กรณีสรุปความ:** เขียน "ดูรายละเอียดใน..." ต่อด้วยคัมภีร์
```
๑ ดูรายละเอียดใน วิ.มหา. (บาลี) ๑/๓-๖/๕๕-๕๗, วิ.มหา. (ไทย) ๑/๓-๖/๗๑-๗๔.
```

**กรณีข้อธรรมพร้อมคำอธิบาย:** อ้าง 2 ช่วง
```
๑ องฺ.ปญฺจก. (ไทย) ๒๒/๑/๒/๓๑๐.
² พระธรรมปิฎก (ป.อ. ปยุตฺโต), พจนานุกรมพุทธศาสตร์ ฉบับประมวลธรรม, พิมพ์ครั้งที่ ๑๑, (กรุงเทพมหานคร: บริษัท สหธรรมิก จำกัด, ๒๕๔๕), หน้า ๑๒๔.
```

### ข. หนังสือทั่วไป (ภาษาไทย — อ้างครั้งแรกในบท)
**รูปแบบ:** ผู้แต่ง, *ชื่อเรื่อง*, พิมพ์ครั้งที่ (ถ้ามี), (สถานที่พิมพ์: สำนักพิมพ์, ปีพิมพ์), หน้า.

**ตัวอย่าง:**
```
¹ สุรพล สุยะพรหม, ปรัชญาการเมือง, (กรุงเทพมหานคร: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย, ๒๕๕๔), หน้า ๕.
```

### ค. หนังสือทั่วไป (ภาษาอังกฤษ)
**รูปแบบ:** Author, *Title*, Edition (if any), (Place: Publisher, Year), page(s).

**ตัวอย่าง:**
```
¹ Walpola Rahula, What the Buddha Taught, (New York: Grove Press, 1963), p. 8.
² Boonton Dockthaisong, Disruptive Public Administration 5G, (Bangkok: Mahachulalongkornrajavidyalaya press, 2020), p. 55.
```
*หมายเหตุ: อักษรโรมันใช้แบบอักษร "Calibri Light"*

### ง. หนังสือแปล
**รูปแบบ:** ผู้แต่ง, *ชื่อเรื่อง*, แปลโดย ชื่อผู้แปล, พิมพ์ครั้งที่, (สถานที่: สำนักพิมพ์, ปี), หน้า.

**ตัวอย่าง:**
```
¹ พระดับบลิว ราหุล, พระพุทธเจ้าสอนอะไร, แปลโดย ชูศักดิ์ ทิพย์เกสรและคณะ, พิมพ์ครั้งที่ ๓, (กรุงเทพมหานคร: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย, ๒๕๔๗), หน้า ๗๗.
```

### จ. บทความในหนังสือรวมบทความ
**รูปแบบ:** ผู้เขียน, "ชื่อบทความ", ใน *ชื่อเรื่อง*, รวบรวมโดย ชื่อผู้รวบรวม, (สถานที่: สำนักพิมพ์, ปี), หน้า.

**ตัวอย่าง:**
```
¹ สุรพล สุยะพรหม, "ความสัมพันธ์ไทย–จีน มิติที่สำคัญทางพระพุทธศาสนา", ใน สารนิพนธ์พุทธศาสตรบัณฑิต ของมหาวิทยาลัย รุ่นที่..., (กรุงเทพมหานคร: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย, ๒๕๔๗): ๖๒-๗๑.
```

### ฉ. บทความในวารสาร
**รูปแบบ:** ผู้เขียน, "ชื่อบทความ", *ชื่อวารสาร*, ปีที่ ฉบับที่ (เดือน ปี): เลขหน้า.

**ตัวอย่าง:**
```
¹ บุญทัน ดอกไธสง, "วิวัฒนาการรัฐประศาสนศาสตร์", วารสาร มจร สังคมศาสตร์ปริทรรศน์, ปีที่ ๖ ฉบับที่ ๒ (๒๕๖๐): ๑๕-๒๗.
² Surapon Suyaporm and Nun Nerashar Saksirisampant, "The Buddhist Administrative Method of the Monastery Meditation Centers", BU ACADEMIC REVIEW, Vol. 13 No. 2 (July–December 2014): 109-121.
```

### ช. ดุษฎีนิพนธ์/วิทยานิพนธ์/สารนิพนธ์
**รูปแบบ:** ชื่อผู้วิจัย, "ชื่อเรื่อง", ระดับและสาขาวิชา, (คณะ/บัณฑิตวิทยาลัย: ชื่อสถาบัน, ปีสำเร็จ), หน้า.

**ตัวอย่าง:**
```
¹ พระเทพปริยัติเมธี (สฤษดิ์ สิริธโร), "กระบวนการสร้างความปรองดองในสังคมไทยตามรูปแบบพุทธวิธีการชำระอธิกรณ์", ดุษฎีนิพนธ์พุทธศาสตรดุษฎีบัณฑิต สาขาวิชารัฐประศาสนศาสตร์, (บัณฑิตวิทยาลัย: มหาวิทยาลัยมหาจุฬาลงกรณราชวิทยาลัย, ๒๕๕๖), หน้า ๒๐.
² Mr. Sombat Namburi, "A Causal Relationship Model of Buddhist Integration of Change Leadership in 21st Century in Tourism Authority of Thailand", A Dissertation of Doctor of Philosophy in Public Administration, (Graduate School: Mahachulalongkornrajavidyalaya University, 2019), p. 82.
```

### ซ. รายงานวิจัย
**รูปแบบ:** ชื่อผู้วิจัย, "ชื่อหัวข้อ", รายงานวิจัย, (หน่วยงาน: สถาบัน, ปี), หน้า.

**ตัวอย่าง:**
```
¹ สุรพล สุยะพรหม และคณะ, "กลไกการพัฒนาส่งเสริมสุขภาวะโดยการยกระดับจิตการพึ่งตนเองของแกนนำผู้สูงอายุพุทธธรรม", รายงานวิจัย, (กรุงเทพมหานคร: สำนักงานคณะกรรมการการอุดมศึกษา, ๒๕๖๐), หน้า ๙.
```

## ๓. การอ้างซ้ำ

### ไม่มีเอกสารอื่นมาคั่น (อ้างซ้ำทันที)
- เรื่องเดียวกัน หน้าเดียวกัน: **"เรื่องเดียวกัน."** (ไทย) / **"Ibid."** (อังกฤษ)
- เรื่องเดียวกัน หน้าต่าง: **"เรื่องเดียวกัน, หน้า ๒๗."** / **"Ibid., p. 27."**
- ข้อความเดิม หน้าเดิม: **"อ้างแล้ว."** / **"Loc. cit."**
- ข้อความเดิม หน้าต่าง: **"อ้างแล้ว, หน้า ๒๗."** / **"Loc. cit., p. 27."**

### มีเอกสารอื่นมาคั่น
ลงแบบย่อ: **ชื่อผู้แต่ง, ชื่อเรื่อง, หน้า.** (ตัดสถานที่/สำนักพิมพ์/ปีออก)

```
ตัวอย่างเชิงอรรถไทย:
¹ พระเทพเวที (ประยุทธ์ ปยุตฺโต), กรรม ตามนัยแห่งพุทธธรรม, (กรุงเทพมหานคร: มูลนิธิพุทธธรรม, ๒๕๒๔), หน้า ๒๗.
² เรื่องเดียวกัน.  [= หน้า ๒๗ เรื่องเดิม]
³ อ้างแล้ว.  [= หน้า ๒๗ เรื่องเดิม]
⁴ นันทนา นันทวโรภาส, สื่อสารการเมือง: ทฤษฎีและการประยุกต์, (พระนครศรีอยุธยา: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย, ๒๕๖๓), หน้า ๘.
⁵ พระเทพเวที (ประยุทธ์ ปยุตฺโต), กรรม ตามนัยแห่งพุทธธรรม, หน้า ๓๑.  [= มีเอกสารคั่น ลงแบบย่อ]
```

## ๕. รูปแบบบรรณานุกรม

**หลักการ:** รูปแบบล้อตามเชิงอรรถ **เปลี่ยน ","  ทั้งหมดเป็น "."** (ยกเว้นหน้าปีพิมพ์ในวงเล็บ คงเครื่องหมาย "," ไว้)
**การพิมพ์:** ชิดขอบซ้าย; บรรทัดต่อมาย่อหน้าเข้า; "บรรณานุกรม" พิมพ์กลาง Before 12pt ห่างจากขอบบน 1.5 นิ้ว

### ตัวอย่างบรรณานุกรมภาษาไทย

**ก. ข้อมูลปฐมภูมิ**
```
มหาวิทยาลัยมหาจุฬาลงกรณราชวิทยาลัย. พระไตรปิฎกภาษาบาลี ฉบับมหาจุฬาเตปิฏก. กรุงเทพมหานคร: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย, ๒๕๓๕.
```

**ข. ข้อมูลทุติยภูมิ — หนังสือ:**
```
สุรพล สุยะพรหม. ปรัชญาการเมือง. กรุงเทพมหานคร: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย, ๒๕๕๔.
ธิติวุฒิ หมั่นมี. การบริหารกิจการคณะสงฆ์. กรุงเทพมหานคร: โรงพิมพ์มหาจุฬาลงกรณราชวิทยาลัย, ๒๕๕๔.
```

**บทความในวารสาร:**
```
บุญทัน ดอกไธสง. "วิวัฒนาการรัฐประศาสนศาสตร์". วารสาร มจร สังคมศาสตร์ปริทรรศน์. ปีที่ ๖ ฉบับที่ ๒ (๒๕๖๐): ๑๕-๒๗.
```

**ดุษฎีนิพนธ์:**
```
จรูญศักดิ์ สุนทรเดชา. "การพัฒนารูปแบบการเฝ้าระวังพฤติกรรมสุขภาพผู้สูงอายุ". ดุษฎีนิพนธ์ปรัชญาดุษฎีบัณฑิต สาขาวิชารัฐประศาสนศาสตร์. บัณฑิตวิทยาลัย: มหาวิทยาลัยมหาจุฬาลงกรณราชวิทยาลัย, ๒๕๖๒.
```

**สัมภาษณ์:**
```
ผู้ให้ข้อมูลสำคัญลำดับที่ ๑. สัมภาษณ์. ๑๒ มกราคม ๒๕๖๔.
ผู้เข้าร่วมสนทนากลุ่มเฉพาะลำดับที่ ๑. สนทนากลุ่มเฉพาะ. ๑๒ มกราคม ๒๕๖๔.
```

## ๖. หมายเหตุสำคัญ

**ชื่อผู้แต่ง (ไทย):** เรียงชื่อ ตามด้วยนามสกุล (เช่น สมภาร พรมทา, คึกฤทธิ์ ปราโมช, ม.ร.ว.)
**ชื่อผู้แต่ง (อังกฤษ):** เรียงนามสกุลก่อน ตามด้วยชื่อ (เช่น Rahula, Walpola)
**พระภิกษุ:** ใช้ชื่อ-ฉายา ตามจริง (เช่น พระมหาราชัน จิตฺตปาโล); ถ้ามีสมณศักดิ์ ใส่สมณศักดิ์ตามชื่อ
