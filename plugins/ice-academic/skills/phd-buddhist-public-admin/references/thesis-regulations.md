## 1. ส่วนนำ

### 1.1 บทคัดย่อ (Abstract) — รูปแบบ OMF บังคับ
| องค์ประกอบ | คำอธิบาย |
|---|---|
| **O = Objective** | วัตถุประสงค์การวิจัย — ต้องตรงกับบทที่ 1 ทุกข้อ |
| **M = Methodology** | รูปแบบวิจัย, ประชากร/กลุ่มตัวอย่าง, เครื่องมือ, ค่าความเชื่อมั่น, สถิติ |
| **F = Findings** | สรุปผลตามวัตถุประสงค์ — แต่ละข้อ = 1 ย่อหน้า |

**กฎบังคับ:**
- ความยาวรวมทุกส่วน **ไม่เกิน 2 หน้า**
- บทคัดย่อภาษาอังกฤษ: อนุโลมตามภาษาไทย ใช้ **ศัพท์เฉพาะทางการวิจัย** เท่านั้น
- เชิงปริมาณ: ระบุ ประชากร กลุ่มตัวอย่าง วิธีสุ่ม ค่าความเชื่อมั่น เครื่องมือ สถิติ
- เชิงคุณภาพ: ระบุ ผู้ให้ข้อมูลสำคัญ ผู้ทรงคุณวุฒิ เครื่องมือ วิธีวิเคราะห์

### 1.2 กิตติกรรมประกาศ — ลำดับบังคับ (ไม่เกิน 1 หน้า)
① ผู้บริหารหลักสูตร → ② อาจารย์ที่ปรึกษา → ③ คณะกรรมการสอบ → ④ ผู้เชี่ยวชาญตรวจเครื่องมือ → ⑤ ผู้ให้ข้อมูลสำคัญ/กลุ่มตัวอย่าง → ⑥ ผู้มีอุปการคุณ

### 1.3 สารบัญ — กฎบังคับ
- ขึ้นหน้าใหม่แยกต่างหากจากเนื้อหาส่วนอื่น
- ต้องประกอบด้วย **5 บท** ครบถ้วน
- **หัวข้อในสารบัญ** กับหัวข้อในแต่ละบท **ต้องมีข้อความตรงกัน**
- **เลขหน้าในสารบัญ** กับเลขหน้าในแต่ละบท **ต้องตรงกัน**

### 1.4 สารบัญตาราง / สารบัญแผนภาพ
- เลขลำดับ = **บทเป็นตัวตั้ง** + ลำดับ (เช่น ตาราง 3.1, แผนภาพ 2.3)
- หากชื่อยาว ≥ 2 แถว: เลขลำดับอยู่แถวแรก, หมายเลขหน้าอยู่แถวสุดท้าย

### 1.5 คำอธิบายสัญลักษณ์และคำย่อ
- เฉพาะคำย่อพระไตรปิฎก **ที่ปรากฏในเชิงอรรถจริง** เท่านั้น

- **แยก** ฉบับภาษาบาลี และภาษาไทย

## 2. บทที่ 1 — บทนำ

### 2.1 ความเป็นมาและความสำคัญของปัญหา
**กฎบังคับ:**
- เรียงลำดับ **จากเรื่องใหญ่ → เรื่องเล็ก** (macro → micro)
- ปัญหาที่นำมาวิจัย ต้องเป็นปัญหา **ที่เป็นที่ยอมรับโดยทั่วไป** และ **ต้องได้รับการแก้ไข**
- ต้องมี **การอ้างอิงเชิงอรรถ** บอกที่มาของข้อมูลอย่างถูกต้องทุกจุดที่มีข้อมูล
- เชื่อมโยงเนื้อหาให้เป็นเนื้อเรื่องเดียวกัน → สรุปสู่คำถามวิจัย
- **ความยาวรวม: ประมาณ 4–5 หน้า**

### 2.2 คำถามการวิจัย
- เขียนเป็น **ประโยคคำถาม** สั้นกะทัดรัด
- จำนวนข้อ **ตรงกับ** วัตถุประสงค์
- โดยทั่วไปกำหนด **3 ข้อ**
- ต้องเป็นคำถามที่ **สามารถหาคำตอบทางการวิจัยได้**

### 2.3 วัตถุประสงค์การวิจัย
- เขียนเป็น **ประโยคบอกเล่า** สั้นกะทัดรัด
- ต้องขึ้นต้นด้วยคำว่า **"เพื่อ..."** (เพื่อศึกษา / เพื่อวิเคราะห์ / เพื่อนำเสนอ)
- **3 ข้อ** ตรงกับคำถามวิจัย (จับคู่ 1:1)

### 2.4 ขอบเขตการวิจัย — 4 ด้านบังคับ
| ด้าน | กฎบังคับ |
|---|---|
| **ด้านเนื้อหา** | ระบุตามตัวแปรต้น-ตัวแปรตาม; **อ้างอิงเชิงอรรถ** แหล่งที่มาของตัวแปร |
| **ด้านประชากร** (ปริมาณ) | ระบุกลุ่ม จำนวน **อ้างอิงแหล่งที่มา** |
| **ด้านผู้ให้ข้อมูลสำคัญ** (คุณภาพ) | ระบุกี่กลุ่ม กลุ่มละกี่รูป/คน |
| **ด้านพื้นที่** | ระบุหน่วยงาน/องค์กร ที่ตั้ง **ให้เหตุผล** ว่าเลือกเพราะอะไร |
| **ด้านระยะเวลา** | ระบุ พ.ศ. เริ่ม-สิ้นสุด + จำนวนเดือน (≥ 4–5 เดือน) |

### 2.5 สมมติฐานการวิจัย (ถ้ามี)
- สอดคล้องกับวัตถุประสงค์
- ตั้งจาก **หลักเหตุผล/ทฤษฎี** ไม่ตั้งลอย
- ทดสอบได้ด้วยสถิติ
- วัตถุประสงค์ที่ตั้งสมมติฐาน: เพื่อ **เปรียบเทียบ** / **ศึกษาความสัมพันธ์** / **ศึกษาปัจจัย**

### 2.6 นิยามศัพท์เฉพาะ — กฎเคร่งครัด
- ⚠️ **ไม่มีการอ้างอิงใดๆ ทั้งสิ้น** ในส่วนนิยามศัพท์เฉพาะ — นิยามเองล้วนๆ
- นำคำหลักของชื่อเรื่องวิจัยมานิยามเสมอ
- นิยามทุกตัวแปรที่ใช้ในการวิจัย
- ต้องนำไปสู่การสร้าง **เครื่องมือวิจัย** ได้

### 2.7 ประโยชน์ที่ได้รับ
- เขียนเป็นข้อ (ประโยคบอกเล่า)
- **3 ข้อแรก** สอดคล้องกับวัตถุประสงค์
- **ข้อสุดท้าย**: การนำองค์ความรู้ไปใช้ประโยชน์ในทางปฏิบัติ

## 3. บทที่ 2 — แนวคิด ทฤษฎีและงานวิจัยที่เกี่ยวข้อง

### 3.1 แนวคิด ทฤษฎี — กฎบังคับ
- แยกเป็น **หัวข้อย่อยตามตัวแปร**
- **ทุกตัวแปรที่อยู่ในกรอบแนวคิด** ต้องนำมาทบทวนในบทนี้
- ⚠️ **ไม่เขียนชื่อนักวิชาการในเนื้อหา** → ใส่ในเชิงอรรถ (เว้นแต่มีคำเชื่อมโยงตลอดเนื้อหา)
- สรุปเป็น **ความเรียง** เมื่อจบแต่ละหัวข้อย่อย
- สรุปเป็น **ตาราง** ท้ายสุดของทุกหัวข้อย่อย
- แต่ละหัวข้อย่อยอ้างอิง **≥ 5 แหล่ง**
- แผนภาพจากผู้อื่น ต้องอ้างอิงเชิงอรรถ เว้นแต่ผู้วิจัยสร้างเอง

### 3.2 หลักพุทธธรรม — กฎบังคับ
- หลักธรรม **ต้องตรงหรือสัมพันธ์กับเรื่องที่วิจัย**
- ต้อง **เชื่อมโยงหลักธรรมกับตัวแปรวิจัย**
- อ้างอิง **พระไตรปิฎก ฉบับ มจร** เป็นหลัก
- ขยายความหลักธรรม → อ้างอิง **หนังสือที่น่าเชื่อถือ** เป็นหลัก
- สรุปเป็นตารางท้ายทุกหัวข้อย่อย

### 3.3 ข้อมูลบริบท
- เน้นเฉพาะ **เรื่องที่ทำวิจัย** ไม่ใช่ข้อมูลเชิงประวัติศาสตร์/ภูมิศาสตร์
- ระบุ โครงสร้างองค์กร พันธกิจ แผนงาน โครงการ/กิจกรรม

### 3.4 งานวิจัยที่เกี่ยวข้อง — กฎเคร่งครัด
- แยกตามหัวข้อย่อย/ตัวแปร
- แต่ละหัวข้อย่อย **≥ 3 เรื่อง**
- ⚠️ **ป.เอก: อ้างงานวิจัยระดับ ป.เอก หรือวิจัยทั่วไป/บทความวิจัยเท่านั้น**
- ⚠️ **อ้างงานวิจัย มจร ≥ ร้อยละ 60** ในสาขาที่ตรง
- นำเสนอเฉพาะ **ผลการวิจัย** เท่านั้น — **ห้าม** นำระเบียบวิธีวิจัยมาเขียน
- งานวิจัยต้อง **ทันสมัยเป็นปัจจุบัน**
- สรุปเป็นตารางท้ายแต่ละหัวข้อ

### 3.5 กรอบแนวคิดในการวิจัย — กฎบังคับ
- ระบุ **แหล่งที่มา** ของตัวแปรแต่ละด้าน → สังเคราะห์มาจากหลักธรรม/แนวคิด/ทฤษฎี/เกณฑ์ของใคร
- เขียนเป็น **แผนภาพ** มีลูกศรแสดงทิศทาง IV → DV
- ⚠️ **ทุกตัวแปรต้องมาจากการทบทวนในบทที่ 2 เท่านั้น**
- ระบุ ตัวแปรต้น/ตัวแปรตาม + **อ้างอิงเชิงอรรถแหล่งที่มา**

## 4. บทที่ 3 — วิธีดำเนินการวิจัย

### ก. การวิจัยแบบผสานวิธี (Mixed Methods)

#### 3.1 รูปแบบการวิจัย
- ระบุชัดเจนว่าเป็นการวิจัยแบบผสานวิธี
- ระบุว่าอะไรเป็น "ตัวนำ" อะไรเป็น "ตัวสนับสนุน"

#### 3.2 ส่วนเชิงคุณภาพ

**ผู้ให้ข้อมูลสำคัญ (สัมภาษณ์เชิงลึก):**
| กฎ | ค่า |
|---|---|
| จำนวนขั้นต่ำ | **≥ 17 รูป/คน** (ผสานวิธี) / **≥ 25 รูป/คน** (คุณภาพเดี่ยว) |
| กลุ่มผู้ให้ข้อมูล | กำหนดนโยบาย / ผู้ปฏิบัติงาน / ผู้เกี่ยวข้อง-ผู้รับบริการ / นักวิชาการ / นักวิชาการด้านพระพุทธศาสนา |
| คุณสมบัติ | หลากหลาย ครอบคลุม เกี่ยวข้องจริง |

**ผู้ทรงคุณวุฒิสนทนากลุ่มเฉพาะ:**
| กฎ | ค่า |
|---|---|
| จำนวน | **9–12 รูป/คน** |
| เงื่อนไข | ⚠️ **ห้ามซ้ำ** กับผู้ให้ข้อมูลสำคัญ |
| อาจารย์ประจำหลักสูตร | ⚠️ **ต้องเข้าร่วม ≥ 1 รูป/คน** |

**เครื่องมือวิจัย — กฎบังคับ:**
- ต้องสอดคล้องกับวัตถุประสงค์และรูปแบบวิจัย
- ต้องสร้างให้ **ครบทุกตัวแปร** ในกรอบแนวคิด
- ต้องสอดคล้องกับ **นิยามศัพท์เฉพาะ** (บทที่ 1)
- ต้องตอบวัตถุประสงค์ได้ **ครบทุกข้อ**

**คุณภาพเครื่องมือเชิงคุณภาพ:**
- ผ่าน Validity (IOC) โดยผู้เชี่ยวชาญ
- ผู้เชี่ยวชาญ **5 รูป/คน** (≥ 2 จากภายนอก มจร)
- ผู้เชี่ยวชาญ ≥ 3 ด้าน: ① เนื้อหา/ทฤษฎี ② ระเบียบวิธีวิจัย ③ พระพุทธศาสนา/การปฏิบัติงาน

#### 3.3 ส่วนเชิงปริมาณ

**ประชากรและกลุ่มตัวอย่าง:**
- ระบุประชากร อ้างอิงแหล่งที่มา
- แสดงวิธีสุ่ม + ขนาดกลุ่มตัวอย่าง + สูตรคำนวณ/ตาราง
- ทำตารางจำแนกสัดส่วน

**เครื่องมือเชิงปริมาณ — ผ่าน 2 ขั้นตอนบังคับ:**
| ขั้นตอน | รายละเอียด |
|---|---|
| **① Validity** | IOC หรือ CVI โดยผู้เชี่ยวชาญ 5 รูป/คน (≥ 2 ภายนอก) |
| **② Reliability** | Try out **30 ชุด** กับกลุ่มลักษณะใกล้เคียงกลุ่มตัวอย่างจริง |

**สถิติที่ใช้:**
- สถิติพรรณนา: ความถี่, ร้อยละ, ค่าเฉลี่ย (X̄), S.D.
- สถิติอ้างอิง: t-test, F-test, Pearson Correlation, Regression, SEM

### ข. การวิจัยเชิงคุณภาพ (อย่างเดียว)
- ผู้ให้ข้อมูลสำคัญ **≥ 25 รูป/คน**
- ส่วนอื่นเหมือนข้อ 3.2

## 5. บทที่ 4 — ผลการวิเคราะห์ข้อมูล

### 5.1 เชิงคุณภาพ — กฎเคร่งครัด
- สาระต้องสอดคล้องตามวัตถุประสงค์
- ⚠️ **ห้ามนำข้อความจากหนังสือมาเป็นผลวิจัย** → ต้องมาจากการสัมภาษณ์/สนทนากลุ่มที่ปฏิบัติจริงเท่านั้น
- มีข้อมูลตัวอย่างสนับสนุน (**ปรโตโฆสะ**)
- ผู้วิจัยต้อง **สรุปวิเคราะห์/สังเคราะห์เอง** ทุกประเด็นหลัก (**โยนิโสมนสิการ**)
- สรุปเป็น **ตารางแจกแจงความถี่** จัดกลุ่มประเด็นสาระ + จำนวนผู้ให้ข้อมูล

### 5.2 เชิงปริมาณ
- นำเสนอในรูปแบบ **ตาราง + คำอธิบายท้ายตาราง**
- สัญลักษณ์สถิติ **ถูกต้องและสอดคล้องกัน** ทั้งเล่ม

### 5.3 องค์ความรู้ — กฎบังคับ

**5.3.1 องค์ความรู้ที่ได้รับจากการวิจัย:**
- สร้าง **แผนภาพองค์ความรู้** ตามกรอบแนวคิด — ⚠️ **ไม่ใช่การเขียนกรอบแนวคิดเข้ามาอีกครั้ง**
- แสดงผลวิจัยตามวัตถุประสงค์ทั้ง 3 ข้อ
- อธิบาย/ขยายความ **≥ 3 หน้า**

**5.3.2 องค์ความรู้สังเคราะห์ใหม่เชิงนวัตกรรม:**
- เขียนเป็นแผนภาพ **ที่ไม่ซ้ำ** กับ 5.3.1 — เชิงสร้างสรรค์โดยผู้วิจัย
- อธิบาย/ขยายความ **≥ 2 หน้า**

## 6. บทที่ 5 — สรุป อภิปรายผลและข้อเสนอแนะ

### 6.1 สรุปผลการวิจัย
- สรุปตามวัตถุประสงค์แต่ละข้อ กระชับ ชัดเจน
- ⚠️ สรุปผล **โดยรวม** ตามวัตถุประสงค์ — **ไม่แยก** สรุปเชิงปริมาณ/เชิงคุณภาพ

### 6.2 อภิปรายผล — กฎบังคับ
- **เชื่อมโยง** กับงานวิจัยในบทที่ 2 (สอดคล้อง/ไม่สอดคล้อง + เหตุผล)
- ระบุ: **ค้นพบอะไร → เหตุใดจึงเป็นเช่นนั้น → เปรียบเทียบกับงานวิจัยของใคร**
- ⚠️ **ป.เอก: อ้างเฉพาะงานวิจัย ป.เอก / วิจัยทั่วไป / บทความวิจัย ที่ทบทวนในบทที่ 2 เท่านั้น**
- อภิปรายให้ครอบคลุมทุกตัวแปร
- อ้างอิงเชิงอรรถในทุกงานวิจัยที่อภิปรายเชื่อมโยง

### 6.3 ข้อเสนอแนะ — 3 ระดับบังคับ
| ระดับ | คำอธิบาย |
|---|---|
| **5.3.1 เชิงนโยบาย** | มาจากค่าเฉลี่ยต่ำสุด + ผลสัมภาษณ์; เรื่องใหญ่/กว้าง; จำนวนน้อยแต่ครอบคลุม |
| **5.3.2 เชิงปฏิบัติการ** | ลงรายละเอียดจากนโยบาย; **จำนวนมากกว่า** 5.3.1 |
| **5.3.3 เพื่อการวิจัยครั้งต่อไป** | เนื้อหาใกล้เคียง + เทคนิควิจัยที่เหมาะสม |

## 7. บรรณานุกรม — กฎบังคับ

- **รูปแบบ:** ล้อตามเชิงอรรถ เปลี่ยน "," ทั้งหมดเป็น "." ยกเว้นหน้าปีพิมพ์
- **เรียงลำดับ:** ตามอักษรชื่อผู้แต่ง (ก – อย่างเคร่งครัด)
- คำนำหน้าชื่อ → นำไปไว้หลังนามสกุล
- แยกภาษาไทย / ภาษาอังกฤษ (อังกฤษ: นามสกุลก่อนชื่อเสมอ)
- แยก ข้อมูลปฐมภูมิ / ข้อมูลทุติยภูมิ
- **ลำดับประเภท:** คัมภีร์พระไตรปิฎก → หนังสือทั่วไป → หนังสือแปล → บทความ → ดุษฎีนิพนธ์/วิทยานิพนธ์ → รายงานวิจัย → สัมภาษณ์/สนทนากลุ่ม → สื่ออิเล็กทรอนิกส์
- ⚠️ เอกสารที่อ้างในเนื้อหา **ต้องลงในบรรณานุกรมให้ครบทุกเล่ม**

## 8. เชิงอรรถ (Footnote) — กฎบังคับ

### 8.1 ระบบ
- ใช้ระบบ **เชิงอรรถ (Footnote)** — ส่วนล่างของแต่ละหน้า
- ขีดเส้นคั่น ~20 ตัวพิมพ์; Before 12pt; After 6pt; ตัวอักษร **14pt**; indent **0.7 นิ้ว**
- เริ่มที่ **๑ ทุกบท** → นับต่อเนื่องจนจบบท → บทใหม่เริ่ม ๑ ใหม่

### 8.2 รูปแบบหลัก

**พระไตรปิฎก:**
```
ขุ.ธ. (บาลี) ๒๕/๑๒/๒, ขุ.ธ. (ไทย) ๒๕/๑๒/๒.
```
- สรุปข้อความจากพระไตรปิฎก: ใช้ "ดูรายละเอียดใน..." แล้วต่อด้วยคัมภีร์
- อ้างข้อธรรม + คำอธิบาย: อ้าง 2 ช่วง (ช่วงข้อธรรม + ช่วงคำอธิบาย)

**หนังสือทั่วไป:**
```
ผู้แต่ง, ชื่อเรื่อง, พิมพ์ครั้งที่ (ถ้ามี), (สถานที่พิมพ์: สำนักพิมพ์, ปีพิมพ์), หน้า.
```

**หนังสือแปล:**
```
ผู้แต่ง, ชื่อเรื่อง, แปลโดย ชื่อผู้แปล, พิมพ์ครั้งที่, (สถานที่พิมพ์: สำนักพิมพ์, ปีพิมพ์), หน้า.
```

**บทความจากหนังสือรวมบทความ:**
```
ผู้เขียน, "ชื่อบทความ", ใน ชื่อเรื่อง, รวบรวมโดย ชื่อ, (สถานที่พิมพ์: สำนักพิมพ์, ปีพิมพ์): เลขหน้า.
```

**บทความในวารสาร:**
```
ผู้เขียน, "ชื่อบทความ", ชื่อวารสาร, เล่มที่/ปีที่ (เดือน ปี): เลขหน้า.
```

**อ้างซ้ำ (ไม่มีเอกสารคั่น):** "เรื่องเดียวกัน." หรือ "อ้างแล้ว." (เพิ่ม ", หน้า X" ถ้าหน้าต่าง)
**อ้างซ้ำ (มีเอกสารคั่น):** ชื่อผู้แต่ง + ชื่อเรื่อง + หน้า (ตัดสถานที่/สำนักพิมพ์/ปีออก)

**สัมภาษณ์:** `สัมภาษณ์ ผู้ให้ข้อมูลสำคัญลำดับที่ ..., วัน เดือน ปี.`
**สนทนากลุ่ม:** `สนทนากลุ่มเฉพาะ ผู้เข้าร่วมลำดับที่ ..., วัน เดือน ปี.`

**อ้างจากดุษฎีนิพนธ์ของผู้อื่น:** ใช้คำว่า "อ้างใน..."
