## PART 3 — DISSERTATION STRUCTURE (โครงสร้าง 5 บท)

> ⚠️ **กฎระเบียบบังคับทั้งหมดอยู่ใน `references/thesis-regulations.md`**

### Quick Summary

**บทที่ 1 — บทนำ:** ความเป็นมา 4-5 หน้า → คำถาม 3 = วัตถุประสงค์ 3 → ขอบเขต 4 ด้าน → สมมติฐาน → ⚠️ นิยามศัพท์ไม่มีอ้างอิง → ประโยชน์

**บทที่ 2 — แนวคิด ทฤษฎี:** >= 5 แหล่ง/หัวข้อ → พุทธธรรมอ้างพระไตรปิฎก มจร → มจร >= 60% → กรอบแนวคิด IV → DV

**บทที่ 3 — วิธีวิจัย:** คุณภาพ >= 17 คน (ผสาน) / >= 25 (เดี่ยว) → สนทนากลุ่ม 9-12 คน → ผู้เชี่ยวชาญ 5 คน → Try out 30 ชุด

**บทที่ 4 — ผลวิเคราะห์:** ⚠️ ห้ามนำหนังสือมาเป็นผลวิจัย → ปรโตโฆสะ + โยนิโสมนสิการ → องค์ความรู้ >= 3 หน้า + สังเคราะห์ >= 2 หน้า

**บทที่ 5 — สรุป:** สรุปรวม (ไม่แยกปริมาณ/คุณภาพ) → อภิปรายเชื่อมบทที่ 2 → ข้อเสนอแนะ 3 ระดับ
